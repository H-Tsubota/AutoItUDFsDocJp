Function Reference

_WordDocAdd

新しい空のドキュメントを表すオブジェクト変数を返します。

#include <Word.au3>
_WordDocAdd ( ByRef $o_object [, $i_DocumentType = 0 [, $s_Template = "" [, $b_NewTemplate = 0]]] )

 

パラメータ

$o_object Word.Applicationオブジェクトのオブジェクト変数
$i_DocumentType [オプション] 新しいドキュメントの種類を指定します
0 = (デフォルト) 空ドキュメント
1 = ウェブページ
2 = Eメールメッセージ (現在、機能していない)
3 = フレームセット
4 = XMLドキュメント
$s_Template [オプション] 新しいドキュメントに使用するテンプレートの名前を指定します
"" = (デフォルト) 通常のテンプレートが使用されます
$b_NewTemplate [オプション] ドキュメントをテンプレートとして開くかどうかを指定します
0 = (デフォルト) テンプレートとして開きません
1 = テンプレートとして開きます

 

返し値

成功: Word.Applicationを指すオブジェクト変数、ドキュメントオブジェクトを返します
失敗: 0を返し@ERRORを設定します
@Error: $_WordStatus_Success = 正常終了
$_WordStatus_GeneralError = 一般的なエラー
$_WordStatus_InvalidDataType = 無効なデータ型
$_WordStatus_InvalidObjectType = 無効なオブジェクト型
@Extended: 無効なパラメータの数を格納

 

注意

なし。

 

関連

_WordDocClose, _WordDocOpen, _WordDocSave, _WordDocSaveAs

 

例


; *******************************************************
; 例 1 - 空のWordウィンドウを作成し新しい空のドキュメントを追加
; *******************************************************
;
#include <Word.au3>
$oWordApp = _WordCreate ("")
$oDoc = _WordDocAdd ($oWordApp)
