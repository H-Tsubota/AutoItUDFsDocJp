Function Reference

_TempFile

一時ファイルの名前を生成します。このファイルは現在存在しないことが保証されています。

#Include <File.au3>
_TempFile([$s_DirectoryName = @TempDir [, $s_FilePrefix = "~" [, $s_FileExtension = ".tmp" [, $i_RandomLength = 7]]]])

 

パラメータ

$s_DirectoryName [オプション]ファイル名のためのディレクトリ名。デフォルトはユーザーの%TEMP%ディレクトリです。
$s_FilePrefix [オプション]ファイルの接頭辞。デフォルトは"~"
$s_FileExtension [オプション]ファイルの拡張子。デフォルトは".tmp"
$i_RandomLength [オプション]一意に決まる名前を生成するのに使用する文字の数。デフォルトは7

 

返し値

現在存在していない一時ファイルの名前を返します

 

注意

なし。

 

関連

 

例


#include <File.au3>

Dim $s_TempFile, $s_FileName

; @TempDir内で一意に決まるファイル名を生成します。
$s_TempFile = _TempFile()

; 指定したディレクト内でtst_で始まる一意に決まるファイル名を生成します。
$s_FileName = _TempFile("C:\", "tst_", ".txt", 7)

MsgBox(4096, "Info", "Names suitable for new temporary file : " & @CRLF & $s_TempFile & @CRLF & $s_FileName)

Exit
