Function Reference

_GDIPlus_GraphicsGetSmoothingMode

Graphicsオブジェクトの描画品質を取得します。

#Include <GDIPlus.au3>
_GDIPlus_GraphicsGetSmoothingMode($hGraphics)

 

パラメータ

$hGraphics Graphicsオブジェクトのハンドル

 

成功: スムージングモード。 次のいずれかです:
0 - スムージングは適用されていません
1 - 8X4のボックスフィルターを使用したスムージングが適用されています
2 - 8X8のボックスフィルターを使用したスムージングが適用されています
失敗: -1が返され@errorが設定されます

注意

なし。
