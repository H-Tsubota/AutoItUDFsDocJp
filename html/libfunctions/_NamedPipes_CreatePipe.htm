Function Reference

_NamedPipes_CreatePipe

匿名パイプを作成します。

#Include <NamedPipes.au3>
_NamedPipes_CreatePipe(ByRef $hReadPipe, ByRef $hWritePipe [, $tSecurity = 0 [, $iSize = 0]])

 

パラメータ

$hReadPipe パイプの読み取りハンドルを受ける変数。
$hWritePipe パイプの書き込みハンドルを受ける変数。
$tSecurity [オプション]返されたハンドルを子プロセスに継承できるかどうかを決めるtagSECURITY_ATTRIBUTES構造体。
0の場合、ハンドルを継承することはできません。
$iSize [オプション]パイプ用のバッファのバイト単位でのサイズ。0の場合、システムはデフォルトのバッファサイズを使用します。

 

返し値

成功: True
失敗: False

 

注意

なし。
