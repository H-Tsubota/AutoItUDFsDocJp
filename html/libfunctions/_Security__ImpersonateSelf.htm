_Security__ImpersonateSelf

呼び出し元プロセスのセキュリティコンテキストを偽装するアクセストークンを取得します。

#Include <Security.au3>
_Security__ImpersonateSelf([$iLevel = 2])

 

パラメータ

$iLevel [オプション]新しいトークンの偽装レベル:
0 - 匿名。 サーバープロセスはクライアントに関する識別情報を取得できず、クラインアントに偽装することもできません。
1 - 識別。サーバープロセスはセキュリティ識別子、権限といったクライアントに関する識別情報を取得できますが、クラインアントに偽装することはできません。
2 - 偽装。 サーバープロセスはそのローカルシステム上のクライアントセキュリティコンテキストに偽装できますが、 リモートシステム上のクラインアントに偽装することはできません。
3 - 委任。 サーバープロセスはリモートシステム上のクライアントのセキュリティコンテキストに偽装できます。

 

返し値

成功: True
失敗: False

 

注意

なし。
