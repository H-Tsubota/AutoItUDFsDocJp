Function Reference

_SetTime

システムの現在時刻を設定します。

#Include <Date.au3>
_SetTime($iHour, $iMinute [, $iSecond = 0])

 

パラメータ

$iHour 時間。 値: 0-23
$iMinute 分。 値: 0-59
$iSecond [オプション]秒。 値: 0-59

 

返し値

成功: 1
失敗: 0を返し、@errorを設定します。
@error: 0 - 正常終了
1 - 失敗
@extended: GetLastError()
エラーコード: http://msdn.microsoft.com/en-us/library/ms681381.aspx

 

注意

オプションパラメータ(iSecond)が定義されていない場合、この関数は現在の値を変更しません!

 

関連

_SetDate

 

例


#include <Date.au3>

_SetTime(20,15)     ; 時刻を20:15に設定
_SetTime(20,15,30)  ; 時刻を20:15:30に設定
