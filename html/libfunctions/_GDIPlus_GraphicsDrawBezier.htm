Function Reference

_GDIPlus_GraphicsDrawBezier

ベジェ曲線を描画します。

#Include <GDIPlus.au3>
_GDIPlus_GraphicsDrawBezier($hGraphics, $iX1, $iY1, $iX2, $iY2, $iX3, $iY3, $iX4, $iY4[, $hPen = 0])

 

パラメータ

$hGraphics Graphicsオブジェクトのハンドル
$hPen 円弧の描画に使用されるPenオブジェクトのハンドル
$iX1 開始点のX座標
$iY1 開始点のY座標
$iX2 1つめの制御点のX座標
$iY2 1つめの制御点のY座標
$iX3 2つめの制御点のX座標
$iY3 2つめの制御点のY座標
$iX4 終了点のX座標
$iY4 終了点のY座標
$hPen [オプション]ベジェ曲線の描画に使用されるペンのハンドル。0の場合、幅1の黒いペンが使用されます。

 

返し値

成功: True
失敗: False

 

注意

ベジェ曲線はその制御点上を通過しません。
制御点は磁石のように働き、曲線を特定の方向に引っ張ることで曲げます。

 

関連

 

こちらも参照

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hGraphic

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; ベジェ曲線を描画
    _GDIPlus_Startup ()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)

    _GDIPlus_GraphicsDrawBezier ($hGraphic, 50, 50, 100, 5, 125, 25, 250, 50)

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; リソースを破棄
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_Shutdown ()

EndFunc   ;==>_Main
