Function Reference

_NamedPipes_DisconnectNamedPipe

クライアントプロセスから名前付きパイプインスタンスのサーバー端点への接続を切断します。

#Include <NamedPipes.au3>
_NamedPipes_DisconnectNamedPipe($hNamedPipe)

 

パラメータ

$hNamedPipe 名前付きパイプインスタンスのサーバー端点のハンドル.

 

返し値

成功: True
失敗: False

 

注意

名前付きパイプインスタンスのクライアント端点が開いている場合、DisconnectNamedPipe関数はその名前付きパイプの端点を強制的に閉じます。クライアントは次にそのパイプにアクセス使用とした際にエラーを通知されます。
パイプを強制終了されるクライアントはそのパイプ端を閉じるためにCloseHandleを使う必要があります。
