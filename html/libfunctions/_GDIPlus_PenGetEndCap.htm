Function Reference

_GDIPlus_PenGetEndCap

ペンのエンドキャップを取得します。

#Include <GDIPlus.au3>
_GDIPlus_PenGetEndCap($hPen)

 

パラメータ

$hPen Penオブジェクトのハンドル

 

返し値

成功: ペン先の種類:
$GDIP_LINECAPFLAT - 平らなキャップ
$GDIP_LINECAPSQUARE - 四角いキャップ
$GDIP_LINECAPROUND - 丸いキャップ
$GDIP_LINECAPTRIANGLE - 三角形のキャップ
$GDIP_LINECAPNOANCHOR - 直線の終端を描画しません
$GDIP_LINECAPSQUAREANCHOR - 直線の終端を四角で止めます
$GDIP_LINECAPROUNDANCHOR - 直線の終端を丸で止めます
$GDIP_LINECAPDIAMONDANCHOR - 直線の終端をひし形で止めます
$GDIP_LINECAPARROWANCHOR - 直線の終端を矢印で止めます
$GDIP_LINECAPCUSTOM - 直線の終端をCustomLineCapで作成します
失敗: -1が返され@errorが設定されます

 

注意

なし。

 

関連

_GDIPlus_PenSetEndCap

 

こちらも参照

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hGraphic, $hPen

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; リソースを作成
    _GDIPlus_Startup ()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)
    $hPen = _GDIPlus_PenCreate (0xFF000000, 8)
    _GDIPlus_PenSetEndCap($hPen, $GDIP_LINECAPARROWANCHOR)

    ; ペンとキャップを表示
    MsgBox(4096, "Information", "Pen end cap type: " & _GDIPlus_PenGetEndCap ($hPen))

    ; 矢印を描画
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 130, 390, 130, $hPen)
    _GDIPlus_PenSetWidth($hPen, 6)
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 150, 390, 150, $hPen)
    _GDIPlus_PenSetWidth($hPen, 8)
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 170, 390, 170, $hPen)

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; リソースを破棄
    _GDIPlus_PenDispose ($hPen)
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_ShutDown ()

EndFunc   ;==>_Main
