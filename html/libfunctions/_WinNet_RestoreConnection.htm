Function Reference

_WinNet_RestoreConnection

ネットワークリソースへの接続を復元します。

#Include <WinNet.au3>
_WinNet_RestoreConnection([$sDevice = "" [, $hWnd = 0 [, $fUseUI = True]]])

 

パラメータ

$sDevice [オプション]"Z:"など接続先のドライブのローカル名。
空白の場合、この関数は現在のユーザーのレジストリに登録された全ての恒久的ドライブに再接続します。
$hWnd [オプション]ネットワーク接続の確立時にユーザーにユーザー名とパスワードを求めるユーザーインターフェイス を表示するのにこの関数が使用する親ウィンドウのハンドル。
0の場合、オーナーウィンドウ無しになります。
$fUseUI [オプション]Trueの場合、呼び出し側にユーザー名/パスワードのプロンプトを表示します。

 

返し値

成功: True
失敗: False

 

注意

なし。
