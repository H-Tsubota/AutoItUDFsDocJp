Function Reference

_DateIsLeapYear

与えられた年がうるう年かどうかを調べます。

#Include <Date.au3>
_DateIsLeapYear($iYear)

 

パラメータ

$iYear 調べる年(4桁)

 

返し値

成功: 1を返します
失敗: うるう年でない場合、0を返し@Errorを設定します
@error: 0 - 正常終了
1 - 無効な年

 

注意

なし。

 

関連

 

例


#include <Date.au3>

If _DateIsLeapYear( @YEAR ) Then
    MsgBox( 4096, "Leap Year", "This year is a leap year." )
Else
    MsgBox( 4096, "Leap Year", "This year is not a leap year." )
EndIf
