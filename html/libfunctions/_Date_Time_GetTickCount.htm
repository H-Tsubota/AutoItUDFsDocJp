Function Reference

_Date_Time_GetTickCount

Windowsが開始されてからの経過ミリ秒数を取得します。

#Include <Date.au3>
_Date_Time_GetTickCount()

 

パラメータ

なし。

 

返し値

Windowsが開始されてからの経過ミリ秒数を返します。

 

注意

経過時間はDWORD値として保持されています。
Windowsが49.7日間以上連続実行された場合、時刻はゼロにリセットされます。


 

関連

 

例


#include <GuiConstantsEx.au3>
#include <Date.au3>
#include <WindowsConstants.au3>

Global $iMemo

_Main()

Func _Main()
    Local $hGUI

    ; GUIを作成
    $hGUI = GUICreate("Time", 400, 300)
    $iMemo = GUICtrlCreateEdit("", 2, 2, 396, 296, $WS_VSCROLL)
    GUICtrlSetFont($iMemo, 9, 400, 0, "Courier New")
    GUISetState()

    ; ミリ秒数を取得
    MemoWrite("Tick count .: " & _Date_Time_GetTickCount())

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
   
EndFunc   ;==>_Main

; メモコントロールに1行書き込む
Func MemoWrite($sMessage)
    GUICtrlSetData($iMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
