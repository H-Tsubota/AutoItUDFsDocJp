Function Reference

_SQLite_Shutdown

SQLite.dllをアンロードします。

#include <SQLite.au3>
_SQLite_Shutdown ( )

 

パラメータ

なし。

 

返し値

なし。

 

注意

_SQLite_Startup()が失敗、または実行されていない場合、AutoItランタイムエラーが投げられスクリプトが終了します!
AutoItは開かれている全てのdllを自動で閉じますが、_SQLite_Shutdownの呼び出しをおこなった方がいいでしょう。

 

関連

_SQLite_Startup

 

例


#include <SQLite.au3>
#include <SQLite.dll.au3>
_SQLite_Startup ()
ConsoleWrite("_SQLite_LibVersion=" &_SQLite_LibVersion() & @CRLF)
If @error Then
    MsgBox(16, "SQLite Error", "SQLite.dll Can't be Loaded!")
    Exit - 1
EndIf
_SQLite_Shutdown ()
