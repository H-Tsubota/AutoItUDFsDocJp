Function Reference

_FTP_Close

_FTP_Openによるセッションを閉じます。

#Include <FTPEx.au3>
_FTP_Close($l_InternetSession)

 

パラメータ

$l_InternetSession _FTP_Open()の返し値

 

返し値

成功: 1
失敗: 0

 

注意

なし。

 

関連

_FTP_Open

MSDNライブラリでInternetCloseHandleを検索して下さい。

#include <FTPEx.au3>

$Open = _FTP_Open('MyFTP Control')
; ...
