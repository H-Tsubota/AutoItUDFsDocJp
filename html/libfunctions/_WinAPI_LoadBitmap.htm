Function Reference

_WinAPI_LoadBitmap

モジュールの実行ファイルから指定されたビットマップリソースをロードします。

#Include <WinAPI.au3>
_WinAPI_LoadBitmap($hInstance, $sBitmap)

 

パラメータ

$hInstance ロードされるビットマップを格納した実行ファイルを持つモジュールのインスタンスのハンドル
$sBitmap ロードされるビットマップリソースの名前。
代わりに下位ワードをリソース識別子、上位ワードを0で構成された値を使用することもできます。

 

返し値

成功: 指定されたビットマップのハンドル
失敗: 0

 

注意

なし。
