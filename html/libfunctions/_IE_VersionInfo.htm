Function Reference

_IE_VersionInfo

IE.au3のバージョン情報を格納した配列を返します。

#include <IE.au3>
_IE_VersionInfo ( )

 

パラメータ

なし。<

 

返し値

成功: 配列を返します ($IEAU3VersionInfo)
$IEAU3VersionInfo[0] = リリースの種類 (T=テスト、V=プロダクト)
$IEAU3VersionInfo[1] = メジャーバージョン
$IEAU3VersionInfo[2] = マイナーバージョン
$IEAU3VersionInfo[3] = サブバージョン
$IEAU3VersionInfo[4] = リリースの日付け (YYYYMMDD)
$IEAU3VersionInfo[5] = 表示バージョン (e.g. V2.1-0)
失敗: なし。

 

注意

この配列を使うとUDFのバージョンが必須バージョンより古いのか、新しいのかを簡単に確かめられます。

 

関連

None.

 

例


; *******************************************************
; 例 1 - IE.au3のバージョン情報を取得し、表示します
; *******************************************************
;
#include <IE.au3>
$aVersion = _IE_VersionInfo ()
MsgBox(0, "IE.au3 Version", $aVersion[5] & " released " & $aVersion[4])
