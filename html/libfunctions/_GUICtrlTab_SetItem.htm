Function Reference

_GUICtrlTab_SetItem

タブの情報を設定します。

#Include <GuiTab.au3>
_GUICtrlTab_SetItem($hWnd, $iIndex[, $sText = -1[, $iState = -1[, $iImage = -1[, $iParam = -1]]]])

 

パラメータ

$hWnd コントロールのハンドル
$iIndex ゼロ始まりのアイテムインデックス
$sText [オプション]アイテムのテキスト
$iState [オプション]アイテムの状態
$iImage [オプション]ゼロ始まりのアイテム画像
$iParam [オプション]アプリケーション定義のデータ

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlTab_GetItem

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $aItem, $hTab

    ; GUIを作成
    GUICreate("Tab Control Set Item", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; タブ1のパラメータを取得/設定
    _GUICtrlTab_SetItem($hTab, 0, -1, -1, -1, 1234)
    $aItem = _GUICtrlTab_GetItem($hTab, 0)
    MsgBox(4160, "Information", "Tab 1 parameter: " & $aItem[3])

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
