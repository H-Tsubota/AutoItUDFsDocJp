Function Reference

_Security__LookupPrivilegeValue

権限値に対応するローカルの一意な識別子(LUID)を取得します。

#Include <Security.au3>
_Security__LookupPrivilegeValue($sSystem, $sName)

 

パラメータ

$sSystem 権限名を取得するシステムの名前を指定します。
空白の場合、この関数はローカルシステム上の権限名を取得しようとします。
$sName 権限の名前を指定します

 

返し値

成功: 既知の権限のLUID
失敗: 0

 

注意

なし。
