Function Reference

_GUICtrlHeader_SetHotDivider

ホットディバイダーカラーを変更します。

#Include <GuiHeader.au3>
_GUICtrlHeader_SetHotDivider($hWnd, $iFlag, $iInputValue)

 

パラメータ

$hWnd コントロールのハンドル
$iFlag $iInputValueで表される値の種類を指定する値。
この値は次のうちの1つです:
 True - $iInputValueがポインタのクライアント座標であることを表します
False - $iInputValueがディバイダーのインデックス値であることを表します
$iInputValue $iFlagに従って取り扱われる値

 

返し値

成功: コントロールがハイライトするディバイダーのインデックスと同じ値

 

注意

この関数はヘッダーコントロールにHDS_DRAGDROPが設定されている場合に自動的に処理される機能に影響します。
コントロールのオーナーがドラッグ&ドロップ操作を手動制御する場合に使用されることを想定しています。

 

関連

 

例


#include <GuiConstantsEx.au3>
#include <GuiHeader.au3>

Opt('MustDeclareVars', 1)

$Debug_HDR = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hGUI, $hHeader, $iTicks, $iIndex, $begin

    ; GUIを作成
    $hGUI = GUICreate("Header", 400, 300)
    $hHeader = _GUICtrlHeader_Create ($hGUI)
    GUISetState()

    ; 列を追加
    _GUICtrlHeader_AddItem ($hHeader, "Column 1", 100)
    _GUICtrlHeader_AddItem ($hHeader, "Column 2", 100)
    _GUICtrlHeader_AddItem ($hHeader, "Column 3", 100)
    _GUICtrlHeader_AddItem ($hHeader, "Column 4", 100)

    ; ユーザーが終了するまでループ
    Do
        If TimerDiff($begin) > 500 Then
            $iIndex = Mod($iIndex + 1, 4)
            _GUICtrlHeader_SetHotDivider ($hHeader, False, $iIndex)
            $begin = TimerInit()
        EndIf
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>_Main
