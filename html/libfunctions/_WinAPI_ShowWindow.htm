Function Reference

_WinAPI_ShowWindow

指定されたウィンドウの表示状態を設定します。

#Include <WinAPI.au3>
_WinAPI_ShowWindow($hWnd [, $iCmdShow = 5])

 

パラメータ

$hWnd ウィンドウのハンドル
$iCmdShow [オプション]ウィンドウがどのように表示されるかを指定します:
@SW_HIDE - ウィンドウを非表示にし、他のウィンドウをアクティブ化します
@SW_MAXIMIZE - 指定されたウィンドウを最大化します
@SW_MINIMIZE - 指定されたウィンドウを最小化し、Zオーダー順で次のトップレベルのウィンドウをアクティブ化します
@SW_RESTORE - ウィンドウをアクティブ化し、表示します
@SW_SHOW - 現在のサイズと位置でウィンドウをアクティブ化し、表示します
@SW_SHOWDEFAULT - STARTUPINFO構造体で指定されたSW_フラグに基づいて表示状態を設定します
@SW_SHOWMAXIMIZED - ウィンドウをアクティブ化し、最大化ウィンドウとして表示します
@SW_SHOWMINIMIZED - ウィンドウをアクティブ化し、最小化ウィンドウとして表示します
@SW_SHOWMINNOACTIVE - 最小化ウィンドウとして表示します
@SW_SHOWNA - 現在の状態でウィンドウを表示します
@SW_SHOWNOACTIVATE - 最新のサイズと位置でウィンドウを表示します
@SW_SHOWNORMAL - ウィンドウをアクティブ化し、表示します

 

返し値

成功: True
失敗: False

 

注意

なし。
