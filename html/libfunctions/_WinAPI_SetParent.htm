Function Reference

_WinAPI_SetParent

指定された子ウィンドウの親ウィンドウを変更します。

#Include <WinAPI.au3>
_WinAPI_SetParent($hWndChild, $hWndParent)

 

パラメータ

$hWndChild 子ウィンドウのウィンドウハンドル
$hWndParent 新しい親ウィンドウのハンドル。0の場合、デスクトップウィンドウが新しい親ウィンドウになります。

 

返し値

成功: 前の親ウィンドウのハンドル
失敗: 0

 

注意

アプリケーションはポップアップ、オーバーラップ、子ウィンドウの親ウィンドを設定するためにSetParentを使用することができます。
新しい親ウィンドウと子ウィンドウは同じアプリケーションに所属していなければなりません。
