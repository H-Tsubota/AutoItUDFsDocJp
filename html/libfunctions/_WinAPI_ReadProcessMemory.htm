Function Reference

_WinAPI_ReadProcessMemory

指定されたプロセスのメモリを読み取ります。

#Include <WinAPI.au3>
_WinAPI_ReadProcessMemory($hProcess, $pBaseAddress, $pBuffer, $iSize, ByRef $iRead)

 

パラメータ

$hProcess メモリを読み取るプロセスの開くハンドルを識別します
$pBaseAddress 指定された読み取るプロセスの基底アドレスを指します
$pBuffer アドレス空間からの内容を受け取るバッファを指します
$iSize 指定されたプロセスからの読み取り要求バイト数を指定します
$iRead 実際に指定されたバッファへ転送されたバイト数

 

返し値

成功: True
失敗: False

 

注意

なし。
