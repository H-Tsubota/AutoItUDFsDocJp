Function Reference

_GDIPlus_ArrowCapSetWidth

矢尻の幅を設定します。

#Include <GDIPlus.au3>
_GDIPlus_ArrowCapSetWidth($hArrowCap, $fWidth)

 

パラメータ

$hArrowCap ArrowCapオブジェクトのハンドル
$fWidth 矢尻の根元の幅を単位長さで指定します。

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GDIPlus_ArrowCapGetWidth

 

こちらも参照

MSDNライブラリでGdipSetAdjustableArrowCapWidthを検索して下さい。

 

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hGraphic, $hPen, $hEndCap

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; リソースを作成
    _GDIPlus_Startup ()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)
    $hPen = _GDIPlus_PenCreate (0xFF000000, 4)
    $hEndCap = _GDIPlus_ArrowCapCreate (3, 6)
    _GDIPlus_PenSetCustomEndCap ($hPen, $hEndCap)

    ; 矢印1を描画
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 120, 390, 120, $hPen)

    ; 矢印2を描画
    _GDIPlus_ArrowCapSetWidth ($hEndCap, _GDIPlus_ArrowCapGetWidth ($hEndCap) + 2)
    _GDIPlus_PenSetCustomEndCap ($hPen, $hEndCap)
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 150, 390, 150, $hPen)

    ; 矢印3を描画
    _GDIPlus_ArrowCapSetWidth ($hEndCap, _GDIPlus_ArrowCapGetWidth ($hEndCap) + 2)
    _GDIPlus_PenSetCustomEndCap ($hPen, $hEndCap)
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 190, 390, 190, $hPen)


    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; リソースを破棄
    _GDIPlus_ArrowCapDispose ($hEndCap)
    _GDIPlus_PenDispose ($hPen)
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_Shutdown ()
EndFunc   ;==>_Main
