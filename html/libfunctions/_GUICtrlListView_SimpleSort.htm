Function Reference

_GUICtrlListView_SimpleSort

リストビューコントロールをソートします(限定版)。

#Include <GuiListView.au3>
_GUICtrlListView_SimpleSort($hWnd, ByRef $vDescending, $iCol)

 

パラメータ

$hWnd コントロールのハンドル
$vDescending 次の値が使用できます:
 True - 降順にソート
 False - 昇順にソート
配列 - 次のフォーマット:
[0] - 1番目の列
[1] - 2番目の列
[n] - 最後の列
$iCol 列数

 

返し値

なし。

 

注意

これはごく基本的なソート関数です。
より応用的なソートをおこなうにはGUICtrlRegisterListViewSortを参照してください。

 

関連

GUICtrlRegisterListViewSort
