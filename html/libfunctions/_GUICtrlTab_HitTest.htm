Function Reference

_GUICtrlTab_HitTest

点がコントロール内のどの位置かを判定します。

#Include <GuiTab.au3>
_GUICtrlTab_HitTest($hWnd, $iX, $iY)

 

パラメータ

$hWnd コントロールのハンドル
$iX テストするX座標
$iY テストするY座標

 

返し値

成功: 次のフォーマットの配列:
        [0] - タブのゼロ始まりのインデックス。指定位置にタブがない場合は-1です。
        [1] - ヒットテストの結果:
        1 - 指定位置はタブ上ではありません
        2 - 指定位置はタブのアイコン上です
        4 - 指定位置はタブのテキスト上です

 

注意

なし。

 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $aHit, $hTab

    ; GUIを作成
    GUICreate("Tab Control HitTest", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; ヒットテストをおこなう
    $aHit = _GUICtrlTab_HitTest($hTab, 80, 10)
    MsgBox(4160, "Information", "Point [80,10] is over tab " & $aHit[0])

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
