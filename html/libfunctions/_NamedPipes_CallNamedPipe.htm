_NamedPipes_CallNamedPipe

名前付きパイプでの読み取り/書き込み操作を実行します。

#Include <NamedPipes.au3>
_NamedPipes_CallNamedPipe($sPipeName, $pInpBuf, $iInpSize, $pOutBuf, $iOutSize, ByRef $iRead[, $iTimeOut = 0])

 

パラメータ

$sPipeName パイプ名
$pInpBuf パイプへ書き込まれるデータを格納しているバッファへのポインタ
$iInpSize 書き込みバッファのバイト単位でのサイズ
$pOutBuf パイプから読み取られるデータを受けるバッファへのポインタ
$iOutSize 読み取りバッファのバイト単位でのサイズ
$iRead 返し時にパイプから読み取られたバイト数が格納されます
$iTimeOut [オプション]名前付きパイプが利用可能になるまで待機する際の待機時間をミリ秒数で指定します。
さらに次の特別な数値を指定できます:
-1 - ずっと待機します
 0 - CreateNamedPipe呼び出し時に指定されたデフォルトのタイムアウト時間を使用
 1 - 待機しません。パイプが利用可能でない場合はエラーを返します

 

返し値

成功: True
失敗: False

 

注意

CallNamedPipeの呼び出しはCreateFile(CreateFileがパイプを直ちに開けない場合はWaitNamedPipe)の呼び出しに、TransactNamedPipeはCloseHandleに相当します。
CreateFileはアクセスフラグ = GENERIC_READ | GENERIC_WRITEのとハンドル継承フラグ=Falseで呼び出されます。　
パイプがバイト型パイプの場合、CallNamedPipeは失敗します。
