Function Reference

_WinAPI_FindWindow

クラス名とウィンドウ名が一致するトップレベルウィンドウのハンドルを取得します。

#Include <WinAPI.au3>
_WinAPI_FindWindow($sClassName, $sWindowName)

 

パラメータ

$sClassName クラス名を指定する文字列またはクラス名文字列を識別するアトム。
このパラメータがアトムの場合、GlobalAddAtomの呼び出しによって作成されたグローバルアトムでなければなりません。
アトムは(16ビット値)は$sClassName文字列の下位ワードに配置し、上位ワードはゼロにしなければなりません。
$sWindowName ウィンドウ名を指定する文字列。このパラメータが空白の場合、全てのウィンドウ名が一致します。

 

返し値

成功: ウィンドウのハンドル
失敗: 0

 

注意

なし。
