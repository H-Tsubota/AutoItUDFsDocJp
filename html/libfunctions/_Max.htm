Function Reference

_Max

2つの数のうちどちらの方が大きいかを評価します。

#include <Math.au3>
_Max ( $nNum1, $nNum2 )

 

パラメータ

$nNum1 1つめの数。
$nNum2 2つめの数。

 

返し値

成功: 2つの数のうち大きい方を返します。
失敗: 0を返します。
@Error: 0 = 正常終了。
1 = $nNum1が数字ではありません。
2 = $nNum2が数字ではありません。

 

注意

この関数は整数だけでなく浮動小数でも動作します。

 

関連

_Min

 

例


#include <Math.au3>

$x = _Max(25, 10)  ; 25を返す
