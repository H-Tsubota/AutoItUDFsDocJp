Function Reference

_ExcelBookNew

新しいワークブックを作成し、そのオブジェクトの識別子を返します。

#Include <Excel.au3>
_ExcelBookNew([$fVisible = 1])

 

パラメータ

$fVisible [オプション]ワークブックを表示するかどうかのフラグ(0=非表示、1=表示)

 

返し値

成功: 新しいオブジェクトの識別子を返します
失敗: 0を返し@errorに次の値を設定します
@error: 1 - ExcelのCOMオブジェクトを作成できません
2 - $fVisibleパラメータが数値ではありません

 

注意

なし。

 

関連

_ExcelBookAttach

 

例


; *******************************************************
; 例 1 - Microsoft Excel ウィンドウを作成し、.xls ファイルを開く
; *******************************************************
#include <Excel.au3>
$oExcel = _ExcelBookNew()
