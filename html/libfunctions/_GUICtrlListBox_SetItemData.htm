Function Reference

_GUICtrlListBox_SetItemData

指定されたアイテムに関連付けられる値を設定します。

#Include <GuiListBox.au3>
_GUICtrlListBox_SetItemData($hWnd, $iIndex, $iValue)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex アイテムのゼロ始まりのインデックスを指定します
$iValue アイテムに関連付けられる値を指定します

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlListBox_GetItemData

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIListBox.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_LB = False ; ListBox関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

; 警告:ビルトイン関数を使用して作成されたアイテムに使用してはならない
; アイテムデータは核文字列のControlID

_Main()

Func _Main()
    Local $hListBox

    ; GUIを作成
    GUICreate("List Box Set Item Data", 400, 296)
    $hListBox = GUICtrlCreateList("", 2, 2, 396, 296)
    GUISetState()

    ; 文字列を追加
    _GUICtrlListBox_BeginUpdate($hListBox)
    For $iI = 1 To 9
        _GUICtrlListBox_AddString($hListBox, StringFormat("%03d : Random string", Random(1, 100, 1)))
    Next
    _GUICtrlListBox_EndUpdate($hListBox)

    ; アイテムデータを設定
    _GUICtrlListBox_SetItemData($hListBox, 4, 1234)

    ; アイテムデータを取得
    MsgBox(4160, "Information", "Item 5 Data: " & _GUICtrlListBox_GetItemData($hListBox, 4))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
