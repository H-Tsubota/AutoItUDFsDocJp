Function Reference

_GUICtrlComboBoxEx_ShowDropDown

コンボボックスのリストボックスの表示/非表示を切り替えます。

#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_ShowDropDown($hWnd[, $fShow = False])

 

パラメータ

$hWnd コントロールのハンドル
$fShow [オプション]ドロップダウンリストボックスの表示/非表示を指定します:
 True - リストボックスを表示
False - リストボックスを非表示

 

返し値

なし。

 

注意

この関数は$CBS_SIMPLEスタイルで作成されたコンボボックスには動作しません。

 

関連

_GUICtrlComboBoxEx_GetDroppedState

 

例


#include <GuiComboBoxEx.au3>
#include <GuiImageList.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_CB = False ; ComboBox/ComboBoxEx関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hGUI, $hImage, $hCombo
   
    ; GUIを作成
    $hGUI = GUICreate("ComboBoxEx Show Drop Down", 400, 300)
    $hCombo = _GUICtrlComboBoxEx_Create ($hGUI, "", 2, 2, 394, 100)
    GUISetState()

    $hImage = _GUIImageList_Create (16, 16, 5, 3)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 110)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 131)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 165)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 168)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 137)
    _GUIImageList_AddIcon ($hImage, @SystemDir & "\shell32.dll", 146)
    _GUIImageList_Add ($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap ($hCombo, 0xFF0000, 16, 16))
    _GUIImageList_Add ($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap ($hCombo, 0x00FF00, 16, 16))
    _GUIImageList_Add ($hImage, _GUICtrlComboBoxEx_CreateSolidBitMap ($hCombo, 0x0000FF, 16, 16))
    _GUICtrlComboBoxEx_SetImageList ($hCombo, $hImage)

    For $x = 0 To 8
        _GUICtrlComboBoxEx_AddString ($hCombo, StringFormat("%03d : Random string", Random(1, 100, 1)), $x, $x)
    Next

    ; ドロップダウン表示
    _GUICtrlComboBoxEx_ShowDropDown ($hCombo, True)

    Sleep(500)

    ; ドロップダウン非表示
    _GUICtrlComboBoxEx_ShowDropDown ($hCombo)


    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
