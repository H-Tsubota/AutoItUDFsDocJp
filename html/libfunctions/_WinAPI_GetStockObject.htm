Function Reference

_WinAPI_GetStockObject

定義済みのペン、ブラシ、フォント、パレットのハンドルを取得します。

#Include <WinAPI.au3>
_WinAPI_GetStockObject($iObject)

 

パラメータ

$iObject ストックオブジェクトの種類を指定します。このパラメータは次の値のいずれかです:
$BLACK_BRUSH - 黒いブラシ
$DKGRAY_BRUSH - 暗い灰色のブラシ
$GRAY_BRUSH - 灰色のブラシ
$HOLLOW_BRUSH - 空のブラシ (NULL_BRUSHと同じ)
$LTGRAY_BRUSH - 明るい灰色ブラシ
$NULL_BRUSH - 空のブラシ (HOLLOW_BRUSHと同じ)
$WHITE_BRUSH - 白いブラシ
$BLACK_PEN - 黒いペン
$NULL_PEN - 空のペン
$WHITE_PEN - 白いペン
$ANSI_FIXED_FONT - Windowsの固定ピッチ(固定幅)システムフォント
$ANSI_VAR_FONT - Windowsの可変ピッチ(可変幅)システムフォント
$DEVICE_DEFAULT_FONT - デバイス依存のフォント
$DEFAULT_GUI_FONT - ユーザーインターフェイスオブジェクト用のデフォルトフォント
$OEM_FIXED_FONT - OEM依存の固定ピッチ(固定幅)フォント
$SYSTEM_FONT - システムフォント
$SYSTEM_FIXED_FONT - Windows Ver3.0以前で使用されている固定ピッチ(固定幅)システムフォント
$DEFAULT_PALETTE - デフォルトパレット。 このパレットはシステムパレット内の静的な色で構成されます。

 

返し値

成功: 要求された論理オブジェクト
失敗: 0

 

注意

なし。
