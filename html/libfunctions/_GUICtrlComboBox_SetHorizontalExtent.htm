Function Reference

_GUICtrlComboBox_SetHorizontalExtent

リストボックスの水平スクロールできる幅をピクセル単位で設定します。

#Include <GuiComboBox.au3>
_GUICtrlComboBox_SetHorizontalExtent($hWnd, $iWidth)

 

パラメータ

$hWnd コントロールのハンドル
$iWidth リストボックスのスクロール可能幅をピクセル単位で指定します。

 

返し値

なし。

 

注意

リストボックスの幅が$iWidthより小さい場合、水平スクロールバーはリストボックスのアイテムを水平にスクロールします。

リストボックスの幅が$iWidth以上の場合、水平スクロールバーは非表示になります。さらにコンボボックスに$CBS_DISABLENOSCROLLスタイルが設定されている場合は無効になります。

 

関連

_GUICtrlComboBox_GetHorizontalExtent

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIComboBox.au3>
#include <GuiConstantsEx.au3>
#include <WindowsConstants.au3>

Opt('MustDeclareVars', 1)

$Debug_CB = False ; ComboBox/ComboBoxEx関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hCombo

    ; GUIを作成
    GUICreate("ComboBox Set Horizontal Extent", 400, 296)
    $hCombo = GUICtrlCreateCombo("", 2, 2, 396, 296, BitOR($CBS_SIMPLE, $CBS_DISABLENOSCROLL, $WS_HSCROLL))
    GUISetState()

    ; ファイルを追加
    _GUICtrlComboBox_BeginUpdate($hCombo)
    _GUICtrlComboBox_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($hCombo)

    ; 水平範囲を取得
    MsgBox(4160, "Information", "Horizontal Extent: " & _GUICtrlComboBox_GetHorizontalExtent($hCombo))
   
    ; 水平範囲を設定
    _GUICtrlComboBox_SetHorizontalExtent($hCombo, 500)
   
    ; 水平範囲を取得
    MsgBox(4160, "Information", "Horizontal Extent: " & _GUICtrlComboBox_GetHorizontalExtent($hCombo))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
