Function Reference

_Security__OpenThreadToken

スレッドに関連付けられたアクセストークンを開きます。

#Include <Security.au3>
_Security__OpenThreadToken($iAccess[, $hThread = 0 [, $fOpenAsSelf = False]])

 

パラメータ

$iAccess アクセストークンの要求アクセスタイプを指定するアクセスマスク。これらの要求アクセスタイプは
どのアクセスが許可/拒否されるかを決定するためにトークンの 随意アクセス制御リスト(DACL)と比較されます。
$hThread [オプション]アクセストークンを開かれるスレッドのハンドル
$fOpenAsSelf [オプション] OpenThreadToken関数を呼ぶスレッドのセキュリティコンテキストと、スレッドを呼ぶプロセスのセキュリティコンテキストのどちらを使ってアクセスチェックをおこなうかを示します。
このパラメータがFalseの場合、アクセスチェックは呼び出しスレッドのセキュリティコンテキストを使用しておこなわれます。
スレッドがクライアントを偽装している場合、このセキュリティコンテキストにクライアントプロセスのセキュリティコンテキストを使用できます。
このパラメータがTrueの場合、アクセスチェックは呼び出しスレッドのプロセスのセキュリティコンテキストを使用しておこなわれます。

 

返し値

成功: 新しく開かれたアクセストークンのハンドル
失敗: 0

 

注意

なし。

 

関連
