Function Reference

_GUICtrlTab_InsertItem

新しいタブを挿入します。

#Include <GuiTab.au3>
_GUICtrlTab_InsertItem($hWnd, $iIndex, $sText[, $iImage = -1[, $iParam = 0]])

 

パラメータ

$hWnd コントロールのハンドル
$iIndex 新しいタブのインデックス
$sText タブのテキスト
$iImage [オプション]ゼロ始まりのタブ画像インデックス
$iParam [オプション]アプリケーション定義のデータ

 

返し値

成功: 新しいタブのゼロ始まりのインデックス
失敗: -1

 

注意

なし。

 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hTab

    ; GUIを作成
    GUICreate("Tab Control Insert Item", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
