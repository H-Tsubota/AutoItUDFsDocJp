Function Reference

_WinAPI_SetProcessAffinityMask

指定されたプロセスのスレッドのプロセッサアフィニティマスクを設定します。

#Include <WinAPI.au3>
_WinAPI_SetProcessAffinityMask($hProcess, $iMask)

 

パラメータ

$hProcess この関数がアフィニティマスクを設定するプロセスのハンドル
$iMask アフィニティマスク

 

返し値

成功: True
失敗: False

 

注意

アフィニティマスクはビットマスクで各ビットはプロセスのスレッドが実行を許されているプロセッサを表します。
例えば、0x05のマスクを渡すとプロセッサ1と3が実行を許可されます。
