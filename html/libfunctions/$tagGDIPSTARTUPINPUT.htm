Keyword Reference

$tagGDIPSTARTUPINPUT

$tagGDIPSTARTUPINPUT構造体

Global Const $tagGDIPSTARTUPINPUT = "int Version;ptr Callback;int NoThread;int NoCodecs"

 

パラメータ

Version Microsoft Windows GDI+のバージョンを指定します
Callback デバグビルドでアサーション、警告のためにGDI+が呼べるコールバック関数へのポインタ
NoThread GDI+バックグラウンドスレッドを抑制するかどうか指定するブール値。 Trueに設定した場合、GdiplusStartupはフック関数へのポインタとアンフック関数へのポインタを返します。 バックグラウンドスレッドを置き換えるために、これらの関数を適切に呼ぶ必要があります。 フック、アンフック関数の呼び出しを自分でおこないたくない場合はこのメンバーをFalseにしてください。
NoCodecs GDI+の外部イメージコーデックを抑制するかどうか指定するブール値。 GDI+ version 1.0は外部イメージコーデックをサポートしていないため、このパラメータは無視されます。

 

注意

なし。
