Function Reference

_WinAPI_CreateSolidBitmap

ソリッドカラーのビットマップを作成します。

#Include <WinAPI.au3>
_WinAPI_CreateSolidBitmap($hWnd, $iColor, $iWidth, $iHeight [, $bRGB = 1])

 

パラメータ

$hWnd ビットマップが表示されるウィンドウのハンドル
$iColor RGBで規定されるビットマップの色
$iWidth ビットマップの幅
$iHeight ビットマップの高さ
$bRGB [オプション]Trueの場合、COLOREF (0x00bbggrr)に変換

 

返し値

成功: ビットマップのハンドル
失敗: 0

 

注意

なし。
