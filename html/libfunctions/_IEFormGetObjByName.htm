Function Reference

_IEFormGetObjByName

名前によってフォームへのオブジェクト参照を返します。

#include <IE.au3>
_IEFormGetObjByName ( ByRef $o_object, $s_name [, $i_index = 0] )

 

パラメータ

$o_object InternetExplorer.Applicationのオブジェクト変数、ウィンドウまたはフレームオブジェクト
$s_name 取得したいフォームの名前を指定します
$i_index [オプション]フォーム名が複数ある場合、0始まりのインデックスでインスタンスを指定します
0 (デフォルト) または正の整数の場合、そのインデックスのインスタンスを返します
-1 の場合、指定したフォームのコレクションを返します

 

返し値

成功: フォームオブジェクトを指すオブジェクト変数を返します。@EXTENDED = フォームの数
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
3 ($_IEStatus_InvalidDataType) = 無効なデータ型
7 ($_IEStatus_NoMatch) = 一致するものがありません
@Extended: 無効なパラメータの番号が格納されています

 

注意

欲しいフォームが名前もIDも持っていない場合、_IEFormGetCollectionを使ってインデックスによって参照を取得してください。

 

関連

_IEFormGetCollection, _IEFormReset, _IEFormSubmit

 

例


; *******************************************************
; 例 1 - 名前によって特定のフォームへの参照を取得。
;               このケースではGoogle検索エンジンにクエリをサブミット。
;              フォーム名、フォームの要素はページのHTMLソースを見ることでわかる。
; *******************************************************
;
#include <IE.au3>
$oIE = _IECreate ("http://www.google.com")
$oForm = _IEFormGetObjByName ($oIE, "f")
$oQuery = _IEFormElementGetObjByName ($oForm, "q")
_IEFormElementSetValue ($oQuery, "AutoIt IE.au3")
_IEFormSubmit ($oForm)
