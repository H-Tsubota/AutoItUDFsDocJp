Function Reference

_WinAPI_RegisterWindowMessage

システム全体を通してユニークであることが保証されている新しいウィンドウメッセージを定義します。

#Include <WinAPI.au3>
_WinAPI_RegisterWindowMessage($sMessage)

 

パラメータ

$sMessage 登録されるメッセージを指定する文字列

 

返し値

成功: 0xC000から0xFFFFの範囲のメッセージ識別子
失敗: 0

 

注意

RegisterWindowMessage関数は2つの共同して動作するアプリケーション間の通信用のメッセージを登録するのに使用されます。
2つの異なるアプリケーションが同じメッセージ文字列を登録すると、その2つのアプリケーションは同じメッセージ値を返すようになります。
メッセージはセッションが終了するまで登録され続けます。
