Function Reference

_GUICtrlSlider_SetLineSize

スライダーのラインのサイズを設定します。

#Include <GuiSlider.au3>
_GUICtrlSlider_SetLineSize($hWnd, $iLineSize)

 

パラメータ

$hWnd コントロールのハンドル
$iLineSize 新しいラインのサイズ

 

返し値

成功: 以前のラインのサイズ

 

注意

ラインサイズのデフォルト設定は1です。

 

関連

_GUICtrlSlider_GetLineSize

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiSlider.au3>

Opt('MustDeclareVars', 1)

$Debug_S = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hSlider

    ; GUIを作成
    GUICreate("Slider Set Line Size", 400, 296)
    $hSlider = GUICtrlCreateSlider(2, 2, 396, 20, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS, $TBS_ENABLESELRANGE))
    GUISetState()

    ; ラインサイズを取得
    MsgBox(4160, "Information", "Line Size: " & _GUICtrlSlider_GetLineSize($hSlider))
   
    ; ラインサイズを設定
    _GUICtrlSlider_SetLineSize($hSlider, 4)

    ; ラインサイズを取得
    MsgBox(4160, "Information", "Line Size: " & _GUICtrlSlider_GetLineSize($hSlider))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
