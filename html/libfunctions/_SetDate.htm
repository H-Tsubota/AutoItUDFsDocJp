Function Reference

_SetDate

システムの現在の日付けを設定します。

#Include <Date.au3>
_SetDate($iDay [, $iMonth = 0 [, $iYear = 0]])

 

パラメータ

$iDay 日。 値: 1-31
$iMon [オプション]月。 値: 1-12
$iYear [オプション]年。 値: > 0 (Windowsはこの値を強く制限します!!)

 

返し値

成功: 1
失敗: 0を返し、@errorを設定します。
@error: 0 - 正常終了
1 - 失敗
@extended: GetLastError()
エラーコード: http://msdn.microsoft.com/en-us/library/ms681381.aspx

 

注意

オプションパラメータ(iMon、iYear)が定義されていない場合、この関数は現在の値を変更しません!

 

関連

_SetTime

 

例


#include <Date.au3>

_SetDate(20)            ; 現在の日付けを20-xx-xxxxに設定
_SetDate(20,10)         ; 現在の日付けを 20-10-xxxxに設定
_SetDate(20,10,2004)    ; 現在の日付けを20-10-2004に設定
