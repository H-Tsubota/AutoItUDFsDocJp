Function Reference

_WinAPI_SetDefaultPrinter

ローカルコンピュータ上の現在のユーザーのデフォルトプリンタを設定します。

#Include <WinAPI.au3>
_WinAPI_SetDefaultPrinter($sPrinter)

 

パラメータ

$sPrinter デフォルトプリンタ名。リモートのプリンタでは名前のフォーマットは\\server\printernameになります。
ローカルのプリンタでは名前のフォーマットはprinternameになります。
このパラメータが""で既にデフォルトプリンタがある場合、この関数は何もおこないません。
ただしデフォルトプリンタがない場合はこの関数はローカルコンピュータにインストールされたプリンタを列挙し、
最初のプリンタをデフォルトプリンタに設定します。

 

返し値

成功: True
失敗: False

 

注意

なし。
