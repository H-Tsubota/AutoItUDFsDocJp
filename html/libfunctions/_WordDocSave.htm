Function Reference

_WordDocSave

開いているドキュメントを保存します。

#include <Word.au3>
_WordDocSave ( $o_object )

 

パラメータ

$o_object ドキュメントオブジェクトまたはWord.Applicationのオブジェクト変数

 

返し値

成功: 1を返します
失敗: 0を返し@ERRORを設定します
@Error: $_WordStatus_Success = 正常終了
$_WordStatus_GeneralError = 一般的なエラー
$_WordStatus_InvalidDataType = 無効なデータ型
$_WordStatus_InvalidObjectType = 無効なオブジェクト型
@Extended: 無効なパラメータの数を格納

 

注意

以前にドキュメントが保存されたことがない場合、ユーザーがファイルに名前をつけるための"名前をつけて保存"ダイアログボックスが表示されます。

 

関連

_WordDocAdd, _WordDocClose, _WordDocOpen, _WordDocSaveAs

 

例


; *******************************************************
; 例 1 - 既存のWordドキュメントを開きテキストを追加
;               保存して終了
; *******************************************************
;
#include <Word.au3>
$oWordApp = _WordCreate (@ScriptDir & "\Test.doc")
$oDoc = _WordDocGetCollection ($oWordApp, 0)
$oDoc.Range.insertAfter ("This is some text to insert.")
_WordDocSave ($oDoc)
_WordQuit ($oWordApp)
