Function Reference

_DebugBugReportEnv

バグレポートサブミット用の情報を格納した文字列を出力します。

#include <Debug.au3>
_DebugBugReportEnv( )

 

パラメータ

なし。

 

返し値

サブミットに必要な全ての情報を格納した文字列を返します。

 

注意

バグレポートのサブミット時に使用します。

 

関連

なし。

 

例


#include <Debug.au3>

Msgbox(4096, "Bug Report Environment", _DebugBugReportEnv() )
