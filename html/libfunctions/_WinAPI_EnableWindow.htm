Function Reference

_WinAPI_EnableWindow

指定されたウィンドウ、コントロールへのマウス、キーボード入力の有効/無効を切り替えます。

#Include <WinAPI.au3>
_WinAPI_EnableWindow($hWnd [, $fEnable = True])

 

パラメータ

$hWnd 有効/無効を切り替えるウィンドウのハンドル
$fEnable [オプション]ウィンドウの有効/無効を指定します:
 True - ウィンドウ、コントロールを有効にします
False - ウィンドウ、コントロールを無効にします

 

返し値

True: ウィンドウ、コントロールは以前は無効でした。
False: ウィンドウ、コントロールは以前は有効でした。

 

注意

なし。
