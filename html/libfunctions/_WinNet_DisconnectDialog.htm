Function Reference

_WinNet_DisconnectDialog

ネットワークリソース接続を切断するための汎用のブラウジングダイアログボックスを起動します。

#Include <WinNet.au3>
_WinNet_DisconnectDialog($hWnd)

 

パラメータ

$hWnd ダイアログボックスのオーナーウィンドウのハンドル

 

返し値

成功: True
失敗: False

 

注意

なし。
