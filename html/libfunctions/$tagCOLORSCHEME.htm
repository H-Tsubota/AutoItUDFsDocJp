Keyword Reference

$tagCOLORSCHEME

ツールバー、リバー(ReBar)内のボタン描画用の情報を格納しています。

Global Const $tagCOLORSCHEME = "dword Size;dword BtnHighlight;dword BtnShadow"

 

パラメータ

Size この構造体のバイト単位でのサイズ
BtnHighlight ボタンの強調色を表すCOLORREF値。デフォルトの強調色を使用するには$CLR_DEFAULTを使用してください
BtnShadow ボタンの影の色を表すCOLORREF値。デフォルトの影の色を使用するには$CLR_DEFAULTを使用してください

 

注意

なし。
