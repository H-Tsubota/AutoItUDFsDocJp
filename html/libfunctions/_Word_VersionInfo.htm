Function Reference

_Word_VersionInfo

Word.au3のバージョン情報の配列を返します。

#include <Word.au3>
_Word_VersionInfo ( )

 

パラメータ

なし。

 

返し値

成功: 配列($WordAU3VersionInfo)を返します
$WordAU3VersionInfo[0] = リリースの種類 (T=テスト、V=プロダクト)
$WordAU3VersionInfo[1] = メジャーバージョン
$WordAU3VersionInfo[2] = マイナーバージョン
$WordAU3VersionInfo[3] = サブバージョン
$WordAU3VersionInfo[4] = リリースの日付け (YYYYMMDD)
$WordAU3VersionInfo[5] = 表示バージョン (e.g. V1.0-1)
失敗: なし。

 

注意

なし。

 

関連

None.

 

例


; *******************************************************
; 例 1 - Word.au3 バージョン情報を取得、表示
; *******************************************************
;
#include <Word.au3>
$aVersion = _Word_VersionInfo ()
MsgBox(0, "Word.au3 Version", $aVersion[5] & " released " & $aVersion[4])
