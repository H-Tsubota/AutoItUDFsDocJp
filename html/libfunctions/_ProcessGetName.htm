Function Reference

_ProcessGetName

与えたPIDに所属するプロセス名を格納した文字列を返します。

#include <Process.au3>
_ProcessGetName ( $iPID )

 

パラメータ

$iPID 現在実行中のプロセスのPID。

 

返し値

成功: プロセスの名前を返します。
失敗: 空文字列を返し、@errorを設定します。
@error: 1 - プロセスが存在しません
2 - プロセスリスト取得エラー
3 - プロセスが見つかりません

 

注意

ProcessExists()を補足。

 

関連

ProcessExists

 

例


#include <Process.au3>

Run("notepad.exe")
WinWaitActive("[CLASS:Notepad]", "")
$pid = WinGetProcess("[CLASS:Notepad]", "")
$name = _ProcessGetName($pid)

MsgBox(0, "Notepad - " & $pid, $name)
