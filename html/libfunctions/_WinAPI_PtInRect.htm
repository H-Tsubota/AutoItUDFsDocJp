Function Reference

_WinAPI_PtInRect

指定された長方形内に指定された点があるかどうかを判定します。

#Include <WinAPI.au3>
_WinAPI_PtInRect(ByRef $tRect, ByRef $tPoint)

 

パラメータ

$tRect 指定された長方形を格納している$tagRECT構造体
$tPoint 指定された点を格納している$tagPOINT構造体

 

返し値

True: 点は長方形内にあります
False: 点は長方形内にありません

 

注意

なし。

 

関連

$tagRECT, $tagPOINT
