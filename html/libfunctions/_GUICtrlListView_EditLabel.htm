Function Reference

_GUICtrlListView_EditLabel

指定されたアイテムテキストのその場での編集を開始します。

#Include <GuiListView.au3>
_GUICtrlListView_EditLabel($hWnd, $iIndex)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex 編集するアイテムのゼロ始まりのインデックス

 

返し値

成功: アイテムの編集に使用するエディットコントロールのハンドルを返します
失敗: 0

 

注意

ユーザーが編集を完了またはキャンセルした場合、エディットコントロールは破棄されハンドルは有効でなくなります。
エディットコントロールをサブクラス化することができますが、その場合は破棄できなくなります。

 

関連

_GUICtrlListView_CancelEditLabel
