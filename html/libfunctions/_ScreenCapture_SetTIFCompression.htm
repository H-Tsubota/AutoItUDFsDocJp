Function Reference

_ScreenCapture_SetTIFCompression

TIFFスクリーンキャプチャで使用される圧縮方式を設定します。

#Include <ScreenCapture.au3>
_ScreenCapture_SetTIFCompression($iCompress)

 

パラメータ

$iCompress 画像圧縮方式
0 - デフォルトのエンコーダ圧縮
1 - 非圧縮
2 - LZW圧縮

 

返し値

なし。

 

注意

明示的に設定しない場合はTIFスクリーンキャプチャのデフォルト値、LZW圧縮が使用されます。
