Function Reference

_WinAPI_IsWindowVisible

指定されたウィンドウの表示状態を取得します。

#Include <WinAPI.au3>
_WinAPI_IsWindowVisible($hWnd)

 

パラメータ

$hWnd ウィンドウのハンドル

 

返し値

True: ウィンドウは表示されています
Failse: ウィンドウは表示されていません

 

注意

ウィンドウの表示状態は$WS_VISIBLEスタイルのビットで表されます。
$WS_VISIBLEを設定するとウィンドウは表示され、その後の描画ではウィンドウが$WS_VISIBLEスタイルである間、表示され続けます。
