Function Reference

_NamedPipes_GetNamedPipeInfo

指定した名前付きパイプの情報を取得します。

#Include <NamedPipes.au3>
_NamedPipes_GetNamedPipeInfo($hNamedPipe)

 

パラメータ

$hNamedPipe 名前付きパイプインスタンスのハンドル。ハンドルは名前付きパイプへのGENERIC_READアクセスを持っている必要があります。

 

返し値

成功: 次のフォーマットの配列:
$aInfo[0] - ハンドルがサーバー端の場合True。Trueでない場合はクライアント端です。
$aInfo[1] - パイプハンドルがメッセージパイプの場合True。Trueでない場合はバイトパイプです。
$aInfo[2] - 送信データ用バッファのバイト単位でのサイズ
$aInfo[3] - 受信データ用バッファのバイト単位でのサイズ
$aInfo[4] - 作成できるパイプインスタンスの最大数
失敗: @errorを設定します。

 

注意

なし。
