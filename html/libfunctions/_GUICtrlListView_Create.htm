Function Reference

_GUICtrlListView_Create

リストビューコントロールを作成します。

#Include <GuiListView.au3>
_GUICtrlListView_Create($hWnd, $sHeaderText, $iX, $iY[, $iWidth = 150[, $iHeight = 150[, $iStyle = 0x0000000D[, $iExStyle = 0x00000000[, $fCoInit = False]]]]])

 

パラメータ

$hWnd 親ウィンドウまたはオーナーウィンドウのハンドル
$sHeaderText ヘッダーに表示されるテキスト。"|"で区切ります。
$iX コントロールの水平位置
$iY コントロールの垂直位置
$iWidth [オプション]コントロールの幅
$iHeight [オプション]コントロールの高さ
$iStyle [オプション]コントロールのスタイル:
$LVS_ALIGNLEFT - アイコン表示、スモールアイコン表示時にアイテムは左揃えされます。
$LVS_ALIGNTOP - アイコン表示、スモールアイコン表示時にアイテムはコントロール上部に揃えられます。
$LVS_AUTOARRANGE - アイコン表示、スモールアイコン表示時にアイコンが自動整列します。
$LVS_EDITLABELS - アイテムテキストをその場で編集できます。
$LVS_ICON - アイコン表示を指定します。
$LVS_LIST - リスト表示を指定します。
$LVS_NOCOLUMNHEADER - レポート表示で列のヘッダーを表示しません。
$LVS_NOLABELWRAP - アイコン表示でアイテムテキストを1行で表示します。
$LVS_NOSCROLL - スクロールを無効にします。
$LVS_NOSORTHEADER - 列のヘッダーをボタンにしません。
$LVS_OWNERDATA - 仮想コントロールを指定します。
$LVS_OWNERDRAWFIXED - レポート表示でオーナーウィンドウがアイテム描画をおこなえます。
$LVS_REPORT - レポート表示を指定します。
$LVS_SHAREIMAGELISTS - 画像リストを削除しません。
$LVS_SHOWSELALWAYS - 選択設定がある際は常に表示します。
$LVS_SINGLESEL - 一度に1つのアイテムのみしか選択できなくします。
$LVS_SMALLICON - スモールアイコン表示を指定します。
$LVS_SORTASCENDING - アイテムインデックスを昇順にソートします。
$LVS_SORTDESCENDING - アイテムインデックスを降順にソートします。
デフォルト: $LVS_REPORT、$LVS_SINGLESEL、$LVS_SHOWSELALWAYS
強制 : $WS_CHILD、$WS_VISIBLE
$iExStyle [オプション]コントロールの拡張スタイル:
$LVS_EX_BORDERSELECT - アイテムが選択された際にアイテムの枠の色が変わります。
$LVS_EX_CHECKBOXES - リストビューコントロールのアイテムのチェックボックスを有効にします。
$LVS_EX_DOUBLEBUFFER - ちらつきを抑えるためにダブルバッファで描画をおこないます。
$LVS_EX_FLATSB - リストビューの平らなスクロールバーを有効にします。
$LVS_EX_FULLROWSELECT - アイテムが選択された際にアイテムとその全てのサブアイテムをハイライトします。
$LVS_EX_GRIDLINES - アイテムとサブアイテムの周りにグリッド線を表示します。
$LVS_EX_HEADERDRAGDROP - ドラッグ&ドロップによる列の順序変更を有効にします。
$LVS_EX_INFOTIP - ツールチップ表示前に$LVN_GETINFOTIP通知メッセージが送信されます。
$LVS_EX_LABELTIP - 設定されていない場合、ラージアイコンモードでのみ部分的に非表示のラベルの展開をおこないます。
$LVS_EX_MULTIWORKAREAS - 1つ以上の作業領域が定義されるまでコントロールはアイコンの自動整列をおこないません。
$LVS_EX_ONECLICKACTIVATE - ユーザーがアイテムをクリックした際にコントロールは$LVN_ITEMACTIVATEメッセージを送信します。
$LVS_EX_REGIONAL - アイテムのアイコンとテキストのみを含むコントロール領域を設定します。
$LVS_EX_SIMPLESELECT - アイコン表示でコントロールの状態画像を右上に移動します。
$LVS_EX_SUBITEMIMAGES - サブアイテムへの画像表示を許可します。
$LVS_EX_TRACKSELECT - コントロールでのホットトラック選択を有効にします。
$LVS_EX_TWOCLICKACTIVATE - ユーザーがアイテムをダブルクリックした際にコントロールは$LVN_ITEMACTIVATEメッセージを送信します。
$LVS_EX_UNDERLINECOLD - アクティブ化されたホットでないアイテムのテキストに下線を表示します。
$LVS_EX_UNDERLINEHOT - アクティブ化されたホットアイテムのテキストに下線を表示します。
$fCoInit [オプション]COMライブラリを使用するためにスレッドを呼ぶことで初期化します

 

返し値

成功: リストビューコントロールのハンドル
失敗: 0

 

注意

この関数は上級者、コントロール動作学習用です。

 

関連

_GUICtrlListView_Destroy, _GUICtrlListView_SetBkImage

Global $hListView

_Main()

Func _Main()

    Local $GUI, $hImage
    $GUI = GUICreate("(UDF Created) ListView Create", 400, 300)

    $hListView = _GUICtrlListView_Create($GUI, "", 2, 2, 394, 268)
    _GUICtrlListView_SetExtendedListViewStyle($hListView, BitOR($LVS_EX_GRIDLINES, $LVS_EX_FULLROWSELECT, $LVS_EX_SUBITEMIMAGES))
    GUISetState()
   
    GUIRegisterMsg($WM_NOTIFY, "WM_NOTIFY")

    ; 画像をロード
    $hImage = _GUIImageList_Create()
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($hListView, 0xFF0000, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($hListView, 0x00FF00, 16, 16))
    _GUIImageList_Add($hImage, _GUICtrlListView_CreateSolidBitMap($hListView, 0x0000FF, 16, 16))
    _GUICtrlListView_SetImageList($hListView, $hImage, 1)

    ; 列を追加
    _GUICtrlListView_InsertColumn($hListView, 0, "Column 1", 100)
    _GUICtrlListView_InsertColumn($hListView, 1, "Column 2", 100)
    _GUICtrlListView_InsertColumn($hListView, 2, "Column 3", 100)

    ; アイテムを追加
    _GUICtrlListView_AddItem($hListView, "Row 1: Col 1", 0)
    _GUICtrlListView_AddSubItem($hListView, 0, "Row 1: Col 2", 1)
    _GUICtrlListView_AddSubItem($hListView, 0, "Row 1: Col 3", 2)
    _GUICtrlListView_AddItem($hListView, "Row 2: Col 1", 1)
    _GUICtrlListView_AddSubItem($hListView, 1, "Row 2: Col 2", 1)
    _GUICtrlListView_AddItem($hListView, "Row 3: Col 1", 2)

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
