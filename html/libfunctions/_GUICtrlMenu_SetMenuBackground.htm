Function Reference

_GUICtrlMenu_SetMenuBackground

メニューの背景ブラシを設定します。

#Include <GuiMenu.au3>
_GUICtrlMenu_SetMenuBackground($hMenu, $hBrush)

 

パラメータ

$hMenu メニューのハンドル
$hBrush 背景に使用するブラシ

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlMenu_GetMenuBackground

 

例


#include <GuiMenu.au3>
#include <WinAPI.au3>
#include <WindowsConstants.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hWnd, $hMain, $hFile, $hBrush

    ; メモ帳を開く
    Run("Notepad.exe")
    WinWaitActive("[CLASS:Notepad]")
    $hWnd = WinGetHandle("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($hWnd)
    $hFile = _GUICtrlMenu_GetItemSubMenu($hMain, 0)

    ; ファイルメニューの背景
    Writeln("File menu background: 0x" & Hex(_GUICtrlMenu_GetMenuBackground($hFile)))
    $hBrush = _WinAPI_GetSysColorBrush($COLOR_INFOBK)
    _GUICtrlMenu_SetMenuBackground($hFile, $hBrush)
    Writeln("File menu background: 0x" & Hex(_GUICtrlMenu_GetMenuBackground($hFile)))

EndFunc   ;==>_Main

; メモ帳にテキストを1行書き込む
Func Writeln($sText)
    ControlSend("[CLASS:Notepad]", "", "Edit1", $sText & @CR)
EndFunc   ;==>Writeln
