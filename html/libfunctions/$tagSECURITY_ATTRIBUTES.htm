Keyword Reference

$tagSECURITY_ATTRIBUTES

オブジェクト用のセキュリティ記述子を格納し、この構造体によって指定されたハンドルが継承可能かを指定します。

Global Const $tagSECURITY_ATTRIBUTES = "int Length;ptr Descriptor;int InheritHandle"

 

パラメータ

Length バイト単位でのこの構造体のサイズ
Descriptor オブジェクトの共有を制御するセキュリティ記述子へのポインタ
InheritHandle Trueの場合、新しいプロセスがハンドルを継承できます

 

注意

なし。
