Function Reference

_Security__OpenProcessToken

呼び出しプロセスのプロセスハンドルを返します。

#Include <Security.au3>
_Security__OpenProcessToken($hProcess, $iAccess)

 

パラメータ

$hProcess アクセストークンを開かれるプロセスのハンドル。
プロセスには$PROCESS_QUERY_INFORMATIONAアクセス許可が与えられていなければなりません。
$iAccess アクセストークンの要求アクセスタイプを指定するアクセスマスクを指定します。

 

返し値

成功: この関数が返った際に新しく開かれたアクセストークンを識別するハンドルへのポインタ。
失敗: 0

 

注意

返されるアクセストークンハンドルは_WinAPI_CloseHandle呼び出しによって閉じてください
