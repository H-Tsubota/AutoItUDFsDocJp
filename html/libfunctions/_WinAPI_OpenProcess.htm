Function Reference

_WinAPI_OpenProcess

既存のプロセスオブジェクトのハンドルを返します。

#Include <WinAPI.au3>
_WinAPI_OpenProcess($iAccess, $fInherit, $iProcessID [, $fDebugPriv = False])

 

パラメータ

$iAccess プロセスオブジェクトへのアクセスを指定します。
$fInherit 返されるハンドルが継承可能かを指定します。
$iProcessID 開くプロセスのプロセス識別子を指定します。
$fDebugPriv [オプション]特定のシステムプロセスはデバグセキュリティ権限が無いと開くことができません。
Trueの場合、この関数は標準のアクセス権限でプロセスを開けなかった場合にプロセスをデバグ権限で開くことを試みます。

 

返し値

成功: オブジェクトのプロセスハンドル
失敗: @errorを設定します。

 

注意

なし。
