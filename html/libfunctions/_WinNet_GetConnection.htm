Function Reference

_WinNet_GetConnection

ローカルデバイスに関連付けられたネットワークリソースの名前を取得します。

#Include <WinNet.au3>
_WinNet_GetConnection($sLocalName)

 

パラメータ

$sLocalName ネットワーク名を取得するローカルデバイスの名前

 

返し値

成功: 接続を確立するために使用されているリモート名
失敗: @errorを設定します。

 

注意

なし。
