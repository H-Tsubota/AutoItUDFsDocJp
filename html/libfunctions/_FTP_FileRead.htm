Function Reference

_FTP_FileRead

_FTP_FileOpen()によって開かれたハンドルからデータを読み込みます。

#Include <FTPEx.au3>
_FTP_FileRead($h_File, $dwNumberOfBytesToRead)

 

パラメータ

$h_File _FTP_FileOpen()によって返されたFTPファイルのハンドル
$dwNumberOfBytesToRead 読み込みバイト数

 

返し値

成功: 読み込んだバイナリ/文字列を返します。
失敗: 0を返し、@errorを設定します:
@error: ファイル終端では-1
その他のエラーでは非ゼロ

 

注意

なし。
