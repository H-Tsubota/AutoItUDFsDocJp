Function Reference

_PathMake

ドライブ、ディレクトリ、ファイル名、拡張子からパスを作成します。一部省略できます。

#Include <File.au3>
_PathMake($szDrive, $szDir, $szFName, $szExt)

 

パラメータ

$szDrive ドライブ(UNC可)。ドライブ文字の場合、: が自動的に付加されます。
$szDir ディレクトリ。末尾のスラッシュが見つからない場合、追加されます(頭のスラッシュは追加されません)
$szFName ファイルの名前
$szExt ファイルの拡張子。拡張子内にピリオドが見つからない場合、補完されます。

 

返し値

フルパスを格納した文字列を返します。

 

注意

パスは渡されたパラメータを元に作成されます。この関数は作成したパスの妥当性を調べないので、パスにあなたのファイルシステムで無効な文字が含まれる可能性があります。

 

関連

_PathFull, _PathSplit, DirCreate, FileChangeDir

 

例


#include <file.au3>
;
$TestPath = _PathMake( "c:", "\temp", "boot", "ini")
MsgBox(0,"demo _PathMake",$TestPath)
