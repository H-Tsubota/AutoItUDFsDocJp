Function Reference

_GUICtrlMonthCal_SetColor

カレンダーの指定された部分の色を設定します。

#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_SetColor($hWnd, $iIndex, $iColor)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex カレンダーのどの色を設定するかを表します:
$MCSC_BACKGROUND - 月の間に表示される背景色
$MCSC_TEXT - 月内のテキスト表示に使用される色
$MCSC_TITLEBK - カレンダータイトルの背景色
$MCSC_TITLETEXT - カレンダータイトル内のテキスト表示に使用される色
$MCSC_MONTHBK - 月内の背景色
$MCSC_TRAILINGTEXT - 初日と末日のテキスト表示に使用される色
$iColor 色値

 

返し値

成功: コントロールの指定された部分の以前の色設定
失敗: -1

 

注意

なし。

 

関連

_GUICtrlMonthCal_GetColor

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiMonthCal.au3>
#include <WindowsConstants.au3>
#include <Constants.au3>

Opt('MustDeclareVars', 1)

$Debug_MC = False ; MonthCal関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

Global $iMemo

_Main()

Func _Main()
    Local $hMonthCal

    ; GUIを作成
    GUICreate("Month Calendar Set Color", 400, 300)
    $hMonthCal = GUICtrlCreateMonthCal("", 4, 4, -1, -1, $WS_BORDER, 0x00000000)
   
    ; メモコントロールを作成
    $iMemo = GUICtrlCreateEdit("", 4, 168, 392, 128, 0)
    GUICtrlSetFont($iMemo, 9, 400, 0, "Courier New")
    GUISetState()

    ; カレンダーの色を取得/設定
    MemoWrite("Background: 0x" & Hex(_GUICtrlMonthCal_GetColor($hMonthCal, $MCSC_MONTHBK), 6))
    _GUICtrlMonthCal_SetColor($hMonthCal, $MCSC_MONTHBK, $CLR_MONEYGREEN)
    MemoWrite("Background: 0x" & Hex(_GUICtrlMonthCal_GetColor($hMonthCal, $MCSC_MONTHBK), 6))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main

; メモ帳にメッセージを書き込む
Func MemoWrite($sMessage)
    GUICtrlSetData($iMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
