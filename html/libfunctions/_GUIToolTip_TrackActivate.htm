Function Reference

_GUIToolTip_TrackActivate

トラッキングツールチップをアクティブ化/非アクティブ化します。

#Include <GuiToolTip.au3>
_GUIToolTip_TrackActivate($hWnd[, $fActivate = True[, $hTool = 0[, $iID = 0]]])

 

パラメータ

$hWnd コントロールのハンドル
$fActivate [オプション]Trueでアクティブ化、Falseで非アクティブ化
$hTool [オプション]ツールがあるウィンドウのハンドル
$iID [オプション]アプリケーション定義のツールの識別子

 

返し値

なし。

 

注意

なし。
