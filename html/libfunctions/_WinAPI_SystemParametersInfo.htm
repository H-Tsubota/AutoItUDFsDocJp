Function Reference

_WinAPI_SystemParametersInfo

システム全体のパラメータの1つの値を取得または設定します。

#Include <WinAPI.au3>
_WinAPI_SystemParametersInfo($iAction [, $iParam = 0 [, $vParam = 0 [, $iWinIni = 0]]])

 

パラメータ

$iAction 取得または設定されるシステム全体のパラメータ
$iParam [オプション]使用方法とフォーマットが要求または設定されるパラメータに依存するパラメータ
$vParam [オプション]使用方法とフォーマットが要求または設定されるパラメータに依存するパラメータ
$iWinIni [オプション]システムパラメータを設定する場合、ユーザープロファイルを更新するかどうか、
また更新する場合は $WM_SETTINGCHANGEメッセージをブロードキャスト送信するかどうかを指定します。
ユーザープロファイルを更新しない場合、このパラメータはゼロにできます。
それ以外の場合は次の値の組み合わせを使用できます:
$SPIF_UPDATEINIFILE - ユーザープロファイルに新しい設定を書き込みます
$SPIF_SENDCHANGE - ユーザープロファイルの更新後に$WM_SETTINGCHANGEメッセージをブロードキャスト送信します

 

返し値

成功: True
失敗: False

 

注意

なし。
