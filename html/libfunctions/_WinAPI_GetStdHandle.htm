Function Reference

_WinAPI_GetStdHandle

標準入力デバイス、標準出力デバイス、標準エラーデバイスのハンドルをハンドルを取得します。

#Include <WinAPI.au3>
_WinAPI_GetStdHandle($iStdHandle)

 

パラメータ

$iStdHandle ハンドルを返す標準デバイス。次の値のいずれかです:
0 - 標準入力デバイスのハンドル
1 - 標準出力デバイスのハンドル
2 - 標準エラーデバイスのハンドル

 

返し値

成功: 指定されたデバイスのハンドル
失敗: -1

 

注意

アプリケーションがSetStdHandleを使用して標準ハンドルの権限を制限していない場合、ハンドルにはGENERIC_READとGENERIC_WRITEのアクセス権限が設定されています。
アプリケーションが関連付けられた標準ハンドルを持っていない場合、返し値は0です。
