Function Reference

_WordDocSaveAs

指定したドキュメントを新しい名前、形式で保存します。

#include <Word.au3>
_WordDocSaveAs ( ByRef $o_object [, $s_FilePath = "" [, $i_Format = 0 [, $b_ReadOnlyRecommended = 0 [, $b_AddToRecentFiles = 0 [, $b_LockComments = 0 [, $s_Password = "" [, $s_WritePassword = ""]]]]]]] )

 

パラメータ

$o_object ドキュメントオブジェクトまたはWord.Applicationのオブジェクト変数
$s_FilePath [オプション] 保存するドキュメントのフルファイルパス。 (注意を参照)
"" = (デフォルト) ドキュメントが以前に保存されたことがない場合、デフォルトの名前(例えばDocument1.doc)が使用されます。
$i_Format [オプション] ドキュメントの保存形式
0 = (デフォルト) Microsoft Word形式
1 = Microsoft Word テンプレート形式
2 = Microsoft Windows テキスト形式
3 = 改行を維持したMicrosoft Windowsテキスト形式
4 = Microsoft DOSテキスト形式
5 = 改行を維持したMicrosoft DOSテキスト形式
6 = リッチテキスト形式 (RTF)
7 = Unicodeテキスト形式またはEncodedテキスト形式
8 = 標準HTML形式
9 = Webアーカイブ形式
10 = フィルタリング済みHTML形式
11 = 拡張マークアップ言語(XML)形式
$b_ReadOnlyRecommended [オプション] ドキュメントを開く際にMicrosoft Wordが読み取り専用を推奨するかどうか指定します。
0 = (デフォルト) 読み取り専用を推奨しません
1 = 読み取り専用を推奨します
$b_AddToRecentFiles [オプション] ファイル名をファイルメニュー下部の最近使用したファイルのリストに追加するかどうか指定します。
0 = (デフォルト) 追加しません
1 = 追加します
$b_LockComments [オプション] ドキュメントのコメントをロックするかどうか指定します。
0 = (デフォルト) コメントをロックしません
1 = コメントをロックします
$s_Password [オプション] ドキュメントを開くためのパスワード文字列。 (注意を参照)
$s_WritePassword [オプション] ドキュメントの変更を保存するためのパスワード文字列。 (注意を参照)

 

返し値

成功: 1を返します
失敗: 0を返し@ERRORを設定します
@Error: $_WordStatus_Success = 正常終了
$_WordStatus_GeneralError = 一般的なエラー
$_WordStatus_InvalidDataType = 無効なデータ型
$_WordStatus_InvalidObjectType = 無効なオブジェクト型
@Extended: 無効なパラメータの数を格納

 

注意

指定された名前のドキュメントが既に存在する場合、ドキュメントはユーザーに対する警告なしに上書きされます。
ファイルを保存できず、WordErrorHandlerが登録されていない場合、スクリプトは停止します。

アプリケーション内にハードコーディングされたパスワードを使用しないでください。パスワードが処理で必要な場合はユーザーにパスワードを求め、変数に保存し、その変数をコード内で使用してください。

 

関連

_WordDocAdd, _WordDocClose, _WordDocOpen, _WordDocSave

 

例


; *******************************************************
; 例 1 - 新しい空のドキュメントでWordウィンドウを作成。テキストを追加
;               保存操作を行い、終了
; *******************************************************
;
#include <Word.au3>
$oWordApp = _WordCreate ()
$oDoc = _WordDocGetCollection ($oWordApp, 0)
$oDoc.Range.Text = "This is some text to insert."
_WordDocSaveAs ($oDoc, @ScriptDir & "\Test.doc")
_WordQuit ($oWordApp)
