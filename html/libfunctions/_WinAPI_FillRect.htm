Function Reference

_WinAPI_FillRect

指定されたブラシを使用して長方形を塗りつぶします。

#Include <WinAPI.au3>
_WinAPI_FillRect($hDC, $ptrRect, $hBrush)

 

パラメータ

$hDC デバイスコンテキストのハンドル
$ptrRect 塗りつぶされる長方形のクライアント座標を格納している$tagRECT構造体へのポインタ
$hBrush 長方形を塗りつぶすのに使用されるブラシのハンドル

 

返し値

成功: True
失敗: False

 

注意

$hBrushパラメータで識別されるブラシは論理ブラシまたはカラー値のいずれかです。
論理ブラシのハンドルを指定すると_WinAPI_CreateSolidBrushを呼び出します。
また_WinAPI_GetStockObject関数を使用することで保存されているブラシの1つのハンドルを取得できます。
$hBrushパラメータにカラー値を指定すると標準のシステムカラーの1つになります(値1が選択色に追加される必要があります)。
