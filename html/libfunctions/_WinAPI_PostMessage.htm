Function Reference

_WinAPI_PostMessage

メッセージキューにメッセージを配置し、処理を返します。

#Include <WinAPI.au3>
_WinAPI_PostMessage($hWnd, $iMsg, $iwParam, $ilParam)

 

パラメータ

$hWnd メッセージを受け取るウィンドウプロシージャを持つウィンドウを識別します。
このパラメータが0xFFFF(HWND_BROADCAST)の場合、メッセージは無効、非表示、非所有、オーバーラップ、ポップアップウィンドウを含む
システムの全てのトップレベルウィンドウに送信されます。
ただしメッセージは子ウィンドウには送信されません。
$iMsg 送信されるメッセージを指定します。
$iwParam 1つめのメッセージパラメータ
$ilParam 2つめのメッセージパラメータ

 

返し値

成功: True
失敗: False

 

注意

なし。
