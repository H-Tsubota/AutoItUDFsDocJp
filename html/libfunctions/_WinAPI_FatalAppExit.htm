Function Reference

_WinAPI_FatalAppExit

メッセージボックスを表示し、アプリケーションを終了します。

#Include <WinAPI.au3>
_WinAPI_FatalAppExit($sMessage)

 

パラメータ

$sMessage メッセージボックスに表示される文字列

 

返し値

なし。

 

注意

他の方法で終了できない場合のみアプリケーションはFatalAppExitを呼びます。
FatalAppExitは常にアプリケーションのメモリを解放し、そのファイルを閉じることができるわけではなく、
その場合はシステムの一般的なエラーがおきます。
予期しないエラーが起きたアプリケーションは自身の全てのメモリを解放し、メインメッセージループから処理を返すことで終了します。
