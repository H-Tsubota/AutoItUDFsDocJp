Function Reference

_GDIPlus_GraphicsGetDC

Graphicsオブジェクトのデバイスコンテキストのハンドルを取得します。

#Include <GDIPlus.au3>
_GDIPlus_GraphicsGetDC($hGraphics)

 

パラメータ

$hGraphics Graphicsオブジェクトのハンドル

 

返し値

成功: GraphicsオブジェクトのDC
失敗: 0

 

注意

各_GDIPlus_GraphicsGetDCの呼び出しには対になる_GDIPlus_GraphicsReleaseDCの呼び出しが必要です。
それらの呼び出しの間ではGraphicsオブジェクトのメソッドの呼び出しをしないでください。
