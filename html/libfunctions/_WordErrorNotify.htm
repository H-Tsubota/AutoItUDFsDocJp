Function Reference

_WordErrorNotify

Word.au3が自動で警告とエラーを(コンソールに)通知するかどうかを指定します。

#include <Word.au3>
_WordErrorNotify ( [$f_notify = -1] )

 

パラメータ

$f_notify [オプション] 通知のオン・オフを指定します
-1 = (デフォルト) 現在の設定を返します
True = オンにします
False = オフにします

 

返し値

成功: $f_notify = -1の場合、現在の通知設定を返します。それ以外の場合、1を返します
失敗: 0を返します

 

注意

Word.au3はデフォルトで診断情報、警告、エラーをコンソールに書き出します。この情報はSciTeエディタを使用してスクリプトを実行すると読むことができます。

この関数でこの機能のオン・オフを切り替えられます。

 

関連

None.

 

例


; *******************************************************
; 例 1 - _WordErrorNotifyの現在の状態を調べ、オンの場合はオフに、オフの場合はオンにする
; *******************************************************
;
#include <Word.au3>
If _WordErrorNotify () Then
    MsgBox(0, "_WordErrorNotify Status", "Notification is ON, turning it OFF")
    _WordErrorNotify (1)
Else
    MsgBox(0, "_WordErrorNotify Status", "Notification is OFF, turning it ON")
    _WordErrorNotify (0)
EndIf
