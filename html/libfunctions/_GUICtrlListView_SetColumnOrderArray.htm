Function Reference

_GUICtrlListView_SetColumnOrderArray

コントロール内での列の左から右への順番を設定します。

#Include <GuiListView.au3>
_GUICtrlListView_SetColumnOrderArray($hWnd, $aOrder)

 

パラメータ

$hWnd コントロールのハンドル
$aOrder コントロール内での列のインデックス値を保持した整数値の配列。
配列は次のフォーマットでなくてはなりません:
[0] - 配列内のアイテムの数
[1] - 1番目の列のインデックス
[2] - 2番目の列のインデックス
[n] - 最後の列のインデックス

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlListView_GetColumnOrder, _GUICtrlListView_GetColumnOrderArray, _GUICtrlListView_SetColumnOrder

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiListView.au3>

Opt('MustDeclareVars', 1)

$Debug_LV = False ; ListView関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $a_order[5] = [4, 3, 2, 0, 1], $hListView

    GUICreate("ListView Set Column Order Array", 400, 300)
    $hListView = GUICtrlCreateListView("Column 1|Column 2|Column 3|Column 4", 2, 2, 394, 268)
    GUISetState()

    ; 列順を設定
    MsgBox(4160, "Information", "Changing column order")
    _GUICtrlListView_SetColumnOrderArray($hListView, $a_order)

    ; 列順を表示
    $a_order = _GUICtrlListView_GetColumnOrderArray($hListView)
    MsgBox(4160, "Information", StringFormat("Column order: [%d, %d, %d, %d]", $a_order[1], $a_order[2], $a_order[3], $a_order[4]))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    GUIDelete()
EndFunc   ;==>_Main
