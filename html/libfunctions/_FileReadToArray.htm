Function Reference

_FileReadToArray

指定したファイルを配列に読み込みます。

#Include <File.au3>
_FileReadToArray($sFilePath, ByRef $aArray)

 

パラメータ

$sFilePath 読み込むファイルのパスとファイル名。
$aArray ファイルの内容を保存する配列。

 

返し値

成功: 1
失敗: 0を返し@errorを設定します
@Error: 1 = 指定されたファイルが開けません
2 = ファイルを分割できません

 

注意

$aArray[0]には配列に読み込んだデータの数が保存されます。

 

関連

_FileWriteFromArray

 

例


#include <file.au3>
Dim $aRecords
If Not _FileReadToArray("error.log",$aRecords) Then
   MsgBox(4096,"Error", " Error reading log to Array     error:" & @error)
   Exit
EndIf
For $x = 1 to $aRecords[0]
    Msgbox(0,'Record:' & $x, $aRecords[$x])
Next
