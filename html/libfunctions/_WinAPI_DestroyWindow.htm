Function Reference

_WinAPI_DestroyWindow

指定されたウィンドウを破棄します。

#Include <WinAPI.au3>
_WinAPI_DestroyWindow($hWnd)

 

パラメータ

$hWnd 破棄されるウィンドウのハンドル

 

返し値

成功: True
失敗: False

 

注意

_WinAPI_DestroyWindowを使用して異なるスレッドによって作成されたウィンドウを破棄することはできません。

 

関連

_WinAPI_CreateWindowEx
