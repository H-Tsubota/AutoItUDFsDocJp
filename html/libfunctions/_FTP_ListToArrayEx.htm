Function Reference

_FTP_ListToArrayEx

現在のリモートディレクトリ内のファイル/ディレクトリの名前、サイズ、属性、時刻を取得します。

#Include <FTPEx.au3>
_FTP_ListToArrayEx($l_FTPSession [, $Return_Type = 0 [, $l_Flags = 0 [, $b_Fmt = 1 [, $l_Context]]]])

 

パラメータ

$l_FTPSession _FTP_Connect()の返し値
$Return_type [オプション] 0 = ファイルとディレクトリの両方、1 = ディレクトリ、2 = ファイル
$l_Flags [オプション] _FTP_FindFileFirst()を参照
$b_Fmt [オプション] 日付文字列の形式 : 1 = yyyy/mm/dd, 0 = mm/dd/yyyy.
$l_Context [オプション] このサーチと任意のアプリケーションデータを関連付けるアプリケーション定義の値を保持した変数。
ステータスコールバック関数を設定するためにアプリケーションが既に_FTP_SetStatusCallback()を呼び終わっている場合のみ使用できます。

 

返し値

成功: 2次元配列。注意を参照。
失敗: Array[0][0] = 0

 

注意

Array[0][0] = 見つかったエントリーの数

Array[x][0] ファイル名
Array[x][1] ファイルサイズ
Array[x][2] ファイル属性
Array[x][3] ファイル変更時刻
Array[x][4] ファイル作成時刻
Array[x][5] ファイルアクセス時刻

 

関連

_FTP_Connect, _FTP_SetStatusCallback

 

例


#include <FTPEx.au3>
#include <Array.au3>

$server = 'ftp.csx.cam.ac.uk'
$username = ''
$pass = ''

$Open = _FTP_Open('MyFTP Control')
$Conn = _FTP_Connect($Open, $server, $username, $pass)

$aFile = _FTP_ListToArrayEx($Conn, 0)
_ArrayDisplay($aFile)

$Ftpc = _FTP_Close($Open)
