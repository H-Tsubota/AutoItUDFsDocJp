Keyword Reference

$tagLVHITTESTINFO

ヒットテストの情報を格納します。

Global Const $tagLVHITTESTINFO = "int X;int Y;int Flags;int Item;int SubItem"

 

パラメータ

X ヒットテストのX位置
Y ヒットテストのY位置
Flags ヒットテストの結果。次の値の1つまたは組み合わせです:
$LVHT_ABOVE - コントロールクライアント領域より上です
$LVHT_BELOW - コントロールクライアント領域より下です
$LVHT_NOWHERE - クライアントウィンドウの内側ですが、リストアイテムには触れていません
$LVHT_ONITEMICON - アイテムのアイコン上です
$LVHT_ONITEMLABEL - アイテムのテキスト上です
$LVHT_ONITEMSTATEICON - アイテムの状態イメージ上です
$LVHT_TOLEFT - コントロールクライアント領域より左です
$LVHT_TORIGHT - コントロールクライアント領域より右です
Item 一致するアイテムのインデックスを受け取ります。またサブアイテムのヒットテストの場合、この値はサブアイテムの親アイテムを表します。
SubItem 一致するサブアイテムのインデックスを受け取ります。アイテムのヒットテストの場合、このメンバーはゼロになります。

 

注意

なし。
