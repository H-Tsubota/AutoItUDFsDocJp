Function Reference

_GUICtrlStatusBar_SetBkColor

背景色を設定します。

#Include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetBkColor($hWnd, $iColor)

 

パラメータ

$hWnd コントロールのハンドル
$iColor 新しい背景色。
CLR_DEFAULT値を設定するとステータスバーはデフォルトの背景色を使用します。

 

返し値

以前の背景色を返します。背景色がデフォルトカラーの場合、CLR_DEFAULTを返します。

 

注意

RGB COLORREFカラーが返されます。
また渡す色は16進数BGRカラーまたはRGB COLORREFでなければなりません。

 

関連

 

例


#include <GuiConstantsEx.au3>
#include <GuiStatusBar.au3>
#include <WinAPI.au3>
#include <Constants.au3>

Opt('MustDeclareVars', 1)

$Debug_SB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()

    Local $hGUI, $hStatus
    Local $aParts[3] = [75, 150, -1]
   
    ; GUIを作成
    $hGUI = GUICreate("(Example 1) StatusBar Set BkColor", 400, 300)
    $hStatus = _GUICtrlStatusBar_Create ($hGUI)
    GUISetState()

    ; パーツを設定
    _GUICtrlStatusBar_SetParts ($hStatus, $aParts)
    _GUICtrlStatusBar_SetText ($hStatus, "Part 1")
    _GUICtrlStatusBar_SetText ($hStatus, "Part 2", 1)

    ; 背景色を設定
    _GUICtrlStatusBar_SetBkColor ($hStatus, $CLR_MONEYGREEN)

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
