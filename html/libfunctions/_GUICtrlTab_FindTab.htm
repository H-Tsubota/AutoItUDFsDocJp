Function Reference

_GUICtrlTab_FindTab

特定のテキストのあるタブを検索します。

#Include <GuiTab.au3>
_GUICtrlTab_FindTab($hWnd, $sText[, $fInStr = False[, $iStart = 0]])

 

パラメータ

$hWnd コントロールのハンドル
$sText 検索テキスト
$fInStr [オプション]Trueの場合、タブテキストの任意位置からテキストを検索します
$iStart [オプション]検索開始位置のタブのゼロ始まりのインデックス

 

返し値

成功: 検索一致したゼロ始まりのタブインデックス
失敗: -1

 

注意

検索では大文字小文字は区別されません。

 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hTab

    ; GUIを作成
    GUICreate("Tab Control Find Tab", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; "Tab 2"を検索
    MsgBox(4160, "Information", '"Tab 2" at index ' & _GUICtrlTab_FindTab($hTab, "Tab 2"))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
