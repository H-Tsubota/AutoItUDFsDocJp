Function Reference

_GUICtrlTab_SetCurSel

タブを選択します。

#Include <GuiTab.au3>
_GUICtrlTab_SetCurSel($hWnd, $iIndex)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex ゼロ始まりのアイテムインデックス

 

返し値

成功: 以前に選択されていたタブのインデックス
失敗: -1

 

注意

この関数を使用してタブが選択された際はタブコントロールは$TCN_SELCHANGING通知メッセージ、$TCN_SELCHANGE通知メッセージを送信しません。

 

関連

_GUICtrlTab_GetCurSel

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hTab

    ; GUIを作成
    GUICreate("Tab Control Set Current Selected", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; 現在の選択を取得/設定
    _GUICtrlTab_SetCurSel($hTab, 1)
    MsgBox(4160, "Information", "Current Selection: " & _GUICtrlTab_GetCurSel($hTab))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
