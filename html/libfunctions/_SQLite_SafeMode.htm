Function Reference

_SQLite_SafeMode

SafeModeを有効または無効にします。

#include <SQLite.au3>
_SQLite_SafeMode ( $fSafeModeState )

 

パラメータ

$fSafeModeState True/FalseによってSafeModeを有効/無効にします

 

返し値

$SQLITE_OKを返します。

 

注意

SaveModeは不正なクエリやデータベースハンドルが渡されることによってSQLiteがクラッシュすることを防ぎます。
デフォルトでは有効です。

 

関連

なし。

 

例


#include <SQLite.au3>
#include <SQLite.dll.au3>

_SQLite_Startup()
ConsoleWrite("_SQLite_LibVersion=" &_SQLite_LibVersion() & @CRLF)
_SQLite_SafeMode(False)
_SQLite_Exec(-1,"CREATE tblTest (a,b,c);"); データベースを開いていない。SafeMode = Falseの場合、クラッシュします。
_SQLite_Shutdown()
