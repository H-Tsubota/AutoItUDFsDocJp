Function Reference

_GDIPlus_ImageGetHorizontalResolution

画像の水平解像度をDPI(1インチあたりのピクセル数)単位で返します。

#Include <GDIPlus.au3>
_GDIPlus_ImageGetHorizontalResolution($hImage)

 

パラメータ

$hImage Imageオブジェクトのハンドル

 

返し値

成功: DPI(1インチあたりのピクセル数)を表す整数
失敗: 0を返し、@errorが設定されます
@error: 0 - 正常終了
10 - 無効な画像ハンドル

 

注意

@errorが4のエラー時にはGDI+UDFは画像ハンドルの代わりに-1または0を返します。

 

関連

_GDIPlus_ImageGetVerticalResolution

 

こちらも参照

MSDNライブラリでGdipGetImageHorizontalResolutionを検索して下さい。

 

例


#AutoIt3Wrapper_Au3Check_Parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6

#include <GuiConstantsEx.au3>
#include <WindowsConstants.au3>
#include <GDIPlus.au3>
#include <ScreenCapture.au3>
#include <WinAPI.au3>


Opt('MustDeclareVars', 1)

Global $iMemo

_Main()

Func _Main()
    Local $hBitmap, $hImage

    ; GUI作成
    GUICreate("GDI+", 600, 400)
    $iMemo = GUICtrlCreateEdit("", 2, 2, 596, 396, $WS_VSCROLL)
    GUICtrlSetFont($iMemo, 9, 400, 0, "Courier New")
    GUISetState()

    ; GDI+ライブラリを初期化
    _GDIPlus_Startup()

    ; 32ビットのビットマップをキャプチャ
    $hBitmap = _ScreenCapture_Capture("")
    $hImage = _GDIPlus_BitmapCreateFromHBITMAP($hBitmap)

    ; 画像の水平解像度をDPI(1インチあたりのドット数)単位で表示
    MemoWrite("HorizontalResolution DPI : " & _GDIPlus_ImageGetHorizontalResolution($hImage));

    ; リソースを破棄
    _GDIPlus_ImageDispose($hImage)
    _WinAPI_DeleteObject($hBitmap)

    ; GDI+ライブラリを閉じる
    _GDIPlus_Shutdown()

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

EndFunc   ;==>_Main

; メモコントロールに1行書き込み
Func MemoWrite($sMessage = '')
    GUICtrlSetData($iMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
