_WinAPI_WideCharToMultiByte

Unicode文字列をマルチバイト文字列に変換します。

#Include <WinAPI.au3>
_WinAPI_WideCharToMultiByte($pUnicode [, $iCodePage = 0 [, $iFlags = 0 [, $bRetString = True]]])

 

パラメータ

$pUnicode 変換されるUnicodeテキストを格納した文字列、DllStruct、またはバイト配列構造体へのポインタ
$iCodePage [オプション]変換処理で使用するコードページ:
0 - 現在のシステムのWindows ANSIコードページ
1 - 現在のシステムのOEMコードページ
2 - 現在のシステムのMacintoshコードページ
3 - 現在のスレッドのWindows ANSIコードページ
42 - Symbolコードページ
65000 - UTF-7
65001 - UTF-8
$iFlags [オプション]変換タイプを表すフラグ。MSDNを参照して下さい。 (デフォルトは0)
$bRetString [オプション]文字列、DllStructのいずれを返すかを表すフラグ (デフォルトはTrue : 文字列)

 

返し値

マルチバイト文字列を格納した文字列またはDllStructを返します。

 

注意

なし。
