Function Reference

_GDIPlus_PenGetWidth

ペンの幅を取得します。

#Include <GDIPlus.au3>
_GDIPlus_PenGetWidth($hPen)

 

パラメータ

$hPen Penオブジェクトのハンドル

 

返し値

成功: ペンの幅
失敗: -1が返され@errorが設定されます

 

注意

なし。

 

関連

_GDIPlus_PenSetWidth

 

こちらも参照

MSDNライブラリでGdipGetPenWidthを検索して下さい。

 

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hGraphic, $hPen

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; 直線の描画
    _GDIPlus_Startup ()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)
    $hPen = _GDIPlus_PenCreate (0xFF000000, 8)
    _GDIPlus_PenSetWidth($hPen, 4)
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 10, 390, 10, $hPen)

    ; ペンの幅を表示
    MsgBox(4096, "Information", "Pen Width: " & _GDIPlus_PenGetWidth ($hPen))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; リソースを破棄
    _GDIPlus_PenDispose ($hPen)
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_ShutDown ()

EndFunc   ;==>_Main
