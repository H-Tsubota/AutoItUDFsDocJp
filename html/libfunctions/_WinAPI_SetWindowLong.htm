Function Reference

_WinAPI_SetWindowLong

指定されたウィンドウの情報を設定します。

#Include <WinAPI.au3>
_WinAPI_SetWindowLong($hWnd, $iIndex, $iValue)

 

パラメータ

$hWnd ウィンドウのハンドル
$iIndex 設定される値のゼロ始まりのオフセットを指定します。有効な値はゼロから拡張ウィンドウメモリ数より4小さい値までです。
例えば12バイト以上の拡張メモリを指定した場合、値8が3番目の32ビット整数のインデックスになります。
拡張ウィンドウメモリ以外の値を設定するためには次の値のいずれかを指定してください:
$GWL_EXSTYLE - 拡張ウィンドウスタイルを設定します
$GWL_STYLE - ウィンドウスタイルを設定します
$GWL_WNDPROC - ウィンドウプロシージャのアドレスを設定します
$GWL_HINSTANCE - アプリケーションインスタンスのハンドルを設定します
$GWL_HWNDPARENT - 存在する場合は親ウィンドウのハンドルを設定します
$GWL_ID - ウィンドウの識別子を設定します
$GWL_USERDATA - ウィンドウに関連付けられた32ビット値を設定します
$iValue 置換用の値を指定します。

 

返し値

成功: 以前の値
失敗: 0

 

注意

定義済み定数用にConstants.au3が必要です。

 

関連
