Function Reference

_GUICtrlMenu_SetItemState

メニューアイテムの状態を設定します。

#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemState($hMenu, $iItem, $iState[, $fState = True[, $fByPos = True]])

 

パラメータ

$hMenu メニューハンドル
$iItem メニューアイテムの位置または識別子
$iState メニューアイテムの状態:
$MFS_CHECKED - アイテムはチェックされています
$MFS_DEFAULT - アイテムはデフォルトアイテムです
$MFS_DISABLED - アイテムは無効です
$MFS_GRAYED - アイテムは無効です
$MFS_HILITE - アイテムはハイライトされています
$fState [オプション]アイテム状態の設定:
 True - 設定を有効にします
False - 設定を無効にします
$fByPos [オプション]メニュー識別子フラグ:
 True - $iItemはゼロ始まりのアイテム位置です
False - $iItemはメニューアイテムの識別子です

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlMenu_GetItemState, _GUICtrlMenu_GetItemStateEx

 

例


#include <GuiMenu.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hWnd, $hMain, $hFile

    ; メモ帳を開く
    Run("Notepad.exe")
    WinWaitActive("[CLASS:Notepad]")
    $hWnd = WinGetHandle("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($hWnd)
    $hFile = _GUICtrlMenu_GetItemSubMenu($hMain, 0)

    ; "開く"アイテムの状態を取得/設定
    _GUICtrlMenu_SetItemState($hFile, 1, $MFS_CHECKED)
    Writeln("Open item state: " & _GUICtrlMenu_GetItemStateEx($hFile, 1))

EndFunc   ;==>_Main

; メモ帳にテキストを1行書き込む
Func Writeln($sText)
    ControlSend("[CLASS:Notepad]", "", "Edit1", $sText & @CR)
EndFunc   ;==>Writeln
