Function Reference

_IEImgClick

画像のマウスクリックをシミュレートします。altテキスト、name、srcの部分文字列で指定します。

#include <IE.au3>
_IEImgClick ( ByRef $o_object, $s_linkText [, $s_mode = "src" [, $i_index = 0 [, $f_wait = 1]]] )

 

パラメータ

$o_object InternetExplorer.Applicationのオブジェクト変数、ウィンドウまたはフレームオブジェクト
$s_linkText $s_modeで指定された属性での画像指定テキスト
$s_mode [オプション]検索モードを指定します
src = (デフォルト) 画像のURLで検索します
name = 画像のnameで検索します
alt = 画像のaltテキストで検索します
$i_index [オプション]一致する画像が複数見つかった場合、どれを使用するか指定する0始まりのインデックス
$f_wait [オプション]ページのロード中に待機するかどうかを指定します
0 = 直ちに処理を返します。ページのロードを待ちません
1 = (デフォルト) ページのロードが完了してから処理を返します

 

返し値

成功: -1を返します
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
1 ($_IEStatus_GeneralError) = 一般的なエラー
3 ($_IEStatus_InvalidDataType) = 無効なデータ型
4 ($_IEStatus_InvalidObjectType) = 無効なオブジェクト型
5 ($_IEStatus_InvalidValue) = 無効な値
6 ($_IEStatus_LoadWaitTimeout) = ロード待機タイムアウト
7 ($_IEStatus_NoMatch) = 一致するものがありません
8 ($_IEStatus_AccessIsDenied) = アクセス拒否
9 ($_IEStatus_ClientDisconnected) = クライアント切断
@Extended: 無効なパラメータの番号が格納されています

 

注意

なし。

 

関連

_IEImgGetCollection, _IEFormImageClick, _IELoadWait

 

例


; *******************************************************
; 例 1 - AltテキストでIMGをクリック
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
_IEImgClick ($oIE, "AutoIt Homepage Image", "alt")

; *******************************************************
; 例 2 - nameでIMGをクリック
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
_IEImgClick ($oIE, "AutoItImage", "name")

; *******************************************************
; 例 3 - srcの部分文字列でIMGをクリック
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
_IEImgClick ($oIE, "autoit_6_240x100.jpg", "src")

; *******************************************************
; 例 4 - 全src文字列でIMGをクリック
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
_IEImgClick ($oIE, "http://www.autoitscript.com/images/autoit_6_240x100.jpg")
