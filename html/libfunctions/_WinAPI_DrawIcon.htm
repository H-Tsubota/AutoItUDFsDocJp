Function Reference

_WinAPI_DrawIcon

指定されたデバイスコンテキストにアイコンまたはカーソルを描画します。

#Include <WinAPI.au3>
_WinAPI_DrawIcon($hDC, $iX, $iY, $hIcon)

 

パラメータ

$hDC アイコンまたはカーソルが描画されるデバイスコンテキストのハンドル
$iX アイコンの左上隅のX座標
$iY アイコンの左上隅のY座標
$hIcon 描画されるアイコンのハンドル

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連
