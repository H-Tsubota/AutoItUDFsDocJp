Function Reference

_GUICtrlHeader_SetItemFormat

アイテムのフォーマットを設定します。

#Include <GuiHeader.au3>
_GUICtrlHeader_SetItemFormat($hWnd, $iIndex, $iFormat)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex ゼロ始まりのアイテムインデックス
$iFormat 次のフォーマット識別子の組み合わせです:
HDF_CENTER - アイテムの内容は中央揃えです
HDF_LEFT - アイテムの内容は左揃えです
HDF_RIGHT - アイテムの内容は右揃えです
HDF_BITMAP - アイテムはビットマップを表示します
HDF_BITMAP_ON_RIGHT - ビットマップがテキストの右に表示されます
HDF_OWNERDRAW - コントロールのオーナーがアイテムを描画します
HDF_STRING - アイテムは文字列を表示します
HDF_IMAGE - 画像リストからの画像を1つ表示します
HDF_RTLREADING - テキスト方向を親ウィンドウのテキストの逆方向にします
HDF_SORTDOWN - このアイテムに下矢印を描画します(Windows XP以降)
HDF_SORTUP - このアイテムに上矢印を描画します(Windows XP以降)

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_GUICtrlHeader_GetItemFormat

 

例


#include <GuiConstantsEx.au3>
#include <GuiHeader.au3>
#include <GuiImageList.au3>
#include <WinAPI.au3>

Opt('MustDeclareVars', 1)

$Debug_HDR = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

Global $iMemo

_Main()

Func _Main()
    Local $hGUI, $hHeader, $hImage

    ; GUIを作成
    $hGUI = GUICreate("Header", 400, 300)
    $hHeader = _GUICtrlHeader_Create ($hGUI)
    $iMemo = GUICtrlCreateEdit("", 2, 24, 396, 274, 0)
    GUICtrlSetFont($iMemo, 9, 400, 0, "Courier New")
    GUISetState()

    ; 画像から画像リストを作成
    $hImage = _GUIImageList_Create (11, 11)
    _GUIImageList_Add ($hImage, _WinAPI_CreateSolidBitmap ($hGUI, 0xFF0000, 11, 11))
    _GUIImageList_Add ($hImage, _WinAPI_CreateSolidBitmap ($hGUI, 0x00FF00, 11, 11))
    _GUIImageList_Add ($hImage, _WinAPI_CreateSolidBitmap ($hGUI, 0x0000FF, 11, 11))
    _GUICtrlHeader_SetImageList ($hHeader, $hImage)

    ; 列を追加
    _GUICtrlHeader_AddItem ($hHeader, "Column 1", 100, 0, 0)
    _GUICtrlHeader_AddItem ($hHeader, "Column 2", 100, 0, 1)
    _GUICtrlHeader_AddItem ($hHeader, "Column 3", 100, 0, 2)
    _GUICtrlHeader_AddItem ($hHeader, "Column 4", 100)

    ; 列1のフォーマットを設定
    _GUICtrlHeader_SetItemFormat ($hHeader, 0, BitOR($HDF_CENTER, $HDF_STRING))

    ; 列1のフォーマットを表示
    MemoWrite("Column 1 format: " & "0x" & Hex(_GUICtrlHeader_GetItemFormat ($hHeader, 0)))

    Sleep(1000)
   
    ; 列1のフォーマットをリセット
    _GUICtrlHeader_SetItemFormat ($hHeader, 0, BitOR($HDF_LEFT, $HDF_STRING, $HDF_IMAGE))

    ; 列1のフォーマットを表示
    MemoWrite("Column 1 format: " & "0x" & Hex(_GUICtrlHeader_GetItemFormat ($hHeader, 0)))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
EndFunc   ;==>_Main

; メモコントロールに1行書き込み
Func MemoWrite($sMessage)
    GUICtrlSetData($iMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
