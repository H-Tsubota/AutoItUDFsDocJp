Function Reference

_WinAPI_GetBkMode

指定されたデバイスコンテキストの現在の背景ミックスモードを返します。

#Include <WinAPI.au3>
_WinAPI_GetBkMode($hDC)

 

パラメータ

$hDC 背景モードが返されるデバイスコンテキストのハンドル

 

返し値

成功: OPAQUEまたはTRANSPARENTどちらかの現在の背景ミックスモードを指定する値が返されます。
失敗: 0

 

注意

デバイスコンテキストの背景ミックスモードはテキスト、ハッチブラシ、実線でないペンスタイルに影響を与えます。

 

関連

_WinAPI_SetBkMode, _WinAPI_DrawText, _WinAPI_CreatePen, _WinAPI_SelectObject

 

こちらも参照

MSDNライブラリでGetBkModeを検索して下さい。

 

例


#include <WinAPI.au3>
#include <WindowsConstants.au3>
#include <FontConstants.au3>

Global $tRECT, $hFont, $hOldFont, $hDC

HotKeySet("{ESC}", "_Exit")

$tRECT = DllStructCreate($tagRect)
DllStructSetData($tRECT, "Left", 5)
DllStructSetData($tRECT, "Top", 5)
DllStructSetData($tRECT, "Right", 250)
DllStructSetData($tRECT, "Bottom", 50)

$hDC = _WinAPI_GetDC(0)
$hFont = _WinAPI_CreateFont(50, 0, 0, 0, 400, False, False, False, $DEFAULT_CHARSET, _
        $OUT_DEFAULT_PRECIS, $CLIP_DEFAULT_PRECIS, $DEFAULT_QUALITY, 0, 'Arial')
$hOldFont = _WinAPI_SelectObject($hDC, $hFont)

_WinAPI_SetTextColor($hDC, 0x0000FF)
_WinAPI_SetBkColor($hDC, 0x000000)

MsgBox(4096, "Information", "GetBkMode: " & _WinAPI_GetBkMode($hDC))

; 次の行をコメント化すると背景色が透過色から黒色になる
_WinAPI_SetBkMode($hDC, $TRANSPARENT)

MsgBox(4096, "Information", "GetBkMode: " & _WinAPI_GetBkMode($hDC))

While 1
    _WinAPI_DrawText($hDC, "Hello world!", $tRECT, $DT_CENTER)
    Sleep(100)
WEnd

Func _Exit()
    _WinAPI_SelectObject($hDC, $hOldFont)
    _WinAPI_DeleteObject($hFont)
    _WinAPI_ReleaseDC(0, $hDC)
    _WinAPI_InvalidateRect(0, 0)
    $tRECT = 0
    Exit
EndFunc   ;==>_Exit
