Function Reference

_IEPropertySet

ブラウザ、DOM要素の選択されたプロパティを設定します。

#include <IE.au3>
_IEPropertySet ( ByRef $o_object, $s_property, $newvalue )

 

パラメータ

$o_object InternetExplorer.Applicationのオブジェクト変数
$s_property プロパティ選択 (注意を参照)
$newvalue ブラウザプロパティに設定する新しい値

 

返し値

成功: 1を返します
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
3 ($_IEStatus_InvalidDataType) = 無効なデータ型
4 ($_IEStatus_InvalidObjectType) = 無効なオブジェクト型
5 ($_IEStatus_InvalidValue) = 無効な値
@Extended: 無効なパラメータの番号が格納されています

 

注意

より詳しい情報はMSDNを参照。

次の表は使用できる各プロパティの説明です。DOM要素のものとブラウザのものがあります。

ブラウザプロパティ

プロパティ 説明
"addressbar" オブジェクトのアドレスバーの表示/非表示を指定します。
"contenteditable" オブジェクトがマウス、キーボードで編集できるかどうかを表すブール値を指定します。
"height" ブラウザメインウィンドウの高さを指定します。
"innerhtml" 要素の開始、終了タグを除く要素のレンダリングHTMLを設定します。
"innertext" 要素のレンダリングテキスト(タグを含まない)を設定します。通常、outertextと同じです。
"left" ブラウザメインウィンドウの左端のスクリーン座標を指定します。
"menubar" ブラウザメニューバーの表示/非表示を表すブール値を指定します。
"offline" ブラウザが現在オフラインモードで動作しているかどうかを表すブール値を指定します。
"outertext" 要素のレンダリングテキスト(タグを含まない)を設定します。通常、innertextと同じです。
"outerhtml" 要素の開始、終了タグを含む要素のレンダリングHTMLを設定します。
"resizable" オブジェクトがリサイズ可能かどうかを表すブール値を指定します。
"silent" ブラウザがダイアログボックスを表示可能かどうかを表すブール値を指定します。
"statusbar" ブラウザのステータスバーが表示されているかどうかを表す値を指定します。
"statustext" ブラウザのステータスバーのテキストを指定します。
"theatermode" ブラウザがシアターモードかどうかを表すブール値を指定します。シアターモードではブラウザのメインウィンドウがスクリーン全体を占め、最小限のナビゲーションボタンのみのツールバーが表示されます。
"title" ドキュメントタイトルを設定します。これはウィンドウタイトルとは異なります。通常、ウィンドウタイトルはドキュメントタイトルから始まりますが、Windowsレジストリで指定された追加テキストがそれに付け加えられます。
"toolbar" ブラウザツールバーの表示/非表示を表すブール値を指定します。
"top" ブラウザメインウィンドウの上端のスクリーン座標を指定します。
"width" ブラウザメインウィンドウの幅を指定します。

 

関連

_IEPropertyGet, _IEAction, _IEBodyWriteHTML, _IEDocInsertHTML, _IEDocInsertText

 

例


; *******************************************************
; 例 1 - 基礎的なサンプルをブラウザで開き、アドレスバーが表示されているかどうかを調べます。
;               表示されていない場合は表示します。
;               さらにステータスバーに表示されているテキストを変更します。
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
If Not _IEPropertyGet ($oIE, "statusbar") Then _IEPropertySet ($oIE, "statusbar", True)
_IEPropertySet ($oIE, "statustext", "Look What I can Do")
Sleep(1000)
_IEPropertySet ($oIE, "statustext", "I can change the status text")
