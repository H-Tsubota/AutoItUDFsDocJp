Function Reference

_GDIPlus_Startup

Microsoft Windows GDI+を初期化します。

#Include <GDIPlus.au3>
_GDIPlus_Startup()

 

パラメータ

なし。

 

返し値

成功: True
失敗: False

 

注意

GDI+オブジェクトを作成する前に_GDIPlus_Startupを呼んでください。
GDI+はMicrosoft Windows NT 4.0 SP6、Windows 2000、Windows 98、Windows Meのオペレーティングシステムでは
アプリケーション用の再配布を必要とします。

 

関連

_GDIPlus_Shutdown

 

こちらも参照

MSDNライブラリでGdiplusStartupを検索して下さい。

 

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>
#include <ScreenCapture.au3>
#include <WinAPI.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hBMP, $hBitmap, $hGraphic

    ; スクリーンの左上隅をキャプチャ
    $hBMP = _ScreenCapture_Capture ("", 0, 0, 400, 300)

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; GDI+ライブラリを初期化
    _GDIPlus_Startup ()

    ; ビットマップをGUIに描画
    $hBitmap = _GDIPlus_BitmapCreateFromHBITMAP ($hBMP)
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)
    _GDIPlus_GraphicsDrawImage ($hGraphic, $hBitmap, 0, 0)

    ; リソースを破棄
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_ImageDispose ($hBitmap)
    _WinAPI_DeleteObject ($hBMP)

    ; GDI+ライブラリを閉じる
    _GDIPlus_ShutDown ()

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE


EndFunc   ;==>_Main
