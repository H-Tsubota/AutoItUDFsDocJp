Keyword Reference

$tagTBBUTTON

ツールバーのボタンの情報を格納します。

Global Const $tagTBBUTTON = "int Bitmap;int Command;byte State;byte Style;short Reserved;int Param;int String"

 

パラメータ

Bitmap ボタン画像のゼロ始まりのインデックス。このメンバーを$I_IMAGECALLBACKに設定すると、必要な際に画像インデックスを取得するためにツールバーは$TBN_GETDISPINFO通知を送信します。
ボタンが画像を持たないことを表す場合はこのメンバーを$I_IMAGENONEに設定します。ボタンのレイアウトはテキストのみでビットマップ用スペースを含みません。
ボタンがセパレータの場合、Bitmapはピクセル単位でセパレータの幅を決定します。
Command ボタンに関連付けられたコマンド識別子。この識別子はボタンが選択された際に$WM_COMMANDメッセージで使用されます。
State ボタン状態フラグ
Style ボタンスタイルフラグ
Reserved 予約済み
Param アプリケーション定義値
String ボタン文字列、またはボタン用テキストが格納された文字列へのポインタのゼロ始まりのインデックス

 

注意

なし。
