Function Reference

_WinAPI_ShowCursor

カーソルの表示/非表示を切り替えます。

#Include <WinAPI.au3>
_WinAPI_ShowCursor($fShow)

 

パラメータ

$fShow Trueの場合、カーソルは表示され、それ以外の場合は非表示になります

 

返し値

成功: 新しい表示カウンター
失敗: 0

 

注意

この関数はカーソルが表示されるかどうかを判定する内部の表示カウンターを設定します。
カーソルは表示カウントが0以上の場合のみ表示されます。
マウスがインストールされた場合、表示カウントの初期値は0です。
マウスがインストールされていない場合、表示カウントは-1です。
