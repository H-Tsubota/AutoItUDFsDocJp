Function Reference

_IEErrorNotify

IE.au3が警告とエラーを(コンソールに)自動通知するかどうかを指定します。

#include <IE.au3>
_IEErrorNotify ( [$f_notify = -1] )

 

パラメータ

$f_notify [オプション]通知のオン、オフを指定します
-1 = (デフォルト) 現在の設定を返します
True = オン
False = オフ

 

返し値

成功: $f_notify = -1の場合、現在の通知設定が返されます。それ以外の場合、1が返されます。
失敗: 0が返されます。

 

注意

IE.au3はデフォルトでコンソールに警告とエラーの診断情報を書きます。SciTeエディタを使ってスクリプトの実行をおこなうとこの情報は簡単に見ることができます。

この関数でこの機能のオン、オフをおこなえます。

 

関連

_IEErrorHandlerRegister

 

例


; *******************************************************
; 例 1 - _IEErrorNotifyの現在の状態を調べ、オンならばオフ、オフならばオンにする
; *******************************************************
;
#include <IE.au3>
If _IEErrorNotify () Then
    MsgBox(0, "_IEErrorNotify Status", "Notification is ON, turning it OFF")
    _IEErrorNotify (False)
Else
    MsgBox(0, "_IEErrorNotify Status", "Notification is OFF, turning it ON")
    _IEErrorNotify (True)
EndIf
