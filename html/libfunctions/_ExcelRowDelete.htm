Function Reference

_ExcelRowDelete

アクティブなワークシートから行を削除します。

#Include <Excel.au3>
_ExcelRowDelete($oExcel, $iRow [, $iNumRows = 1])

 

パラメータ

$oExcel _ExcelBookOpen()か_ExcelBookNew()の呼び出しで開かれているExcelオブジェクト
$iRow 削除する行の行番号指定
$iNumRows [オプション]削除する行数

 

返し値

成功: 1
失敗: 0を返し@errorに値を設定します
@error: 0 - 正常終了
1 - 指定されたオブジェクトが存在しません
2 - 指定された行が無効です

 

注意

この関数は削除した行以降の全ての行を上側にシフトします

 

関連

 

例


; ***************************************************************
; 例 1 - ワークブックを開きオブジェクト識別子取得後、ループを使ってセルに書き込む。 1行削除し、保存して閉じる。
; *****************************************************************
#include <Excel.au3>

Local $oExcel = _ExcelBookNew() ;新しいブックを作成して表示

For $i = 1 To 5 ;ループ
    _ExcelWriteCell($oExcel, $i, $i, 1) ;1から5の値を使って垂直方向にセルに書き込む
Next

ToolTip("Deleting Rows Soon...")
Sleep(3500)

_ExcelRowDelete($oExcel, 1, 1) ;1行目から1行だけ削除

MsgBox(0, "Exiting", "Press OK to Save File and Exit")
_ExcelBookSaveAs($oExcel, @TempDir & "\Temp.xls", "xls", 0, 1) ; 上書き設定でtempディレクトリに保存
_ExcelBookClose($oExcel) ; 閉じて終わり

; ***************************************************************
; 例 2 - ワークブックを開きオブジェクト識別子取得後、ループを使ってセルに書き込む。 数行削除し、保存して閉じる。
; *****************************************************************

#include <Excel.au3>

Local $oExcel = _ExcelBookNew() ; 新しいブックを開き、表示する

For $i = 1 To 5 ;ループ
    _ExcelWriteCell($oExcel, $i, $i, 1) ;1から5の値を使って垂直方向にセルに書き込む
Next

ToolTip("Deleting Rows Soon...")
Sleep(3500)

_ExcelRowDelete($oExcel, 3, 2) ;3行目から2行削除

MsgBox(0, "Exiting", "Press OK to Save File and Exit")
_ExcelBookSaveAs($oExcel, @TempDir & "\Temp.xls", "xls", 0, 1) ; 上書き設定でtempディレクトリに保存
_ExcelBookClose($oExcel) ; 閉じて終わり
