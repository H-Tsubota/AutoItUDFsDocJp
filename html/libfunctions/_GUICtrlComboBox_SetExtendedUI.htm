Function Reference

_GUICtrlComboBox_SetExtendedUI

デフォルトユーザーインターフェイスまたは拡張ユーザーインターフェイスのどちらかを選択します。

#Include <GuiComboBox.au3>
_GUICtrlComboBox_SetExtendedUI($hWnd[, $fExtended = False])

 

パラメータ

$hWnd コントロールのハンドル
$fExtended [オプション]コンボボックスが拡張を使用するかどうかを指定します。

 

返し値

成功: True
失敗: False

 

注意

デフォルトではF4キーでリストの開閉ができ、下矢印で現在の選択を変更できます。

拡張ユーザーインターフェイスではF4キーは無効になり、下矢印キーでドロップダウンリストが開けるようになります。

 

関連

_GUICtrlComboBox_GetExtendedUI

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIComboBox.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_CB = False ; ComboBox/ComboBoxEx関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

Global $iMemo

_Main()

Func _Main()
    Local $hCombo

    ; GUIを作成
    GUICreate("ComboBox Set Extended UI", 400, 296)
    $hCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    $iMemo = GUICtrlCreateEdit("", 2, 32, 396, 266, 0)
    GUICtrlSetFont($iMemo, 9, 400, 0, "Courier New")
    GUISetState()

    ; ファイルを追加
    _GUICtrlComboBox_BeginUpdate($hCombo)
    _GUICtrlComboBox_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($hCombo)

    ; 拡張UIを取得
    MemoWrite("Extended UI: " & _GUICtrlComboBox_GetExtendedUI($hCombo))
   
    ; 拡張UIを設定
    _GUICtrlComboBox_SetExtendedUI($hCombo, True)
   
    ; 拡張UIを取得
    MemoWrite("Extended UI: " & _GUICtrlComboBox_GetExtendedUI($hCombo))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main

; メモコントロールに1行書き込み
Func MemoWrite($sMessage)
    GUICtrlSetData($iMemo, $sMessage & @CRLF, 1)
EndFunc   ;==>MemoWrite
