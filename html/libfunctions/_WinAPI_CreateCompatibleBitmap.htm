Function Reference

_WinAPI_CreateCompatibleBitmap

指定されたデバイスコンテキストと互換性のあるビットマップを作成します。

#Include <WinAPI.au3>
_WinAPI_CreateCompatibleBitmap($hDC, $iWidth, $iHeight)

 

パラメータ

$hDC デバイスコンテキストを識別します。
$iWidth ビットマップの幅をピクセル単位で指定します。
$iHeight ビットマップの高さをピクセル単位で指定します。

 

返し値

成功: ビットマップのハンドル
失敗: 0

 

注意

ビットマップが不要になった場合は_WinAPI_DeleteObject関数を呼んで削除してください。

 

関連

_WinAPI_DeleteObject, _WinAPI_CreateSolidBitmap
