Function Reference

_WinNet_GetUser

デフォルトユーザー名、または接続の確立に使用されるユーザー名を取得します。

#Include <WinNet.au3>
_WinNet_GetUser($sName)

 

パラメータ

$sName ネットワークリソースへリダイレクトされているローカルデバイス名、またはローカルデバイスのリダイレクトを行うことなく接続を確立したネットワークリソースのリモート名のいずれか。
空白の場合、システムはプロセスの現在のユーザーの名前を返します。

 

返し値

成功: ユーザー名
失敗: @errorを設定します。

 

注意

なし。
