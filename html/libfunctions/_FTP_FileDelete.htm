Function Reference

_FTP_FileDelete

FTPサーバー上からファイルを削除します。

#Include <FTPEx.au3>
_FTP_FileDelete($l_FTPSession, $s_RemoteFile)

 

パラメータ

$l_FTPSession _FTP_Connect()の返し値
$s_RemoteFile ファイルのリモート位置

 

返し値

成功: 1
失敗: 0を返し、@errorを設定します。

 

注意

なし。
