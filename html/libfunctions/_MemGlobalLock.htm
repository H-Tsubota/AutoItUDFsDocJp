Function Reference

_MemGlobalLock

グローバルメモリオブジェクトをロックし、オブジェクトのメモリブロックの最初のバイトを指すポインタを返します。

#Include <Memory.au3>
_MemGlobalLock($hMem)

 

パラメータ

$hMem グローバルメモリオブジェクトのハンドル

 

返し値

成功: オブジェクトのメモリブロックの最初のバイトを指すポインタ
失敗: 0

 

注意

なし。

 

関連

_MemGlobalUnlock, _MemGlobalAlloc
