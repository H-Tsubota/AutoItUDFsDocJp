Function Reference

_IE_Introduction

IE.au3に関する入門的な情報を新しいブラウザウィンドウに表示します。

#include <IE.au3>
_IE_Introduction ( [$s_module = "basic"] )

 

パラメータ

$s_module [オプション]どのモジュールを実行するかを指定します
basic = (デフォルト) 基礎的な導入

 

返し値

成功: InternetExplorer.Applicationオブジェクトを指すオブジェクト変数を返します
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
5 ($_IEStatus_InvalidValue) = 無効な値
@Extended: 無効なパラメータの番号が格納されています

 

注意

AutoItのヘルプファイルは現在、UDFドキュメンテーションが個々の関数について取り上げることを認める一方で、一連の関数をカバーする一般的なドキュメントを持つことを許可していません。この関数はそのような一般的なドキュメントを提供するためのもので、今後追加されていきます。

 

関連

_IE_Example

 

例


; *******************************************************
; 例 1 - 基礎的なIE.au3入門のテキストを表示しているブラウザを開く
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Introduction ("basic")
