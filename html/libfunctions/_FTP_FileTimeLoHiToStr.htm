Function Reference

_FTP_FileTimeLoHiToStr

_FTP_Openのセッションを閉じます。

#Include <FTPEx.au3>
_FTP_FileTimeLoHiToStr($LoDWORD, $HiDWORD [, $bFmt = 0])

 

パラメータ

$LoDWORD ファイル時刻の下位ワード
$HiDWORD ファイル時刻の上位ワード
$bFmt [オプション] 0 の場合、mm/dd/yyyy hh:mm:ssを返します (デフォルト)。
1 の場合、yyyy/mm/dd hh:mm:ssを返します。

 

返し値

成功: $bFmtに従ったDateTime構造体
失敗: "" (空文字列)

 

注意

なし。
