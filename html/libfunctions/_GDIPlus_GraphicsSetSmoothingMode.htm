Function Reference

_GDIPlus_GraphicsSetSmoothingMode

Graphicsオブジェクトの描画品質を設定します。

#Include <GDIPlus.au3>
_GDIPlus_GraphicsSetSmoothingMode($hGraphics, $iSmooth)

 

パラメータ

$hGraphics Graphicsオブジェクトのハンドル
$iSmooth スムージングモード:
0 - スムージングを適用しません
1 - 8X4のボックスフィルターを使用してスムージングを適用します
2 - 8X8のボックスフィルターを使用してスムージングを適用します

 

返し値

成功: True
失敗: False

 

注意

なし。
