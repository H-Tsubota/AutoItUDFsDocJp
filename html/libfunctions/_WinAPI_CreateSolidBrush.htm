Function Reference

_WinAPI_CreateSolidBrush

指定されたソリッドカラーの論理ブラシを作成します。

#Include <WinAPI.au3>
_WinAPI_CreateSolidBrush($nColor)

 

パラメータ

$nColor ブラシの色を指定します

 

返し値

成功: 論理ブラシを識別するためのHBRUSH値
失敗: 0

 

注意

HBRUSHオブジェクトが不要になった際は_WinAPI_DeleteObjectを呼んでオブジェクトを削除してください。
