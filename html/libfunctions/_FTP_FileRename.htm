Function Reference

_FTP_FileRename

FTPサーバー上のファイルの名称を変更します。

#Include <FTPEx.au3>
_FTP_FileRename($l_FTPSession, $s_Existing, $s_New)

 

パラメータ

$l_FTPSession _FTP_Connect()の返し値
$s_Existing 古いファイル名
$s_New 新しい、ファイル名

 

返し値

成功: 1
失敗: 0を返し、@errorに非ゼロを設定します。

 

注意

なし。

 

関連
