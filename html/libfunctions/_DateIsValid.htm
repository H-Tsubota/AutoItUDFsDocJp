Function Reference

_DateIsValid

与えられた日付けが有効な日付けかどうか調べます。

#Include <Date.au3>
_DateIsValid($sDate)

 

パラメータ

$sDate 調べる日付け。

 

返し値

成功: 1を変えします
失敗: 指定した日付けが有効でない場合、0を返します

 

注意

この関数では次の形式の日付け入力が有効です
"yyyy/mm/dd[ hh:mm[:ss]]"または"yyyy/mm/dd[Thh:mm[:ss]]"
"yyyy-mm-dd[ hh:mm[:ss]]"または"yyyy-mm-dd[Thh:mm[:ss]]"
"yyyy.mm.dd[ hh:mm[:ss]]"または"yyyy.mm.dd[Thh:mm[:ss]]"

 

関連

 

例


#include <Date.au3>

$sDate =  @YEAR & "/" & @MON & "/" & @MDAY

If _DateIsValid( $sDate ) Then
  MsgBox( 4096, "Valid Date", "The specified date is valid." )
Else
  MsgBox( 4096, "Valid Date", "The specified date is invalid." )
EndIf
