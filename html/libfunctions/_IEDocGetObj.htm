Function Reference

_IEDocGetObj

任意のDOMオブジェクトを与えると、関連付けられたドキュメントオブジェクトへの参照が返されます。

#include <IE.au3>
_IEDocGetObj ( ByRef $o_object )

 

パラメータ

$o_object InternetExplorer.Applicationのオブジェクト変数、ウィンドウ、フレームまたは任意のDOMオブジェクト

 

返し値

成功: ドキュメントオブジェクトをポイントするオブジェクト変数が返されます
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
3 ($_IEStatus_InvalidDataType) = 無効なデータ型
@Extended: 無効なパラメータの番号が格納されています

 

注意

任意のDOMオブジェクト(e.g. ブラウザ、ウィンドウ、フレーム、アイフレーム、ドキュメント、ドキュメントエレメント)を与えられると、この関数は関連付けられたドキュメントオブジェクトへの参照を返します。渡されるオブジェクトのタイプをコントロールできない状況で一般的な形式で処理を書く際、特に便利です。

 

関連

None.

 

例


; *******************************************************
; Example 1 - AutoItのホームページをブラウザで開き、ドキュメントオブジェクトへの参照を取得後、
;               ドキュメントの属性情報を表示
; *******************************************************
;
#include <IE.au3>
$oIE = _IECreate ("http://www.autoitscript.com")
$oDoc = _IEDocGetObj ($oIE)
MsgBox(0, "Document Created Date", $oDoc.fileCreatedDate)
