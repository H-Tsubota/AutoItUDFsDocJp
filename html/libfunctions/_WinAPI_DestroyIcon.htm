Function Reference

_WinAPI_DestroyIcon

アイコンを破棄し、アイコンによって使用されていたメモリを解放します。

#Include <WinAPI.au3>
_WinAPI_DestroyIcon($hIcon)

 

パラメータ

$hIcon 破棄されるアイコンのハンドル。アイコンは使用中であってはなりません。

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_WinAPI_CopyIcon, _WinAPI_LoadShell32Icon
