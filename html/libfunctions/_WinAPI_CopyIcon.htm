Function Reference

_WinAPI_CopyIcon

他のモジュールから指定されたアイコンをコピーします。

#Include <WinAPI.au3>
_WinAPI_CopyIcon($hIcon)

 

パラメータ

$hIcon コピーされるアイコンのハンドル

 

返し値

成功: 2つめのアイコンのハンドル
失敗: 0

 

注意

CopyIcon関数によってアプリケーション、DLLは他のモジュールにあるアイコンから自分用のハンドルを取得することができます。
アイコンを所有する他のモジュールが解放された場合でもアプリケーションはアイコンを使用することができます。
閉じる前には_WinAPI_DestroyIconを呼んでそのアイコンに関連する全てのシステムリソースを解放してください。
