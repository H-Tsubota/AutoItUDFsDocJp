Function Reference

_WinAPI_GetGuiResources

指定されたプロセスによって利用されているグラフィカルユーザーインターフェイス(GUI)オブジェクトのハンドルの数を取得します。

#Include <WinAPI.au3>
_WinAPI_GetGuiResources([$iflag = 1 [, $hProcess = -1}})

 

パラメータ

$iflag [オプション] 0 (デフォルト) GDIオブジェクトの数を返します。
1 USERオブジェクトの数を返します。
$hProcess [オプション] プロセスのハンドル。デフォルトでは現在のプロセスです。

 

返し値

成功: $iFlagに従って、指定されたプロセスによって利用されているGUIオブジェクトのハンドルの数を返します。
失敗: @errorを設定します

 

注意

なし。
