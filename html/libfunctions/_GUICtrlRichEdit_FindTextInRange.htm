Function Reference

_GUICtrlRichEdit_FindTextInRange

範囲内のテキストを検索します。

#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_FindTextInRange($hWnd, $sText [, $iStart = 0 [, $iEnd = -1 [, $fMatchCase = False [, $fwholeWord = False [, $iBehavior = 0]]]]])

 

パラメータ

$hWnd コントロールのハンドル
$sText 検索テキスト
$iStart [オプション] 検索開始位置
デフォルト: コントロールの冒頭
$iEnd [オプション] 検索終了位置
デフォルト: コントロールの末尾
$fMatchCase [オプション] 検索での大文字小文字の区別
デフォルト: 区別しません。
$fWholeWord [オプション] 単語単位での一致のみ検索
デフォルト: 部分または語全体
$iBehavior [オプション] $FR_MATCHALEFHAMZA、$FR_MATCHDIAC、$FR_MATCHKASHIDAの任意のBitOrの組み合わせ
デフォルト: 0

 

返し値

成功: 値を保持した要素数2の配列
検索テキストが見つかった場合、一致したテキストの文字範囲を返します。例: [45, 52]
見つからなかった場合、[-1, -1]を返します。
失敗: 0を返し、@errorを設定します。
@error: 101 - $hWndがハンドルではありません。
102 - $sText = ""
103 - $iStartが正の数でも非ゼロでも-1でもありません。
104 - $iEndが正の数でも非ゼロでも-1でもありません。
105 - $fMatchCaseはTrueまたはFalseでなければなりません。
106 - $fwholeWordはTrueまたはFalseでなければなりません。
1071 - $iBehaviorが数値ではありません。
1072 - $iBehaviorが不正です。

 

注意

コントロールの冒頭の文字間位置は0です。
デフォルトの検索範囲は0から-1で、全テキストを末尾に向かって検索します。
$iEndを-1に設定するとコントロール末尾に向かって検索します。
$iStartを-1に設定するとコントロール冒頭に向かって検索します。

例


#AutoIt3Wrapper_Au3Check_Parameters= -d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiRichEdit.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Opt('MustDeclareVars', 1)

Global $lblMsg, $hRichEdit

Main()

Func Main()
    Local $hGui, $iMsg, $ai
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName,4) &")", 320, 350, -1, -1)
    $hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $lblMsg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    GUISetState()

    _GuiCtrlRichEdit_AppendText($hRichEdit, @CR & "This is appended text.")

    $ai = _GUICtrlRichEdit_FindTextinRange($hRichEdit, "test", 20, 6)
    Report('"test" found between inter-character positions ' & $ai[0] & " and " & $ai[1])

    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                GUIDelete()
                Exit
        EndSelect
    WEnd
EndFunc   ;==>Main

Func Report($sMsg)
    GUICtrlSetData($lblMsg, $sMsg)
EndFunc   ;==>Report
