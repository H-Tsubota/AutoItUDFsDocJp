Function Reference

_Security__GetTokenInformation

指定した種類のアクセストークンの情報を取得します。

#Include <Security.au3>
_Security__GetTokenInformation($hToken, $iClass)

 

パラメータ

$hToken 情報を取得するアクセストークンのハンドル。
$iClassに$sTokenSourceを指定する場合、ハンドルは$TOKEN_QUERY_SOURCEアクセスでなければなりません。
それ以外の$iClass値の場合、ハンドルは$TOKEN_QUERYアクセスでなければなりません。
$iClass 関数が取得する情報の種類を識別するための値を指定します。

 

返し値

成功: 要求した情報を格納したバイト構造体
失敗: 0

 

注意

なし。
