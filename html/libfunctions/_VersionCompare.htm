Function Reference

_VersionCompare

2つのファイルのバージョンが同じかどうか比較します。

#Include <Misc.au3>
_VersionCompare($sVersion1, $sVersion2)

 

パラメータ

$sVersion1 IN - 1つめのバージョン
$sVersion2 IN - 2つめのバージョン

 

返し値

成功: 次の値を返します:
0 - 2つのバージョンは同じです
1 - バージョン1の方が大きいです
-1 - バージョン2の方が大きいです
失敗: @errorに致命的なエラーイベントでの値が設定されます

 

注意

この関数はまず数値的比較を試みますが代替的に辞書的な比較も試みます。
どちらがおこなわれたかの詳細については@extendedを参照してください。
