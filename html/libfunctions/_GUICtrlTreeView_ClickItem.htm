Function Reference

_GUICtrlTreeView_ClickItem

アイテムをクリックします。

#Include <GuiTreeView.au3>
_GUICtrlTreeView_ClickItem($hWnd, $hItem[, $sButton = "left"[, $fMove = False[, $iClicks = 1[, $iSpeed = 0]]]])

 

パラメータ

$hWnd コントロールのハンドル
$hItem アイテムのハンドル
$sButton [オプション]クリックするボタン
$fMove [オプション]Trueの場合はマウスが動き、Falseの場合はマウスが動きません
$iClicks [オプション]クリック回数
$iSpeed [オプション]マウス動作のスピード

 

返し値

成功: 1
失敗: 0 を返し@errorを設定します。

 

注意

なし。

 

関連
