Function Reference

_WordDocLinkGetCollection

ドキュメント内の全てのリンクを格納したコレクションオブジェクトを返します。

#include <Word.au3>
_WordDocLinkGetCollection ( ByRef $o_object [, $i_index = -1] )

 

パラメータ

$o_object Word.Application、ドキュメントオブジェクトのオブジェクト変数
$i_index [オプション] コレクションを返すか、インデックスされたインスタンスを返すかを指定します。
正の整数の場合、インデックスされたインスタンスが返されます(1始まり)
-1 = (デフォルト) コレクションを返します

 

返し値

成功: ドキュメント内の全てのリンクのオブジェクトコレクションを返します。@EXTENDED = リンク数
失敗: 0を返し@ERROR = 1を設定します
@Error: $_WordStatus_Success = 正常終了
$_WordStatus_InvalidDataType = 無効なデータ型
$_WordStatus_InvalidValue = 無効な値
$_WordStatus_NoMatch = 一致なし
@Extended: 無効なパラメータの数を格納

 

注意

なし。

 

関連

None.

 

例


; *******************************************************
; 例 1 - 新しい空のドキュメントでWordウィンドウを作成、
;               リンクを追加し、リンクのコレクションを取得
; *******************************************************
;
#include <Word.au3>
$oWordApp = _WordCreate ()
$oDoc = _WordDocGetCollection ($oWordApp, 0)
_WordDocAddLink ($oDoc, "", "www.AutoIt3.com", "", "AutoIt" & @CRLF, "Link to AutoIt")
$oLinks = _WordDocLinkGetCollection ($oDoc)
MsgBox(0, "Link Count", @extended)
