Function Reference

_GUICtrlComboBox_ReplaceEditSel

エディットボックス内の選択テキストを置換します。

#Include <GuiComboBox.au3>
_GUICtrlComboBox_ReplaceEditSel($hWnd, $sText)

 

パラメータ

$hWnd コントロールのハンドル
$sText 置換テキストを格納している文字列

 

返し値

なし。

 

注意

必須OS: Windows XP以降

$CBS_DROPDOWNスタイルまたは$CBS_DROPDOWNLISTスタイルが設定されたコンボボックスに使用した場合、この関数は失敗します。

 

関連

_GUICtrlComboBox_SetEditText, _GUICtrlComboBox_SetEditSel

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIComboBox.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_CB = False ; ComboBox/ComboBoxEx関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hCombo

    ; GUIを作成
    GUICreate("ComboBox Replace Edit Sel", 400, 296)
    $hCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    GUISetState()

    ; エディットのテキストを設定
    _GUICtrlComboBox_SetEditText($hCombo, "Old Edit Text")

    ; ファイルを追加
    _GUICtrlComboBox_BeginUpdate($hCombo)
    _GUICtrlComboBox_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($hCombo)

    Sleep(500)
   
    ; エディットボックス内の選択テキストを設定
    _GUICtrlComboBox_SetEditSel($hCombo, 0, -1)

    Sleep(500)

    ; エディットボックス内のテキストを制限
    _GUICtrlComboBox_ReplaceEditSel($hCombo, "New Edit Text")

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
