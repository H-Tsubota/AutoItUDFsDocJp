Function Reference

_GUIToolTip_GetDelayTime

初期時間、ポップアップ時間、再表示時間を取得します。

#Include <GuiToolTip.au3>
_GUIToolTip_GetDelayTime($hWnd, $iDuration)

 

パラメータ

$hWnd コントロールのハンドル
$iDuration どの時間値が取得されるかを指定するフラグ:
0 - マウスポインタ他のツールに移った場合にツールチップウィンドウの表示にかかる時間
1 - ツールの外接長方形内にマウスポインタがとどまっている場合にツールチップウィンドウが非表示になるまでの時間
2 - ウィンドウが表示されるまでにツールの外接長方形内にマウスポインタがとどまっていなければならない時間

 

返し値

成功: ミリ秒単位での指定された時間

 

注意

なし。
