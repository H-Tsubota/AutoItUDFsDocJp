Function Reference

_GUICtrlListBox_SetTabStops

タブストップ位置を設定します。

#Include <GuiListBox.au3>
_GUICtrlListBox_SetTabStops($hWnd, $aTabStops)

 

パラメータ

$hWnd コントロールのハンドル
$aTabStops 次のフォーマットの配列:
[0] - 配列内のタブストップの数(n)
[1] - 最初のタブストップ
[2] - 2番目のタブストップ
[n] - N番目のタブストップ

 

返し値

成功: True
失敗: False

 

注意

$aTabStops内の整数はリストボックスで選択されているフォントの平均文字幅の1/4を単位としています。
例えばタブストップ4は1文字分、タブストップ6は1.5文字分に相当します。
ただしリストボックスがダイアログボックスの一部の場合、整数はダイアログテンプレートの値を単位とします。
タブストップは昇順にソートされている必要があります。後方タブはサポートされていません。


 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIListBox.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_LB = False ; ListBox関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $aTabs[4] = [3, 100, 200, 300], $hListBox

    ; GUIを作成
    GUICreate("List Box Set Tab Stops", 400, 296)
    $hListBox = GUICtrlCreateList("", 2, 2, 396, 296, BitOR($LBS_STANDARD, $LBS_USETABSTOPS))
    GUISetState()

    ; タブストップを設定
    _GUICtrlListBox_SetTabStops($hListBox, $aTabs)
   
    ; タブされる文字を追加
    _GUICtrlListBox_AddString($hListBox, "Column 1" & @TAB & "Column 2" & @TAB & "Column 3")

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
