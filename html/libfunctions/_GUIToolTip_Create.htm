Function Reference

_GUIToolTip_Create

ツールチップコントロールを作成します。

#Include <GuiToolTip.au3>
_GUIToolTip_Create($hWnd[, $iStyle = 0x00000003])

 

パラメータ

$hWnd ツールチップウィンドウを所有するコントロールのハンドル
$iStyle [オプション]ツールチップのスタイル:
$TTS_ALWAYSTIP - ツールチップコントロールのオーナーウィンドウが非アクティブでもカーソルが乗った際にツールチップが表示されます。
このスタイルが設定されていない場合、ツールチップはツールのオーナーウィンドウがアクティブな場合のみ表示されます。
$TTS_BALLOON - ツールチップは丸い角でアイテムを指す突起のあるマンガの"ふきだし(バルーン)"のように表示されます。
$TTS_NOANIMATE - Windows 98、Windows 2000上でのツールチップのスライディングアニメーションを無効にします。
それ以前のOSではこのスタイルは無視されます。
$TTS_NOFADE - Windows 2000上でのツールチップのフェージングアニメーションを無効にします。
Windows NT以前のOS、Windows 95とWindows 98ではこのスタイルは無視されます。
$TTS_NOPREFIX - システムが文字列からアンパサンド文字(&)を取り除かないようにします。
このスタイルが設定されていない場合、システムは自動的にアンパサンド文字を取り除きます。
これによってアプリケーションはメニューアイテムとツールチップコントロールのテキストの両方で同じ文字列を使用できるようになります。
$TTS_CLOSE - ツールチップをキャンセルできるように"閉じる"アイコンを表示します。
デフォルト: $TTS_ALWAYSTIP, $TTS_NOPREFIX

 

返し値

成功: ツールチップウィンドウのハンドル
失敗: 0

 

注意

なし。
