Function Reference

_StringRepeat

指定した回数だけ文字列を繰り返します。

#Include <String.au3>
_StringRepeat($sString, $iRepeatCount)

 

パラメータ

$sString 繰り返す文字列
$iRepeatCount 文字列を繰り返す回数

 

返し値

成功: 指定回数だけ繰り返した文字列を返します
失敗: 空文字列を返し、@errorを設定します
@Error: 0 - 正常終了
1 - パラメータの一つが不正

 

注意

なし。

 

関連

 

例


#include <String.au3>

Msgbox(0,'',_StringRepeat("+-",40))
Exit
