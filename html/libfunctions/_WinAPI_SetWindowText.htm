Function Reference

_WinAPI_SetWindowText

指定されたウィンドウのタイトルバーのテキストを変更します。

#Include <WinAPI.au3>
_WinAPI_SetWindowText($hWnd, $sText)

 

パラメータ

$hWnd テキストを変更するウィンドウまたはコントロールのハンドル
$sText 新しいタイトルまたはコントロールテキストとして使用される文字列

 

返し値

成功: True
失敗: False

 

注意

目的のウィンドウが現在のプロセスのものだった場合、SetWindowTextは指定されたウィンドウ、コントロールへ送信される$WM_SETTEXTメッセージを発生します。
コントロールが$WS_CAPTION スタイルで作成されたリストボックスコントロールの場合、SetWindowTextはリストボックス全体ではなくコントロールにテキストを設定します。
他のプロセスのコントロールのテキストを設定するにはSetWindowTextを呼ぶ代わりに$WM_SETTEXTメッセージを直接送信してください。
