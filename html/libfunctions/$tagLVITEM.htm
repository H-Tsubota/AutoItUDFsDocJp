Keyword Reference

$tagLVITEM

リストビューアイテムの属性を受け取り、指定します。

Global Const $tagLVITEM = "int Mask;int Item;int SubItem;int State;int StateMask;ptr Text;int TextMax;int Image;int Param;" & _
        "int Indent;int GroupID;int Columns;ptr pColumns"

 

パラメータ

Mask この構造体のどのメンバーが設定データを格納しているか、どのメンバーが要求されているかを指定するフラグのセットです。このメンバーは次のものの1つまたは組み合わせです:
$LVIF_COLUMNS - Columnsが有効です
$LVIF_DI_SETITEM - オペレーティングシステムが要求されたリストアイテムの情報を保持しています
$LVIF_GROUPID - GroupIDが有効です
$LVIF_IMAGE - Imageが有効です
$LVIF_INDENT - Indentが有効です
$LVIF_NORECOMPUTE - コントロールはテキスト情報を取得するためにLVN_GETDISPINFOを発生しません
$LVIF_PARAM - Paramが有効です
$LVIF_STATE - Stateが有効です
$LVIF_TEXT - Textが有効です
Item この構造体が参照するアイテムのゼロ始まりのインデックス
SubItem この構造体が参照するサブアイテムの1始まりのインデックス
State アイテムの状態、状態画像、オーバーレイ画像を表します
StateMask Stateのどのビットが取得、変更されるかを指定する値
Text アイテムテキストを格納する文字列へのポインタ
TextMax Textによって指されるバッファの文字列の終端文字を含むバイト数
Image コントロール画像リスト内にあるアイテムのアイコンのインデックス
Param アイテム固有の値
Indent アイテムをインデントする画像幅数
GroupID アイテムを受け取るタイルビューグループの識別子
Columns このアイテムを表示するためのビュー列数
pColumns 列インデックス配列へのポインタ

 

注意

なし。
