Function Reference

_Assert

アサーション失敗時にメッセージを表示します。

#include <Debug.au3>
_Assert($sCondition [,$bExit = True [,$nCode = 0x7FFFFFFF [,$sLine = @ScriptLineNumber]]])

 

Parameters

$sCondition Trueと評価されるべき条件 (引用符で囲まれた式)
$bExit [オプション]Trueの場合、スクリプトは停止します
$nCode [オプション]スクリプトが停止した場合の終了コード
$sLine [オプション]アサーションが失敗した際の行番号を表示します。この値を変更しなかった場合、デフォルト値では正しい行が表示されます。

 

返し値

条件式の結果（終了しなかった場合のみ有効）

 

注意

返し時に@errorと@extendedは破棄されません。

 

関連

なし。

 

例


#include <Debug.au3>

$sABC= ""
_Assert('$sABC > ""')
