Function Reference

_WinAPI_DeleteDC

指定されたデバイスコンテキストを削除します。

#Include <WinAPI.au3>
_WinAPI_DeleteDC($hDC)

 

パラメータ

$hDC 削除するデバイスコンテキストを識別します。

 

返し値

成功: True
失敗: False

 

注意

アプリケーションは_WinAPI_GetDC関数の呼び出しによってハンドルを取得したDCを削除してはいけません。
そういったDCを解放するには代わりに_WinAPI_ReleaseDC関数を呼ぶ必要があります。

 

関連

_WinAPI_GetDC, _WinAPI_ReleaseDC, _WinAPI_CreateCompatibleDC
