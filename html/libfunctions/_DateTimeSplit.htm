Function Reference

_DateTimeSplit

日付けと時刻の入った文字列を2つの配列に分割します。

#Include <Date.au3>
_DateTimeSplit($sDate, ByRef $asDatePart, ByRef $iTimePart)

 

パラメータ

$sDate つぎの形式のどれか
"yyyy/mm/dd[ hh:mm[:ss]]"
"yyyy/mm/dd[Thh:mm[:ss]]"
"yyyy-mm-dd[ hh:mm[:ss]]"
"yyyy-mm-dd[Thh:mm[:ss]]"
"yyyy.mm.dd[ hh:mm[:ss]]"
"yyyy.mm.dd[Thh:mm[:ss]]"
$asDatePart 日付けが入る配列
$asDatePart[0]は返された値の数
$iTimePart 時刻が入る配列
$asTimePart[0]は返された値の数

 

返し値

成功: 日付と時刻の2つの配列/td>
失敗: 0。@ERRORを次の値に設定します
@error: 0 - 正常終了
1 - 無効な入力日付け

 

注意

なし。

 

関連

_DayValueToDate, _DateAdd, _DateDiff

 

例


#include <Date.au3>

Dim $MyDate
Dim $MyTime
_DateTimeSplit("2005/01/01 14:30",$MyDate,$MyTime)

For $x = 1 to $MyDate[0]
    MsgBox(0,$x,$MyDate[$x])
Next
For $x = 1 to $MyTime[0]
    MsgBox(0,$x,$MyTime[$x])
Next
