Function Reference

_WinNet_CancelConnection

既存のネットワーク接続をキャンセルします。

#Include <WinNet.au3>
_WinNet_CancelConnection($sName [, $fForce = True])

 

パラメータ

$sName リダイレクトされているローカルデバイスまたは切断するリモートネットワークリソースいずれかの名前。
このパラメータにリダイレクトされているローカルデバイスを指定すると、この関数は指定されたデバイスのリダイレクトだけをキャンセルします。
ラメータにリモートネットワークリソースを指定すると、デバイスなしのリモートネットワークとの接続だけを切断します。
$fForce [オプション]接続上に開かれているファイルやジョブがあった場合、接続の切断をおこなうかどうかを指定します。
このパラメータがFalseの場合に開かれているファイルやジョブがあるとこの関数は失敗します。

 

返し値

成功: True
失敗: False

 

注意

この関数は16ビットバージョンのWindowsとの互換性を図るために提供されています。
その他のWindowsベースのアプリケーションではWNet_CancelConnection2関数を使用してください。
