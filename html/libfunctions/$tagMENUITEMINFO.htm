Keyword Reference

$tagMENUITEMINFO

メニューアイテムの情報を格納します。

Global Const $tagMENUITEMINFO = "int Size;int Mask;int Type;int State;int ID;int SubMenu;int BmpChecked;int BmpUnchecked;" & _
        "int ItemData;ptr TypeData;int CCH;int BmpItem"

 

パラメータ

Size この構造体のサイズをバイト単位で指定します
Mask 取得、設定されるメンバー。このメンバーは次の値の1つまたは組み合わせです:
$MIIM_BITMAP - BmpItemを取得、設定します
$MIIM_CHECKMARKS - BmpCheckedとBmpUncheckedを取得、設定します
$MIIM_DATA - ItemDataを取得、設定します
$MIIM_FTYPE - Typeを取得、設定します
$MIIM_ID - IDを取得、設定します
$MIIM_STATE - Stateを取得、設定します
$MIIM_STRING - TypeDataを取得、設定します
$MIIM_SUBMENU - SubMenuを取得、設定します
$MIIM_TYPE - TypeとTypeDataを取得、設定します
Type メニューアイテムの種類。このメンバーは次の値の1つまたは組み合わせです:
$MFT_BITMAP - ビットマップを使用するメニューアイテムを表示
$MFT_MENUBARBREAK - 新しい行または列にメニューアイテムを配置
$MFT_MENUBREAK - 新しい行または列にメニューアイテムを配置
$MFT_OWNERDRAW - メニューオーナーへのメニューアイテム描画権限の割り当て
$MFT_RADIOCHECK - ラジオボタンマークを使用する選択メニューを表示
$MFT_RIGHTJUSTIFY - メニューアイテムとそれ以降のアイテムを右揃えにします
$MFT_RIGHTORDER - メニュー順を右から左に指定
$MFT_SEPARATOR - メニューアイテムにセパレータを指定
State メニューアイテム状態。このメンバーは次の値の1つまたは組み合わせです:
$MFS_CHECKED - メニューアイテムをチェックします
$MFS_DEFAULT - メニューアイテムをデフォルトに指定します
$MFS_DISABLED - メニューアイテムを無効にし、選択できないようにグレイアウトします
$MFS_ENABLED - メニューアイテムを選択できるように有効にします
$MFS_GRAYED - メニューアイテムを無効にし、選択できないようにグレイアウトします
$MFS_HILITE - メニューアイテムを強調表示します
ID メニューアイテムを識別するためのアプリケーション定義の16ビット値
SubMenu メニューアイテムに関連付けられたサブメニューまたはドロップダウンメニューのハンドル
BmpChecked 選択時にアイテムの隣に表示されるビットマップのハンドル
BmpUnchecked 非選択時にアイテムの隣に表示されるビットマップのハンドル
ItemData メニューアイテムに関連付けられたアプリケーション定義の値
TypeData メニューアイテムの内容
CCH メニューアイテムテキストの長さ
BmpItem 表示されるビットマップのハンドル

 

注意

なし。
