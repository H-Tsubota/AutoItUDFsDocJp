Keyword Reference

$tagLOGFONT

フォントの属性を定義します。

Global Const $tagLOGFONT = "int Height;int Width;int Escapement;int Orientation;int Weight;byte Italic;byte Underline;" & _
        "byte Strikeout;byte CharSet;byte OutPrecision;byte ClipPrecision;byte Quality;byte PitchAndFamily;char FaceName[32]"

 

パラメータ

Height フォントの文字セルまたは文字の論理単位での高さ
Width フォントの文字の論理単位での平均幅を指定します
Escapement 文字送りベクトルとX軸がなす角度を度単位で指定します
Orientation 各文字のベースラインとX軸がなす角度を度単位で指定します
Weight フォントの太さを0から1000の範囲で指定します
Italic Trueに設定することで斜体フォントを指定します
Underline Trueに設定することで下線付きフォントを指定します
StrikeOut Trueに設定することで取り消し線付きフォントを指定します
CharSet 文字セットを指定します
OutPrecision 出力精度を指定します
ClipPrecision クリッピング精度を指定します
Quality 出力品質を指定します
PitchAndFamily フォントのファミリーとピッチを指定します
FaceName フォントのタイプフェイス名を指定します

 

注意

なし。
