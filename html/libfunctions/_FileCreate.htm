Function Reference

_FileCreate

指定されたファイルを作成または内容消去します。

#Include <File.au3>
_FileCreate($sFilePath)

 

パラメータ

$sFilePath 作成するファイルのパスとファイル名。

 

返し値

成功: 1
失敗: 0を返し@errorを設定します
@Error: 1 = 指定されたファイルが開けません
2 = ファイルに書き込みができません

 

注意

なし。

 

関連

FileOpen

 

例


#include <File.au3>
If Not _FileCreate("error.log") Then
   MsgBox(4096,"Error", " Error Creating/Resetting log.      error:" & @error)
EndIf
