Function Reference

_NamedPipes_PeekNamedPipe

パイプから削除することなしにパイプからバッファにデータをコピーします。

#Include <NamedPipes.au3>
_NamedPipes_PeekNamedPipe($hNamedPipe)

 

パラメータ

$hNamedPipe パイプのハンドル

 

返し値

成功: 次のフォーマットの配列:
$aInfo[0] - パイプから読み取られたデータ
$aInfo[1] - パイプから読み取られたバイト数
$aInfo[2] - 読み取り可能な総バイト数
$aInfo[3] - このメッセージの読み取り残しバイト数
失敗: @errorを設定します。

 

注意

なし。
