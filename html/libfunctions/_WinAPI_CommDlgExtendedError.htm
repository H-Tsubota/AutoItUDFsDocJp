Function Reference

_WinAPI_CommDlgExtendedError

コモンダイアログボックスのエラー文字列を返します。この文字列はコモンダイアログボックス関数の実行中に起きた最新のエラーを表します。

#Include <WinAPI.au3>
_WinAPI_CommDlgExtendedError()

 

パラメータ

なし。

 

返し値

成功: エラー文字列
失敗: 拡張エラーの16進数文字列 0

 

注意

任意のコモンダイアログボックス関数関数の一般的なエラー文字列を返せます。

 

関連

_WinAPI_GetOpenFileName, _WinAPI_GetSaveFileName
