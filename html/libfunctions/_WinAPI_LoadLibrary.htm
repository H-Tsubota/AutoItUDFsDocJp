Function Reference

_WinAPI_LoadLibrary

指定された実行モジュールを呼び出し元プロセスのアドレス空間にマップします。

#Include <WinAPI.au3>
_WinAPI_LoadLibrary($sFileName)

 

パラメータ

$sFileName Win32実行モジュールの名前(.DLLまたは.EXEファイル)。
指定される名前は実行モジュールのファイル名です。

 

返し値

成功: 実行モジュールのハンドル
失敗: 0

 

注意

なし。

 

関連

_WinAPI_LoadLibraryEx, _WinAPI_FreeLibrary
