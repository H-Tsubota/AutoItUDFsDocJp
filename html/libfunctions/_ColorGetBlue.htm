Function Reference

_ColorGetBlue

与えた色の青色成分を返します。

#Include <Color.au3>
_ColorGetBlue($nColor)

 

パラメータ

$nColor 処理するRGB色

 

返し値

成功: 0-255の範囲の成分色

 

注意

なし。

 

関連

_ColorGetGreen, _ColorGetRed, _ColorGetRGB

 

例


#include <Color.au3>

$iBlueColor = _ColorGetBlue( 0x8080ff )
MsgBox( 4096, "AutoIt", $iBlueColor )
