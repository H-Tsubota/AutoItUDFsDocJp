Function Reference

_DateDayOfWeek

指定した曜日に基づいて、曜日名を返します。

#Include <Date.au3>
_DateDayOfWeek($iDayNum [, $iShort = 0])

 

パラメータ

$iDayNum 曜日番号(1 = 日曜日, 7 = 土曜日)
$iShort [オプション]形式:
0 - 完全な曜日の名前
1 - 曜日の省略名

 

返し値

成功: 曜日の名前
失敗: NULL文字列。@ERROR = 1を設定します

 

注意

この関数は英語での曜日名のみ返します。

 

関連

_DateDaysInMonth, _DateToDayOfWeek, _DateToDayOfWeekISO

 

例


#include <Date.au3>

; 完全な名前を取得
$sLongDayName = _DateDayOfWeek( @WDAY )

; 短縮名を取得
$sShortDayName = _DateDayOfWeek( @WDAY, 1 )

MsgBox( 4096, "Day of Week", "Today is: " & $sLongDayName & " (" & $sShortDayName & ")" )
