Function Reference

_IELoadWaitTimeout

_IELoadWaitのタイムアウト時間の現在の値をミリ秒単位で取得、または設定します。

#include <IE.au3>
_IELoadWaitTimeout ( [$i_timeout = -1] )

 

パラメータ

$i_timeout [オプション]ミリ秒数を取得するか設定するかのオプション
0または正の整数の場合、この値をタイムアウト時間に設定します
-1 = (デフォルト) 現在のタイムアウト時間(グローバル変数$__IELoadWaitTimeoutの値)を返します

 

返し値

成功: $i_timeout = -1の場合、現在のタイムアウト時間を返します。それ以外の場合、1を返します。
失敗: なし。

 

注意

タイムアウト時間を完全に無効にする方法はありませんが、とても大きな正の整数を設定することで機能的にそれに近いことはできます。初期のタイムアウト時間は30000ms（5分）です。

 

関連

_IELoadWait

 

例


; *******************************************************
; 例 1 - 現在のタイムアウト時間を取得
; *******************************************************
;
#include <IE.au3>
$iCurrentTimeout = _IELoadWaitTimeout ()

; *******************************************************
; 例 2 - タイムアウト時間を1分(60000ミリ秒)に設定
; *******************************************************
;
#include <IE.au3>
_IELoadWaitTimeout (60000)
