Function Reference

_ScreenCapture_SetTIFColorDepth

TIFFスクリーンキャプチャで使用される色深度を設定します。

#Include <ScreenCapture.au3>
_ScreenCapture_SetTIFColorDepth($iDepth)

 

パラメータ

$iDepth 画像色深度
 0 - デフォルトのエンコーダ色深度
24 - 24 bit
32 - 32 bit

 

返し値

なし。

 

注意

明示的に設定しない場合はTIFスクリーンキャプチャのデフォルト値、24bitが使用されます。
