Function Reference

_GUIToolTip_SetTitle

標準アイコンとタイトル文字列を追加します。

#Include <GuiToolTip.au3>
_GUIToolTip_SetTitle($hWnd, $sTitle[, $iIcon = 0])

 

パラメータ

$hWnd コントロールのハンドル
$sTitle タイトル文字列
$iIcon [オプション]次の値の1つを設定します:
0 - アイコンなし
1 - 情報アイコン
2 - 警告アイコン
3 - エラーアイコン

 

返し値

成功: True
失敗: False

 

注意

Windows XP SP2以降では$iIconにHICON値を使用することができます。
3より大きい値はすべてHICON値とみなされます。
