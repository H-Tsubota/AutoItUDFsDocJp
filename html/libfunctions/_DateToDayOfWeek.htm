Function Reference

_DateToDayOfWeek

与えられた日付けの曜日番号を返します。

#Include <Date.au3>
_DateToDayOfWeek($iYear, $iMonth, $iDay)

 

パラメータ

$iYear YYYY形式の有効な年
$iMon MM形式の有効な月
$iDay DD形式の有効な日

 

返し値

成功: 返されるされる曜日の範囲は1から7で、1が日曜日を表します
失敗: 0。@ERRORを次の値に設定します
@error: 0 - 正常終了
1 - 無効な入力日付け

 

注意

なし。

 

関連

_DateToDayOfWeekISO, _DateDayOfWeek, _DayValueToDate, _DateAdd, _DateDiff

 

例


#include <Date.au3>

; 与えられた日付けの曜日番号
$iWeekday = _DateToDayOfWeek (@YEAR, @MON, @MDAY)
; @Wdayと同じ
MsgBox(4096, "", "Todays WeekdayNumber is: " & $iWeekDay)
MsgBox(4096, "", "Today is a : " & _DateDayOfWeek($iWeekDay))
