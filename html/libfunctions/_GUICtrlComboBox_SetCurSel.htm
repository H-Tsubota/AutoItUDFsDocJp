Function Reference

_GUICtrlComboBox_SetCurSel

コンボボックスのリスト内の文字列を選択します。

#Include <GuiComboBox.au3>
_GUICtrlComboBox_SetCurSel($hWnd[, $iIndex = -1])

 

パラメータ

$hWnd コントロールのハンドル
$iIndex [オプション]選択する文字列のゼロ始まりのインデックスを指定します。

 

返し値

成功: 選択されたアイテムのインデックス
失敗: -1

 

注意

$iIndexが-1の場合、リスト内の現在の選択が解除されエディットコントロールがクリアされます。

$iIndexがリスト内のアイテム数より大きい場合または$iIndexが-1の場合、-1が返され選択が解除されます。

 

関連

_GUICtrlComboBox_GetCurSel

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GUIComboBox.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_CB = False ; ComboBox/ComboBoxEx関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hCombo

    ; GUIを作成
    GUICreate("ComboBox Set Cur Sel", 400, 296)
    $hCombo = GUICtrlCreateCombo("", 2, 2, 396, 296)
    GUISetState()

    ; ファイルを追加
    _GUICtrlComboBox_BeginUpdate($hCombo)
    _GUICtrlComboBox_AddDir($hCombo, @WindowsDir & "\*.exe")
    _GUICtrlComboBox_EndUpdate($hCombo)

    ; アイテムを選択
    _GUICtrlComboBox_SetCurSel($hCombo, 2)
   
    ; 現在選択しているものを取得
    MsgBox(4160, "Information", "Cur Sel: " & _GUICtrlComboBox_GetCurSel($hCombo))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
