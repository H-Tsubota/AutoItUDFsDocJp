Function Reference

_WeekNumberISO

与えた日付けが年頭から何週目かを計算します。

#Include <Date.au3>
_WeekNumberISO([$iYear = @YEAR[, $iMonth = @MON[, $iDay = @MDAY]]])

 

パラメータ

$iYear [オプション]年 (デフォルト = 今年)
$iMon [オプション]月 (デフォルト = 今月)
$iDay [オプション]日 (デフォルト = 今日)

 

返し値

成功: 与えた日付けが年頭から何週目かを返します
失敗: 0
@Error: 0 - 正常終了
1 - 不正なパラメータ値です
99 - 対応範囲外の日付けです

 

注意

なし。

 

関連

 

例


#include <Date.au3>

MsgBox(0,"Todays weeknumber","Todays weeknumber is " & _WeekNumberISO())
