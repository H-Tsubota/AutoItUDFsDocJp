Function Reference

_EventLog__Backup

イベントログをバックアップファイルに保存します。

#Include <EventLog.au3>
_EventLog__Backup($hEventLog, $sFileName)

 

パラメータ

$hEventLog イベントログのハンドル
$sFileName バックアップファイルの名前

 

返し値

成功: True
失敗: False

 

注意

この関数はイベントログをクリアしません。ユーザーがSE_BACKUP_NAME特権をもっていない場合、この関数は失敗します。

 

関連

_EventLog__OpenBackup

 

例


#include <EventLog.au3>

_Main()

Func _Main()
    Local $hEventLog

    $hEventLog = _EventLog__Open ("", "Application")
    _EventLog__Backup ($hEventLog, "C:\EventLog.bak")
    _EventLog__Close ($hEventLog)
EndFunc   ;==>_Main
