Function Reference

_GUICtrlToolbar_Customize

ツールバーのカスタマイズダイアログボックスを表示します。

#Include <GuiToolBar.au3>
_GUICtrlToolbar_Customize($hWnd)

 

パラメータ

$hWnd コントロールのハンドル

 

返し値

なし。

 

注意

ツールバーのカスタマイズダイアログボックスを表示するにはツールバーは$TBN_QUERYINSERT通知と$TBN_QUERYDELETE通知を処理しなければなりません。
ツールバーがこれらの通知を処理しない場合、$TB_CUSTOMIZEは無効になります。
