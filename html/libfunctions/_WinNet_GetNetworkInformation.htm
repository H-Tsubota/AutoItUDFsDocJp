Function Reference

_WinNet_GetNetworkInformation

特定のネットワークプロバイダに関する拡張情報を返します。

#Include <WinNet.au3>
_WinNet_GetNetworkInformation($sName)

 

パラメータ

$sName 情報が要求されるネットワークプロバイダ

 

返し値

成功: 次のフォーマットの配列:
$aInfo[0] - ネットワークプロバイダソフトウェアのバージョン番号
$aInfo[1] - ネットワークプロバイダソフトウェアの現在の状態:
0 - ネットワークは実行中
1 - ネットワークは利用不可
2 - ネットワークは現在、要求に答えられません
$aInfo[2] - ネットワークプロバイダのインスタンスハンドル
$aInfo[3] - 実行中のネットワークに固有のネットワークタイプの上位ワード
$aInfo[4] - ローカルプリンタデバイスをリダイレクトするための有効なプリント番号を表すビットフラグのセット。低位ビットはLPT1に対応しています。
$aInfo[5] - ディスクドライブのリダイレクトのために使用される有効なローカルディスクデバイスを表すビットフラグのセット。低位ビットはA:に対応しています。
失敗: @errorを設定します。

 

注意

なし。
