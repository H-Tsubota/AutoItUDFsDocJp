Function Reference

_StringProper

ExcelのProper関数と同じように文字列を正しい大文字小文字に変換します。

#Include <String.au3>
_StringProper($s_String)

 

パラメータ

$s_String 入力文字列

 

返し値

成功: 正しい文字列を返します
失敗: ""を返します

 

注意

この関数は非アルファベット文字に続く文字を大文字にします。

 

関連

 

例


#include <String.au3>
;Somebody Lastnames と返す
Msgbox(0,'',_StringProper("somebody lastnames"))
;Some.Body Last(Name) と返す
Msgbox(0,'',_StringProper("SOME.BODY LAST(NAME)"))
Exit
