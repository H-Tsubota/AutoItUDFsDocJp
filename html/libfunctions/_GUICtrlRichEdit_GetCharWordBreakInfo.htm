Function Reference

_GUICtrlRichEdit_GetCharWordBreakInfo

語の改行情報を取得します。

#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetCharWordBreakInfo($hWnd, $iCp)

 

パラメータ

$hWnd コントロールのハンドル
$iCP 興味のある文字の左の文字間位置

 

返し値

成功: カンマで分割された値で構成された文字列:
値1 - 語の改行フラグ:
c - この文字の次で改行されます。
d - 文字は語のデリミッターの最終位置です。行はこのデリミッターの次で改行されます。
w - 文字はホワイトスペースです（末尾のホワイトスペースは行の長さに含まれません）。
値2 - 文字クラス: 数値
失敗: ""を返し、@errorを設定します。
@error: 101 - $hWndがハンドルではありません。
102 - $iCpが数値ではありません。

 

注意

行はデリミッター、文字間、または異なるクラス間で改行されます。

文字クラスはワードブレイクプロシージャで定義されます。デフォルトプロシージャではクラスは次の通りです:
0 =英数字、1 = (ハイフンを除く)他の印刷文字、2 = スペース、3 = タブ、4 = ハイフンまたは段落の終端

 

関連

_GUICtrlRichEdit_GetCharPosOfNextWord, _GUICtrlRichEdit_GetCharPosOfPreviousWord

 

こちらも参照

MSDNライブラリでEM_FINDWORDBREAKを検索して下さい。

#AutoIt3Wrapper_Au3Check_Parameters= -d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiRichEdit.au3>
#include <GUIConstantsEx.au3>
#include <WindowsConstants.au3>

Global $lblMsg, $hRichEdit

Main()

Func Main()
    Local $hGui, $iMsg, $btnNext, $iCp = -1, $s
    $hGui = GUICreate("Example (" & StringTrimRight(@ScriptName,4) &")", 320, 350, -1, -1)
    $hRichEdit = _GUICtrlRichEdit_Create($hGui, "This is a test.", 10, 10, 300, 220, _
            BitOR($ES_MULTILINE, $WS_VSCROLL, $ES_AUTOVSCROLL))
    $lblMsg = GUICtrlCreateLabel("", 10, 235, 300, 60)
    $btnNext = GUICtrlCreateButton("Next", 270, 310, 40, 30)
    GUISetState()

    $s = Chr(9)
    For $i = 32 To 126
        $s &= Chr($i)
    Next
    _GuiCtrlRichEdit_AppendText($hRichEdit, $s & @CR)
    _GuiCtrlRichEdit_AppendText($hRichEdit, "AutoIt v3 is a (freeware) BASIC-like scripting language designed for " _
             & "automating the Windows GUI and general scripting.")
    _GuiCtrlRichEdit_AppendText($hRichEdit, @CRLF & "Another paragraph")
    While True
        $iMsg = GUIGetMsg()
        Select
            Case $iMsg = $GUI_EVENT_CLOSE
                GUIDelete()
                Exit
            Case $iMsg = $btnNext
                $iCp += 1
                _GuiCtrlRichEdit_GotoCharPos($hRichEdit, $iCp)
                GUICtrlSetData($lblMsg, _GuiCtrlRichEdit_GetCharWordBreakInfo($hRichEdit, $iCp))
                ControlFocus($hRichEdit, "", "")
        EndSelect
    WEnd
EndFunc   ;==>Main

Func Report($sMsg)
    GUICtrlSetData($lblMsg, $sMsg)
EndFunc   ;==>Report

        EndIf
    Else

            If $fStartOfWord Then

            Else
                $iWparam = $WB_LEFT
            EndIf
