Keyword Reference

$tagTBBUTTONINFO

ツールバーのボタンの固有情報を受け取り、格納します。

Global Const $tagTBBUTTONINFO = "int Size;int Mask;int Command;int Image;byte State;byte Style;short CX;int Param;ptr Text;int TextMax"

 

パラメータ

Size 構造体のバイト単位でのサイズ
Mask どのメンバーが有効な値を保持しているかを表すフラグのセット:
$TBIF_BYINDEX - $TB_GETBUTTONINFO、$TB_SETBUTTONINFOメッセージとともに送信されるParamはインデックスです
$TBIF_COMMAND - Command有効な情報を格納、または要求されています
$TBIF_IMAGE - Image有効な情報を格納、または要求されています
$TBIF_IMAGELABEL - ImageLabelが使用されることを表します
$TBIF_LPARAM - Param有効な情報を格納、または要求されています
$TBIF_SIZE - CX有効な情報を格納、または要求されています
$TBIF_STATE - State有効な情報を格納、または要求されています
$TBIF_STYLE - Style有効な情報を格納、または要求されています
$TBIF_TEXT - Text有効な情報を格納、または要求されています
Command ボタンのコマンド識別子
Image ボタンの画像インデックス。このメンバーに$I_IMAGECALLBACKを設定すると、ツールバーは必要な際に画像インデックスを取得するために$TBN_GETDISPINFO通知を送信します。このメンバーへの$I_IMAGENONEの設定はボタンに画像を設定しないことを表します。
State ボタンの状態フラグ
Style ボタンのスタイルフラグ
CX ボタンのピクセル単位での幅
Param ボタンに関連付けられたアプリケーション定義の値
Text ボタンテキストを受け取り、格納する文字バッファのアドレス
TextMax Textのバッファのサイズ。ボタン情報が設定されている場合はこのメンバーは無視されます。
ImageLabel アイテムのテキストラベルを画像で置き換えられるかどうかを表します

 

注意

なし。
