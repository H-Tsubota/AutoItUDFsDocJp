Function Reference

_GetIP

ネットワーク/コンピュータのパブリックIPアドレスを取得します。

#include <INet.au3>
_GetIP ( )

 

パラメータ

なし。

 

返し値

成功: パブリックIPアドレスを返します
失敗: -1を返し、@ERROR = 1を設定します。

 

注意

この関数はコンピュータまたはネットワークのパブリックIPアドレスのみを返します(i.e あなたのNATルータのWAN上のパブリックIP)。アドレスはcheckip.dyndns.orgやwww.whatismyip.comといった外部のIPディスカバリーウェブサイトにアクセスすることで決定されます。

 

関連

なし。

 

例


#include <Inet.au3>
$PublicIP = _GetIP()
MsgBox(0, "IP Address", "Your IP Address is: " & $PublicIP)
