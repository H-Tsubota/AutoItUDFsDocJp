Function Reference

_GUICtrlToolbar_FindToolbar

特定のツールバーを見つけます。

#Include <GuiToolBar.au3>
_GUICtrlToolbar_FindToolbar($hWnd, $sText)

 

パラメータ

$hWnd ウィンドウハンドルまたはウィンドウのテキスト
$sText 検索するボタンテキスト

 

返し値

成功: ツールバーのハンドル
失敗: 0

 

注意

なし。

 

関連

 

例


#include <GuiToolbar.au3>
#include <GuiConstantsEx.au3>

Opt('MustDeclareVars', 1)

$Debug_TB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hToolbar
   
    Run("explorer.exe /root, ,::{20D04FE0-3AEA-1069-A2D8-08002B30309D}")
    WinWaitActive("[CLASS:CabinetWClass]")
    Sleep ( 1000 )
    $hToolbar = _GUICtrlToolbar_FindToolbar ("[CLASS:CabinetWClass]", "&File")
    MsgBox(4096, "Information", "File Toolbar handle: 0x" & Hex($hToolbar))

EndFunc   ;==>_Main
