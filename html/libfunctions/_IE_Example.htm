Function Reference

_IE_Example

あなた自身のテスト、またはIE.au3のサンプル使用のためにあらかじめロードされているドキュメントを新しいブラウザウィンドウで表示します。

#include <IE.au3>
_IE_Example ( [$s_module = "basic"] )

 

パラメータ

$s_module [オプション]どのモジュールを実行するかを指定します
basic = (デフォルト) テキスト、リンク、画像の単純なHTMLページ
form = 複数のフォーム要素を持つ単純なHTMLページ
frameset = フレームを持つ単純なHTMLページ
iframe = アイフレームを持つ単純なHTMLページ
table = 表を持つフレームを持つ単純なHTMLページ

 

返し値

成功: InternetExplorer.Applicationオブジェクトを指すオブジェクト変数を返します
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
5 ($_IEStatus_InvalidValue) = 無効な値
@Extended: 無効なパラメータの番号が格納されています

 

注意

この関数はあらかじめ用意されているHTMLを使ってブラウザウィンドウを表示します。このHTMLはヘルプファイル、IE.au3のサンプル全体で使用されています。またIE.au3の関数を使ったテストスクリプトでそれらを使うこともできるようになっています。

 

関連

_IE_Introduction

 

例


; *******************************************************
; 例 1 - 各サンプルページを使ってブラウザウィンドウを作成
; 返されるオブジェクト変数は_IECreateや_IEAttachの返すオブジェクト変数と同じように使用できる
; *******************************************************
;
#include <IE.au3>
$oIE_basic = _IE_Example ("basic")
$oIE_form = _IE_Example ("form")
$oIE_table = _IE_Example ("table")
$oIE_frameset = _IE_Example ("frameset")
$oIE_iframe = _IE_Example ("iframe")
