Function Reference

_WinNet_GetLastError

最新の拡張エラーを取得します。

#Include <WinNet.au3>
_WinNet_GetLastError(ByRef $iError, ByRef $sError, ByRef $sName)

 

パラメータ

$iError 返し時に最新の拡張エラーコードを格納します
$sError 返し時に最新の拡張エラーコードメッセージを格納します
$sName 返し時にエラーを起こしたネットワークプロバイダを格納します

 

返し値

成功: 最新のエラーメッセージ
失敗: @errorを設定します。

 

注意

なし。
