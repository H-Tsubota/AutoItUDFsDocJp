Function Reference

_ScreenCapture_SetBMPFormat

BMPスクリーンキャプチャで使用されるbit形式を設定します。

#Include <ScreenCapture.au3>
_ScreenCapture_SetBMPFormat($iFormat)

 

パラメータ

$iFormat 画像の1pixeｌあたりのbit数(bpp)設定
0 = 16 bpp; 各RGB値、5bit。
1 = 16 bpp; 赤 5bit、緑 6bit、青 5bit。
2 = 24 bpp; 各RGB値、8bit。
3 = 32 bpp; 各RGB値、8bit。アルファ値なし。
4 = 32 bpp; 各RGB値、8bit。アルファ値あり。

 

返し値

なし。

 

注意

明示的に設定しない場合はBMPスクリーンキャプチャのデフォルト値、24bppが使用されます。

 

関連

 

例


#include <ScreenCapture.au3>

; 全画面をキャプチャ
_ScreenCapture_SetBMPFormat(0)
_ScreenCapture_Capture(@MyDocumentsDir & "\GDIPlus_Image.bmp")
