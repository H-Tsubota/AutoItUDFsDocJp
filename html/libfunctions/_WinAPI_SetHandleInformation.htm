Function Reference

_WinAPI_SetHandleInformation

オブジェクトハンドルの特定のプロパティを設定します。

#Include <WinAPI.au3>
_WinAPI_SetHandleInformation($hObject, $iMask, $iFlags)

 

パラメータ

$hObject オブジェクトのハンドル
$iMask 変更されるビットフラグを指定します。
$iFlags オブジェクトハンドルのプロパティを指定します。

 

返し値

成功: True
失敗: False

 

注意

なし。
