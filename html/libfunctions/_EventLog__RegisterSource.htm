Function Reference

_EventLog__RegisterSource

指定したイベントログの登録ているハンドルを取得します。

#Include <EventLog.au3>
_EventLog__RegisterSource($sServerName, $sSourceName)

 

パラメータ

$sServerName イベントログを開くサーバーのUNC名。
空白の場合、操作はローカルコンピュータ上でおこなわれます。
$sSourceName ハンドルを取得するイベントソースの名前。
ソースの名前はEventlogレジストリキー下のログのサブキーでなくてはなりません。

 

返し値

成功: イベントログのハンドル
失敗: 0

 

注意

ソース名が見つからない場合、イベントロギングサービスはアプリケーションログを使用します。その場合、新しいソースは作成されません。 イベントはソースに報告されますが、ソース用のイベント識別子の説明を調べるために指定されるメッセージとカテゴリーメッセージファイルは存在しません。 イベントログのハンドルを閉じるにはDeregisterEventSource関数を使用してください
