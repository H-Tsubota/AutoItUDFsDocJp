Keyword Reference

$tagGDIPBITMAPDATA

ビットマップデータ

Global Const $tagGDIPBITMAPDATA = "uint Width;uint Height;int Stride;uint Format;ptr Scan0;ptr Reserved"

 

パラメータ

Width ビットマップの1スキャンライン当たりのピクセル数
Height ビットマップ内のスキャンライン数
Stride ビットマップの連続するスキャンライン間のバイト単位でのオフセット。ストライドが正の場合ビットマップは、トップ-ダウンです。ストライドが負の場合、ビットマップはボトム-アップです。
Format ビットマップのピクセルフォーマットを指定します
Scan0 ビットマップの最初のスキャンライン(インデックス0)へのポインタ
Reserved 将来の使用のために予約されています

 

注意

なし。
