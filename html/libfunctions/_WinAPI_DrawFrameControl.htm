Function Reference

_WinAPI_DrawFrameControl

指定された種類、スタイルのフレームコントロールを描画します。

#Include <WinAPI.au3>
_WinAPI_DrawFrameControl($hDC, $ptrRect, $nType, $nState)

 

パラメータ

$hDC デバイスコンテキストのハンドル
$ptrRect 長方形の論理座標を格納している$tagRECT構造体へのポインタ
$nType 描画するフレームコントロールの種類を指定します。このパラメータは次の値のいずれかです:
$DFC_BUTTON - 標準のボタン
$DFC_CAPTION - タイトルバー
$DFC_MENU - メニューバー
$DFC_POPUPMENU - Windows 98/Me、Windows 2000/XP: ポップアップメニューアイテム
$DFC_SCROLL - スクロールバー
$nState フレームコントロールの初期状態を指定します。 $nTypeが$DFC_BUTTONの場合、$nStateは次の値のいずれかです:
$DFCS_BUTTON3STATE - 3状態ボタン
$DFCS_BUTTONCHECK - チェックボックス
$DFCS_BUTTONPUSH - プッシュボタン
$DFCS_BUTTONRADIO - ラジオボタン
$DFCS_BUTTONRADIOIMAGE - ラジオボタンのイメージ（正方形とは異なるイメージ）
$DFCS_BUTTONRADIOMASK - ラジオボタンのマスク（正方形とは異なるマスク）
$nTypeが$DFC_CAPTIONの場合、$nStateは次の値のいずれかです:
$DFCS_CAPTIONCLOSE - 閉じるボタン
$DFCS_CAPTIONHELP - ヘルプボタン
$DFCS_CAPTIONMAX - 最大化ボタン
$DFCS_CAPTIONMIN - 最小化ボタン
$DFCS_CAPTIONRESTORE - 復元ボタン
$nTypeが$DFC_MENUの場合、$nStateは次の値のいずれかです:
$DFCS_MENUARROW - サブメニュー矢印
$DFCS_MENUARROWRIGHT - 左向きのサブメニュー矢印。 アラビア語やヘブライ語などの右から左へ書く言語で使用される右から左へカスケードするメニューに使用されます。
$DFCS_MENUBULLET - 黒丸
$DFCS_MENUCHECK - チェックマーク
$nTypeが$DFC_SCROLLの場合、$nStateは次の値のいずれかです:
$DFCS_SCROLLCOMBOBOX - コンボボックスのスクロールバー
$DFCS_SCROLLDOWN - スクロールバーの下向き矢印
$DFCS_SCROLLLEFT - スクロールバーの左向き矢印
$DFCS_SCROLLRIGHT - スクロールバーの右向き矢印
$DFCS_SCROLLSIZEGRIP - ウィンドウの右下隅のサイズ変更グリップ
$DFCS_SCROLLSIZEGRIPRIGHT - ウィンドウの左下隅のサイズ変更グリップ。アラビア語やヘブライ語などの右から左へ書く言語で使用されます。
$DFCS_SCROLLUP - スクロールバーの上向き矢印
以下のスタイルはプッシュボタンの長方形の外接長方形を調整するために使用されます:
$DFCS_ADJUSTRECT - 外接長方形はプッシュボタンを囲む辺を除くように調整されます
以下の値の1つまたは複数を描画されるコントロールの状態設定に使用できます:
$DFCS_CHECKED - ボタンをチェックします
$DFCS_FLAT - ボタンは平らな縁を持ちます
$DFCS_HOT - Windows 98/Me、Windows 2000/XP: ボタンをホットトラッキングします
$DFCS_INACTIVE - ボタンを非アクティブ(グレイアウト)にします
$DFCS_PUSHED - ボタンを押します
$DFCS_TRANSPARENT - Windows 98/Me、Windows 2000/XP: 背景をそのままにします。このフラグは$DFCS_MENUARROWUPまたは$DFCS_MENUARROWDOWNのみと組み合わせられます。

 

返し値

成功: True
失敗: False

 

注意

定義済み定数用にFrameConstants.au3が必要です。
