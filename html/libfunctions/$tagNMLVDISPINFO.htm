Keyword Reference

$tagNMLVDISPINFO

$LVN_GETDISPINFOまたは$LVN_SETDISPINFO通知メッセージの情報を格納します。

Global Const $tagNMLVDISPINFO = $tagNMHDR & ";int Mask;int Item;int SubItem;int State;int StateMask;" & _
        "ptr Text;int TextMax;int Image;int Param;int Indent;int GroupID;int Columns;ptr pColumns"

 

パラメータ

$tagNMHDR 通知メッセージの情報を格納します
Mask この構造体のどのメンバーが設定データを保持しているか、あるいは要求されているかを指定するフラグのセット。 このメンバーは次のフラグのうちの1つまたは組み合わせです:
$LVIF_COLUMNS - Columnsが有効です
$LVIF_DI_SETITEM - オペレーティングシステムがアイテム情報の要求リストを保持しています
$LVIF_GROUPID - GroupIDが有効です
$LVIF_IMAGE - Imageが有効です
$LVIF_INDENT - Indentが有効です
$LVIF_NORECOMPUTE - このコントロールはテキスト情報を取得するためのLVN_GETDISPINFOを発生しません
$LVIF_PARAM - Paramが有効です
$LVIF_STATE - Stateが有効です
$LVIF_TEXT - Textが有効です
Item この構造体が参照するアイテムのゼロ始まりのインデックス
SubItem この構造体が参照するサブアイテムの1始まりのインデックス
State アイテムの状態、状態画像、オーバーレイ画像を表します
StateMask Stateのどのビットが取得、変更されるかを指定する値
Text アイテムテキストを格納している文字列へのポインタ
TextMax Textによって指されるバッファの文字列終了文字を含むバイト数
Image コントロールの画像リストにあるアイテムアイコンのインデックス
Param アイテム固有の値
Indent アイテムのインデントの画像幅数
GroupID アイテムを受け取るタイルビューグループの識別子
Columns このアイテムを表示するためのタイルビューの列数
pColumns 列インデックス配列へのポインタ

 

注意

なし。
