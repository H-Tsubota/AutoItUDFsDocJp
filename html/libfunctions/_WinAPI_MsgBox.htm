Function Reference

_WinAPI_MsgBox

オリジナルのものより広い余白のメッセージボックスを表示します。

#Include <WinAPI.au3>
_WinAPI_MsgBox($iFlags, $sTitle, $sText)

 

パラメータ

$iFlags ウィンドウ作成時に使用するフラグ
$sTitle ウィンドウタイトル
$sText ウィンドウテキスト

 

返し値

なし。

 

注意

この関数は(私の考えでは)より見栄えのするメッセージボックスを作成します。
またユーザーがマウスを動かせるようにBlockInputをオフになっていることを確認します。

 

関連

_WinAPI_ShowMsg, _WinAPI_ShowError
