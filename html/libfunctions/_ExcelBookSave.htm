Function Reference

_ExcelBookSave

指定したExcelオブジェクトのアクティブなワークブックを保存します。

#Include <Excel.au3>
_ExcelBookSave($oExcel [, $fAlerts = 0])

 

パラメータ

$oExcel _ExcelBookOpen()か_ExcelBookNew()の呼び出しで開かれているExcelオブジェクト
$fAlerts [オプション]Excelの警告メッセージの無効/有効を設定するフラグ(0=無効、1=有効) (デフォルトは0)

 

返し値

成功: 1
失敗: 0を返し@errorに次の値を設定します
@error: 1 - 指定されたオブジェクトが存在しません
2 - ファイルが存在し、上書きフラグが設定されていません

 

注意

なし。br>

 

関連

 

例


; ***************************************************************
; 例 1 - 新しいワークブックを開き、オブジェクトの識別子を返す。その後、一切の警告を表示させずに保存する。
; *****************************************************************

#include <Excel.au3>

$oExcel = _ExcelBookNew()

_ExcelBookSave($oExcel) ;警告なしでファイルを保存
If Not @error Then MsgBox(0, "Success", "File was Saved!", 3)
