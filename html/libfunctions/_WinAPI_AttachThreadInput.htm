_WinAPI_AttachThreadInput

スレッドの入力処理機構を他のスレッドにアタッチします。

#Include <WinAPI.au3>
_WinAPI_AttachThreadInput($iAttach, $iAttachTo, $fAttach)

 

パラメータ

$iAttach 他のスレッドにアタッチされるスレッドの識別子
$iAttachTo アタッチされるスレッドの識別子
$fAttach アタッチモード:
True - スレッドはアタッチされます
False - スレッドはデタッチされます

 

返し値

成功: True
失敗: False

 

注意

なし。
