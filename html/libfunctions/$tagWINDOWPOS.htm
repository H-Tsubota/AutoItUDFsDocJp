Keyword Reference

$tagWINDOWPOS

ウィンドウのサイズと位置に関する情報を格納したWINDOWPOS構造体です。

Global Const $tagWINDOWPOS = "hwnd hWnd;int InsertAfter;int X;int Y;int CX;int CY;int Flags"

 

パラメータ

hWnd ウィンドウのハンドル
InsertAfter ウィンドウのZオーダー位置を指定します
X ウィンドウの左端位置を指定します
Y ウィンドウの上端位置を指定します
CX ウィンドウの幅をピクセル単位で指定します
CY ウィンドウの高さをピクセル単位で指定します
Flags ウィンドウ位置を指定します。次の値の1つまたは組み合わせです:
$SWP_DRAWFRAME - ウィンドウの周りのフレームを描画します
$SWP_FRAMECHANGED - ウィンドウサイズが固定されている場合でも、WM_NCCALCSIZEメッセージをウィンドウに送信します
$SWP_HIDEWINDOW - ウィンドウを非表示にします
$SWP_NOACTIVATE - ウィンドウをアクティブ化できないようにします
$SWP_NOCOPYBITS - クライアントエリア内のコンテンツすべてを破棄します
$SWP_NOMOVE - 現在の位置を維持します (x,yパラメータは無視されます)
$SWP_ NOOWNERZORDER - オーナーウィンドウのZオーダー位置を固定します
$SWP_NOREDRAW - 変更を再描画しません
$SWP_NOREPOSITION - SWP_NOOWNERZORDERフラグと同じです
$SWP_NOSENDCHANGING - ウィンドウのWM_WINDOWPOSCHANGINGメッセージの受信を妨げます
$SWP_NOSIZE - 現在のサイズを維持します (cx,cyパラメータは無視されます)
$SWP_NOZORDER - 現在のZオーダー位置を維持します (InsertAfterパラメータは無視されます)
$SWP_SHOWWINDOW - ウィンドウを表示します

 

注意

なし。
