Function Reference

_IEErrorHandlerDeRegister

登録されたユーザーCOMエラーハンドラを無効にします。

#include <IE.au3>
_IEErrorHandlerDeRegister ( )

 

パラメータ

なし

 

返し値

成功: 1を返します
失敗: なし

 

注意

_IEErrorHandlerRegisterの注意を参照。

 

関連

_IEErrorHandlerRegister

 

例


; *******************************************************
; 例 1 - カスタムエラーハンドラを登録したあと、登録抹消
; *******************************************************
;
#include <IE.au3>
; カスタムエラーハンドラの登録
_IEErrorHandlerRegister ("MyErrFunc")
; なんらかの処理
; カスタムエラーハンドラの登録抹消
_IEErrorHandlerDeregister ()
; なんらかの処理

Exit

Func MyErrFunc()
    $HexNumber = Hex($oIEErrorHandler.number, 8)
    MsgBox(0, "", "We intercepted a COM Error !" & @CRLF & _
            "Number is: " & $HexNumber & @CRLF & _
            "Windescription is: " & $oIEErrorHandler.windescription)
    SetError(1) ; この関数が返った時のチェック用
EndFunc   ;==>MyErrFunc
