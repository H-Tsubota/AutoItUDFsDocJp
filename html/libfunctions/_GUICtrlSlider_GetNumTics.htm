Function Reference

_GUICtrlSlider_GetNumTics

スライダーの目盛り数を取得します。

#Include <GuiSlider.au3>
_GUICtrlSlider_GetNumTics($hWnd)

 

パラメータ

$hWnd コントロールのハンドル

 

返し値

成功: 目盛りなしのフラグが設定されている場合、開始目盛りと終了目盛りの2つだけが返されます。
        $TBS_NOTICKSが設定されている場合、ゼロが返されます。
        それ以外の場合、範囲の最大値と最小値の差を目盛り間隔で割り、2を足した値が返されます。

 

注意

なし。

 

関連

_GUICtrlSlider_SetTicFreq

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiSlider.au3>

Opt('MustDeclareVars', 1)

$Debug_S = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hSlider

    ; GUIを作成
    GUICreate("Slider Get Num Tics", 400, 296)
    $hSlider = GUICtrlCreateSlider(2, 2, 396, 20, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS, $TBS_ENABLESELRANGE))
    GUISetState()

    ; 目盛り数を取得
    MsgBox(4160, "Information", "Num Tics: " & _GUICtrlSlider_GetNumTics($hSlider))
   
    ; 目盛り間隔を設定
    _GUICtrlSlider_SetTicFreq($hSlider, 1)

    ; 目盛り数を取得
    MsgBox(4160, "Information", "Num Tics: " & _GUICtrlSlider_GetNumTics($hSlider))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
