Keyword Reference

$tagIMAGEINFO

画像リスト内の画像の情報を格納します。

Global Const $tagIMAGEINFO = "hwnd hBitmap;hwnd hMask;int Unused1;int Unused2;int Left;int Top;int Right;int Bottom"

 

パラメータ

hBitmap 画像を格納しているビットマップのハンドル
hMask 画像用マスクを格納しているモノクロビットマップのハンドル
Unused1 使用されません
Unused2 使用されません
Left 画像長方形の左端
Top 画像長方形の上端
Right 画像長方形の右端
Bottom 画像長方形の底端

 

注意

なし。
