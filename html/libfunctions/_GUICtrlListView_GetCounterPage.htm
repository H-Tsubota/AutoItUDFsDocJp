Function Reference

_GUICtrlListView_GetCounterPage

表示領域垂直方向に収まるアイテムの数を計算します。

#Include <GuiListView.au3>
_GUICtrlListView_GetCounterPage($hWnd)

 

パラメータ

$hWnd コントロールのハンドル

 

返し値

完全に表示されるアイテムの数を返します。

 

注意

現在の表示がアイコン表示、小さなアイコン表示の場合、返される値はリストビューコントロール内のアイテムの総数になります。

 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiListView.au3>

Opt('MustDeclareVars', 1)

$Debug_LV = False ; ListView関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hListView
   
    GUICreate("ListView Get Counter Page", 400, 300)
    $hListView = GUICtrlCreateListView("Column 1|Column 2|Column 3", 2, 2, 394, 268)
    _GUICtrlListView_SetExtendedListViewStyle($hListView, BitOR($LVS_EX_FULLROWSELECT, $LVS_EX_GRIDLINES))
    GUISetState()
   
    ; 1ページあたりのアイテム総数を表示
    MsgBox(4160, "Information", "Items per page: " & _GUICtrlListView_GetCounterPage($hListView))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
