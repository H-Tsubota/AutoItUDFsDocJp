Function Reference

_GUICtrlMonthCal_SetDayState

現在表示されている全ての月の日付け状態を設定します。

#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_SetDayState($hWnd, $aMasks)

 

パラメータ

$hWnd コントロールのハンドル
$aMasks カレンダーで表示されている月と対応する整数の配列

 

返し値

成功: True
失敗: False

 

注意

この関数を使用する場合、カレンダーコントロール作成時に$MCS_DAYSTATEスタイルを設定する必要があります。

 

関連

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiMonthCal.au3>
#include <WindowsConstants.au3>

Opt('MustDeclareVars', 1)

$Debug_MC = False ; MonthCal関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hMonthCal

    ; GUIを作成
    GUICreate("Month Calendar Set Day State", 400, 300)
    $hMonthCal = GUICtrlCreateMonthCal("", 4, 4, -1, -1, BitOR($WS_BORDER, $MCS_DAYSTATE), 0x00000000)

    ; マスクを用意する必要のある月の数を取得。通常は3。
    Local $aMasks[_GUICtrlMonthCal_GetMonthRangeSpan($hMonthCal, True)]

    ; 現在の月の1日目、8日目と16日目を太文字にする。
    ; バイナリマスクでは1000 0000 1000 0001、16進数では0x8081を使用する
    $aMasks[1] = 0x8081
    _GUICtrlMonthCal_SetDayState($hMonthCal, $aMasks)

    GUISetState()

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
