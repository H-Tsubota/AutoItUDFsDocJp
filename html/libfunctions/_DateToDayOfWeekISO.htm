Function Reference

_DateToDayOfWeekISO

与えられた日付けのISOでの曜日番号を返します。

#Include <Date.au3>
_DateToDayOfWeekISO($iYear, $iMonth, $iDay)

 

パラメータ

$iYear YYYY形式の有効な年
$iMon MM形式の有効な月
$iDay DD形式の有効な日

 

返し値

成功: 返される曜日の範囲は0から6で、0が月曜日を表します
失敗: 0。@ERRORを次の値に設定します
@error: 0 - 正常終了
1 - 無効な入力日付け

 

注意

なし。

 

関連

_DateToDayOfWeek, _DateDayOfWeek, _DayValueToDate, _DateAdd, _DateDiff

 

例


#include <Date.au3>
; 与えられた日付けのISO曜日番号。0=月曜 - 6=日曜
$iWeekday = _DateToDayOfWeekISO (@YEAR, @MON, @MDAY)
; @Wdayとは異なる
MsgBox(4096, "", "Todays ISO WeekdayNumber is: " & $iWeekDay)
