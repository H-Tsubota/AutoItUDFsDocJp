Function Reference

_WordDocAddLink

ドキュメントにハイパーリンクを追加します。

#include <Word.au3>
_WordDocAddLink ( ByRef $o_object [, $o_Anchor = "" [, $s_Address = "" [, $s_SubAddress = "" [, $s_ScreenTip = "" [, $s_TextToDisplay = "" [, $s_Target = ""]]]]]] )

 

パラメータ

$o_object ドキュメントオブジェクトまたはWord.Applicationのオブジェクト変数
$o_Anchor [オプション] ハイパーリンクにしたいテキストまたは画像。
"" = (デフォルト) ドキュメント全体を範囲として使用します
$s_Address [オプション] 指定されたリンクのアドレス。アドレスにはeメールアドレス、インターネットアドレスまたはファイル名を使用できます。
"" = (デフォルト) 指定されたドキュメントへのリンクが使用されます
$s_SubAddress [オプション] ブックマーク、名前付き範囲、スライド番号などのリンク先ファイル内の位置名。
$s_ScreenTip [オプション] マウスポインターが指定されたハイパーリンクをポイントした際にスクリーンチップとして表示されるテキスト。
"" = (デフォルト) $s_Addressの値を使用します
$s_TextToDisplay [オプション] 指定されたハイパーリンクの表示テキスト。この引数の値がAnchorで指定されたテキスト、画像と置き換わります。
"" = (デフォルト) $s_Addressの値を使用します
$s_Target [オプション] 指定されたハイパーリンクをロードするフレーム、ウィンドウの名前。

 

返し値

成功: 1を返します
失敗: 0を返し@ERROR = 1を設定します
@Error: $_WordStatus_Success = 正常終了
$_WordStatus_InvalidDataType = 無効なデータ型
$_WordStatus_InvalidObjectType = 無効なオブジェクト型
@Extended: 無効なパラメータの数を格納

 

注意

なし。

 

関連

None.

 

例


; *******************************************************
; 例 1 - 新しい空のドキュメントでWordウィンドウを作成
;               リンクを追加後、リンクのコレクションを取得
; *******************************************************
;
#include <Word.au3>
$oWordApp = _WordCreate ()
$oDoc = _WordDocGetCollection ($oWordApp, 0)
_WordDocAddLink ($oDoc, "", "www.AutoIt3.com", "", "AutoIt" & @CRLF, "Link to AutoIt")
$oLinks = _WordDocLinkGetCollection ($oDoc)
MsgBox(0, "Link Count", @extended)
