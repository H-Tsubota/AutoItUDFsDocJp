Function Reference

_GUICtrlEdit_LineScroll

テキストをスクロールします。

#Include <GuiEdit.au3>
_GUICtrlEdit_LineScroll($hWnd, $iHoriz, $iVert)

 

パラメータ

$hWnd コントロールのハンドル
$iHoriz 水平方向のスクロール文字数を指定します。
$iVert 垂直方向のスクロール行数を指定します。

 

返し値

成功: True
失敗: False

 

注意

コントロールはエディットコントロールのテキストの最終行以降には垂直スクロールしません。

現在の行数と$iVertパラメータで指定された行数の和がエディットコントロールの総行数を超える場合、
その値はエディットコントロールの最終行がエディットコントロールウィンドウの最上部にスクロールされるように調整されます。

_GUICtrlEdit_LineScrollはテキストを垂直方向または水平方向にスクロールします。
_GUICtrlEdit_LineScrollを使用して全ての行の最後の文字以降も水平にスクロールすることができます。

 

関連

_GUICtrlEdit_Scroll

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiEdit.au3>
#include <GuiStatusBar.au3>
#include <GuiConstantsEx.au3>
#include <WindowsConstants.au3>

Opt('MustDeclareVars', 1)

$Debug_Ed = False ; Edit関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $StatusBar, $hEdit, $hGUI
    Local $Wow64 = ""
    If @AutoItX64 Then $Wow64 = "\Wow6432Node"
    Local $sFile = RegRead("HKEY_LOCAL_MACHINE\SOFTWARE" & $Wow64 & "\AutoIt v3\AutoIt", "InstallDir") & "\include\changelog.txt"
    Local $aPartRightSide[2] = [378, -1]

    ; GUI作成
    $hGUI = GUICreate("Edit Line Scroll", 400, 300)
    $hEdit = GUICtrlCreateEdit("", 2, 2, 394, 268, BitOR($ES_WANTRETURN, $WS_VSCROLL))
    $StatusBar = _GUICtrlStatusBar_Create($hGUI, $aPartRightSide)
    _GUICtrlStatusBar_SetIcon($StatusBar, 1, 97, "shell32.dll")
    GUISetState()

    ; 余白を設定
    _GUICtrlEdit_SetMargins($hEdit, BitOR($EC_LEFTMARGIN, $EC_RIGHTMARGIN), 10, 10)

    ; テキストを設定
    _GUICtrlEdit_SetText($hEdit, FileRead($sFile))

    ; スクロール
    _GUICtrlStatusBar_SetText($StatusBar, "Scrolled: " & _GUICtrlEdit_LineScroll($hEdit, 0, _GUICtrlEdit_GetLineCount($hEdit)))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
