Function Reference

_IEBodyReadText

ドキュメントの<body>タグ内のテキストを返します。

#include <IE.au3>
_IEBodyReadText ( ByRef $o_object )

 

パラメータ

$o_object InternetExplorer.Applicationのオブジェクト変数、ウィンドウまたはフレームオブジェクト

 

返し値

成功: ドキュメントの<body>内に含まれるテキストを返します
失敗: 0を返し@ERRORを設定します
@Error: 0 ($_IEStatus_Success) = 正常終了
3 ($_IEStatus_InvalidDataType) = 無効なデータ型
@Extended: 無効なパラメータの番号が格納されています

 

注意

この関数はドキュメントボディーの.innerTextプロパティーを使用しています。.innerTextの使用はほぼ全てのDOMエレメントで可能で、ドキュメント内のテキストを_IEPropertyGetを使用して高速で簡単に解析するための方法です。

 

関連

_IEBodyReadHTML, _IEBodyWriteHTML, _IEPropertyGet

 

例


; *******************************************************
; 例 1 - basicなサンプルをブラウザで開き、ボディーのテキストを読み込み、
;               (HTMLタグのついたコンテンツは取り除かれる) MsgBoxに表示
; *******************************************************
;
#include <IE.au3>
$oIE = _IE_Example ("basic")
$sText = _IEBodyReadText ($oIE)
MsgBox(0, "Body Text", $sText)
