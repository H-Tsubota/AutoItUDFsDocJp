Function Reference

_DateDaysInMonth

指定された月、年に基づいて、月の日数を返します。

#Include <Date.au3>
_DateDaysInMonth($iYear, $iMonthNum)

 

パラメータ

$iYear 4桁の年
$iMonNum 月番号(1 = 1月、 12 = 12月)

 

返し値

成功: 月の日数を返します
失敗: 0を返し、@Errorを設定します
@error: 0 - 正常終了
1 - 無効な月番号、年

 

注意

なし。

 

関連

_DateDayOfWeek

 

例


#include <Date.au3>

$iDays = _DateDaysInMonth( @YEAR,@MON )
MsgBox( 4096, "Days in 月", "This 月 has " & String( $iDays ) & " days in it." )
