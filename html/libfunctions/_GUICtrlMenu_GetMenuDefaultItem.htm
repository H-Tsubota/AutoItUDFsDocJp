_GUICtrlMenu_GetMenuDefaultItem

指定されたメニューのデフォルトメニューアイテムを取得します。

#Include <GuiMenu.au3>
_GUICtrlMenu_GetMenuDefaultItem($hMenu[, $fByPos = True[, $iFlags = 0]])

 

パラメータ

$hMenu メニューのハンドル
$fByPos [オプション]メニューアイテムの識別子、位置のどちらを取得するかを決定します:
 True - メニューアイテムの位置を返します
False - メニューアイテムの識別子を返します
$iFlags [オプション]関数がどのようにメニューアイテムを検索するかを指定します:
0 - 特別な検索パラメータなし
1 - デフォルトメニューアイテムが無効になっている場合でも関数がそれを返すように指定します
2 - デフォルトアイテムがサブメニューを開くものの場合、関数がそのサブメニュー内も再帰的に検索するよう指定します

 

返し値

成功: メニューアイテムの識別子または位置
失敗: -1

 

注意

なし。

例


#include <GuiMenu.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hWnd, $hMain, $hFile

    ; メモ帳を開く
    Run("Notepad.exe")
    WinWaitActive("[CLASS:Notepad]")
    $hWnd = WinGetHandle("[CLASS:Notepad]")
    $hMain = _GUICtrlMenu_GetMenu($hWnd)
    $hFile = _GUICtrlMenu_GetItemSubMenu($hMain, 0)

    ; "ファイル"メニューのデフォルトアイテムを取得/設定
    Writeln("File menu default item: " & _GUICtrlMenu_GetMenuDefaultItem($hFile))
    _GUICtrlMenu_SetMenuDefaultItem($hFile, 1)
    Writeln("File menu default item: " & _GUICtrlMenu_GetMenuDefaultItem($hFile))

EndFunc   ;==>_Main

; メモ帳にテキストを1行書き込む
Func Writeln($sText)
    ControlSend("[CLASS:Notepad]", "", "Edit1", $sText & @CR)
EndFunc   ;==>Writeln
