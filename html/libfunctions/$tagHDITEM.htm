Keyword Reference

$tagHDITEM

ヘッダーコントロール内のアイテムの情報を格納します。

Global Const $tagHDITEM = "int Mask;int XY;ptr Text;hwnd hBMP;int TextMax;int Fmt;int Param;int Image;int Order;int Type;ptr pFilter;int State"

 

パラメータ

Mask 有効なデータを格納している、または埋められいる他の構造体メンバーを表すフラグ
XY アイテムの幅と高さ
Text アイテム文字列のアドレス
hBMP アイテムビットマップのハンドル
TextMax アイテム文字列の長さ
Fmt アイテムのフォーマットを指定するフラグ
Param アプリケーション定義のアイテムデータ
Image 画像リスト内での画像のゼロ始まりのインデックス
Order ヘッダーコントロール内でのアイテムの左から右の出現順序
Type pFilterで指定されたフィルターの型
pFilter アプリケーション定義のデータアイテムのアドレス
State アイテムの状態

 

注意

なし。
