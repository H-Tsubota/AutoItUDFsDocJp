Function Reference

_GDIPlus_BitmapCreateFromGraphics

Graphicsオブジェクト、幅、高さに基づいてBitmapオブジェクトを作成します。

#Include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromGraphics($iWidth, $iHeight, $hGraphics)

 

パラメータ

$iWidth ビットマップの幅をピクセル単位で指定します
$iHeight ビットマップの高さをピクセル単位で指定します
$hGraphics Graphicsオブジェクトのハンドル

 

返し値

成功: Bitmapオブジェクトのハンドル
失敗: 0

注意

Bitmapオブジェクトの使用後は_GDIPlus_BitmapDisposeを呼んでリソースを解放してください。
