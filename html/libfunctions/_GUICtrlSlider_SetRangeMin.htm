Function Reference

_GUICtrlSlider_SetRangeMin

スライダーの最小論理位置を設定します。

#Include <GuiSlider.au3>
_GUICtrlSlider_SetRangeMin($hWnd, $iMinimum)

 

パラメータ

$hWnd コントロールのハンドル
$iMinimum スライダーの最小位置

 

返し値

なし。

 

注意

現在のスライダー位置が新しい最小値より小さい場合、_GUICtrlSlider_SetRangeMin関数がスライダー位置を新しい最小値に設定します。

 

関連

_GUICtrlSlider_GetRangeMin, _GUICtrlSlider_SetRange, _GUICtrlSlider_SetRangeMax

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiSlider.au3>

Opt('MustDeclareVars', 1)

$Debug_S = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hSlider

    ; GUIを作成
    GUICreate("Slider Set Range Min", 400, 296)
    $hSlider = GUICtrlCreateSlider(2, 2, 396, 20, BitOR($TBS_TOOLTIPS, $TBS_AUTOTICKS, $TBS_ENABLESELRANGE))
    GUISetState()

    ; 範囲の最小値を取得
    MsgBox(4160, "Information", "Range Min: " & _GUICtrlSlider_GetRangeMin($hSlider))
   
    ; 範囲の最小値を設定
    _GUICtrlSlider_SetRangeMin($hSlider, 20)

    ; 範囲の最小値を取得
    MsgBox(4160, "Information", "Range Min: " & _GUICtrlSlider_GetRangeMin($hSlider))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
