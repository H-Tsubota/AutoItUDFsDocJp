Function Reference

_GUICtrlTab_SetCurFocus

指定されたタブにフォーカスを設定します。

#Include <GuiTab.au3>
_GUICtrlTab_SetCurFocus($hWnd, $iIndex)

 

パラメータ

$hWnd コントロールのハンドル
$iIndex ゼロ始まりのアイテムインデックス

 

返し値

なし。

 

注意

タブコントロールに$TCS_BUTTONSスタイル(ボタンモード)が設定されている場合、フォーカスの合っているタブと選択されているタブは必ずしも一致しません。
例えば、タブが選択されている際にユーザーは矢印キーを押すことで選択タブを変えることなくフォーカスを別のタブに設定することができます。
ボタンモードでは$TCM_SETCURFOCUSは選択タブを変えることなく、入力フォーカスを指定されたタブに関連付けられたボタンに設定することができます。

タブコントロールに$TCS_BUTTONSスタイルが設定されていない場合にはフォーカスを変更すると選択タブも変更されます。

 

関連

_GUICtrlTab_GetCurFocus

 

例


#AutoIt3Wrapper_au3check_parameters=-d -w 1 -w 2 -w 3 -w 4 -w 5 -w 6
#include <GuiConstantsEx.au3>
#include <GuiTab.au3>

Opt('MustDeclareVars', 1)

$Debug_TAB = False ; 関数に渡されるClassNameを調べる。動作を確認するにはTrueを設定し、他のコントロールのハンドルを使用

_Main()

Func _Main()
    Local $hTab

    ; GUIを作成
    GUICreate("Tab Control Set Current Focus", 400, 300)
    $hTab = GUICtrlCreateTab(2, 2, 396, 296)
    GUISetState()

    ; タブを追加
    _GUICtrlTab_InsertItem($hTab, 0, "Tab 1")
    _GUICtrlTab_InsertItem($hTab, 1, "Tab 2")
    _GUICtrlTab_InsertItem($hTab, 2, "Tab 3")
   
    ; 現在のフォーカスを取得/設定
    _GUICtrlTab_SetCurFocus($hTab, 1)
    MsgBox(4160, "Information", "Current Focus: " & _GUICtrlTab_GetCurFocus($hTab))

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE
    GUIDelete()
EndFunc   ;==>_Main
