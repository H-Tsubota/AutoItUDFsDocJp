Function Reference

_Now

現在の日付けと時刻をPCの形式で返します。

#Include <Date.au3>
_Now()

 

パラメータ

なし。

 

返し値

PCの形式での現在の日付けと時刻を返します。

 

注意

日付け形式がレジストリに見つからなかった場合、mm/dd/yyyy形式の日付けが返されます。
時刻形式がレジストリに見つからなかった場合、hh:mm:ss形式の時刻が返されます。

 

関連

_NowDate, _NowTime

 

例


#include <Date.au3>

MsgBox( 4096, "", "Todays date/Time: " & _Now() )
