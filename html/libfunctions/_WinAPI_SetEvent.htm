Function Reference

_WinAPI_SetEvent

指定されたイベントオブジェクトをシグナル状態に設定します。

#Include <WinAPI.au3>
_WinAPI_SetEvent($hEvent)

 

パラメータ

$hEvent イベントオブジェクトのハンドル

 

返し値

成功: True
失敗: False

 

注意

手動リセットイベントオブジェクトの状態は　ResetEvent関数によって明示的に非シグナル状態に設定されるまではシグナル状態を保ちます。
待機中のスレッドまたは待機関数の1つを呼び出しによって指定されたイベントオブジェクトに対する待機をその後に開始したスレッドはそれがいくつあろうとオブジェクトがシグナル状態になっている際には解放できます。
