Function Reference

_WinNet_CloseEnum

WNetOpenEnum呼び出しによって開始されたネットワークリソースの列挙を終了します。

#Include <WinNet.au3>
_WinNet_CloseEnum($hEnum)

 

パラメータ

$hEnum 列挙インスタンスを識別するためのハンドル。
ハンドルはWNet_OpenEnum関数によって返されます。

 

返し値

成功: True
失敗: False

 

注意

なし。

 

関連

_WinNet_OpenEnum
