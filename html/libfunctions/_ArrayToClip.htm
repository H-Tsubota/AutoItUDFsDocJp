Function Reference

_ArrayToClip

配列の内容をクリップボードに送信します。各要素はキャリッジリターンによって区切られます。

#Include <Array.au3>
_ArrayToClip(Const ByRef $avArray [, $iStart = 0 [, $iEnd = 0]])

 

パラメータ

$avArray クリップボードにコピーされる配列
$iStart [オプション] 配列のコピー開始インデックス
$iEnd [オプション] 配列のコピー終了インデックス

 

返し値

成功:: 1
失敗: 0。@errorを設定します
@error: -1 - ClipPut() 失敗
その他 - エラーコードについては_ArrayToString()の説明を参照

 

注意

なし。

 

関連

_ArrayToString

 

例


#include <Array.au3>

Local $avArray = StringSplit("a,b,c,d,e,f,g,h,i", ",")
_ArrayToClip($avArray, 1)
MsgBox(0, "_ArrayToClip() Test", ClipGet())
