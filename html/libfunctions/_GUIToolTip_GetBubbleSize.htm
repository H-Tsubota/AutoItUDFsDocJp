Function Reference

_GUIToolTip_GetBubbleSize

コントロールの幅と高さを返します。

#Include <GuiToolTip.au3>
_GUIToolTip_GetBubbleSize($hWnd, $hTool, $iID[, $iFlags = 0x000000A0])

 

パラメータ

$hWnd コントロールのハンドル
$hTool ツールがあるウィンドウのハンドル
$iID ツールの識別子
$iFlags [オプション]ツールチップの表示を制御するフラグ

 

返し値

成功: 下位ワードがツールチップの幅、上位ワードがツールチップの高さである値を返します。

 

注意

なし。

 

関連

_GUIToolTip_GetBubbleHeight, _GUIToolTip_GetBubbleWidth
