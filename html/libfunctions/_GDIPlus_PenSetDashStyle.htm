Function Reference

_GDIPlus_PenSetDashStyle

ペンのダッシュスタイルを設定します。

#Include <GDIPlus.au3>
_GDIPlus_PenSetDashStyle($hPen[, $iStyle = 0])

 

パラメータ

$hPen Penオブジェクトのハンドル
$iStyle [オプション]ダッシュスタイル。次のいずれかです:
$GDIP_DASHSTYLESOLID - 実線
$GDIP_DASHSTYLEDASH - 破線
$GDIP_DASHSTYLEDOT - 点線
$GDIP_DASHSTYLEDASHDOT - 一点鎖線
$GDIP_DASHSTYLEDASHDOTDOT - 二点鎖線
$GDIP_DASHSTYLECUSTOM - ユーザー定義のカスタム破線

 

返し値

成功: True
失敗: False

注意

なし。

 

関連

_GDIPlus_PenGetDashStyle

 

こちらも参照

MSDNライブラリでGdipSetPenDashStyleを検索して下さい。

 

例


#include <GuiConstantsEx.au3>
#include <GDIPlus.au3>

Opt('MustDeclareVars', 1)

_Main()

Func _Main()
    Local $hGUI, $hGraphic, $hPen

    ; GUI作成
    $hGUI = GUICreate("GDI+", 400, 300)
    GUISetState()

    ; リソースを作成
    _GDIPlus_Startup ()
    $hGraphic = _GDIPlus_GraphicsCreateFromHWND ($hGUI)
    $hPen = _GDIPlus_PenCreate (0xFF000000, 8)
    _GDIPlus_PenSetDashStyle ($hPen, $GDIP_DASHSTYLEDASHDOT)

    ; ペンのダッシュスタイルを表示
    MsgBox(4096, "Information", "Pen dash style: " & _GDIPlus_PenGetDashStyle ($hPen))

    ; 直線の描画
    _GDIPlus_GraphicsDrawLine ($hGraphic, 10, 150, 390, 150, $hPen)

    ; ユーザーが終了するまでループ
    Do
    Until GUIGetMsg() = $GUI_EVENT_CLOSE

    ; リソースを破棄
    _GDIPlus_PenDispose ($hPen)
    _GDIPlus_GraphicsDispose ($hGraphic)
    _GDIPlus_ShutDown ()

EndFunc   ;==>_Main
