Function Reference

_Security__GetAccountSid

アカウントのセキュリティ識別子(SID)を取得します。

#Include <Security.au3>
_Security__GetAccountSid($sAccount [, $sSystem = ""])

 

パラメータ

$sAccount アカウント名を指定します。
この関数が目的のドメインからアカウントを発見できるようにドメイン名\ユーザー名形式の完全な文字列を使用してください。
$sSystem [オプション]システム名。この文字列にはリモートコンピュータの名前を使用できます。
空文字列を使用した場合、アカウント名に対する処理はローカルシステム上で開始されます。 アカウント名がローカルシステム上で解決できない場合、この関数はローカルシステムの信頼するドメインコントローラを使用して アカウント名を解決しようとします。

 

返し値

成功: バイト構造体でバイナリのSIDを返します
失敗: 0

 

注意

なし。
