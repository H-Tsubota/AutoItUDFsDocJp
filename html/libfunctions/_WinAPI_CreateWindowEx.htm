_WinAPI_CreateWindowEx

オーバーラップウィンドウ、ポップアップウィンドウ、子ウィンドウを作成します。

#Include <WinAPI.au3>
_WinAPI_CreateWindowEx($iExStyle, $sClass, $sName, $iStyle, $iX, $iY, $iWidth, $iHeight, $hParent [, $hMenu = 0 [, $hInstance = 0 [, $pParam = 0]]])

 

パラメータ

$iExStyle 拡張ウィンドウスタイル
$sClass 登録クラス名
$sName ウィンドウ名
$iStyle ウィンドウスタイル
$iX ウィンドウの水平位置
$iY ウィンドウの垂直位置
$iWidth ウィンドウの幅
$iHeight ウィンドウの高さ
$hParent 親ウィンドウまたはオーナーウィンドウのハンドル
$hMenu [オプション]メニューのハンドルまたは子ウィンドウの識別子
$hInstance [オプション]アプリケーションインスタンスのハンドル
$pParam [オプション]ウィンドウ作成データへのポインタ

 

返し値

成功: 新しいウィンドウのハンドル
失敗: 0

 

注意

なし。
