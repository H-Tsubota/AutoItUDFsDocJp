Function Reference

_WinAPI_GetWindowDC

ウィンドウ全体のデバイスコンテキスト(DC)を取得します。

#Include <WinAPI.au3>
_WinAPI_GetWindowDC($hWnd)

 

パラメータ

$hWnd ウィンドウのハンドル

 

返し値

成功: 指定されたウィンドウのデバイスコンテキストのハンドル
失敗: 0

 

注意

GetWindowDCはウィンドウの非クライアント領域での特殊な描画効果のためのものです。
どのようなウィンドウであっても非クライント領域での描画は通常は推奨されません。
GetSystemMetrics関数を使用してタイトルバー、メニュー、スクロールバーなどの非クライント領域のパーツのサイズを取得することができます。
描画完了後、コンテキストハンドルを解放するために_WinAPI_ReleaseDC関数を呼ぶ必要があります。
ウィンドウデバイスコンテキストを解放しないとアプリケーションによる描画要求に対して深刻な影響を与えることになります。
